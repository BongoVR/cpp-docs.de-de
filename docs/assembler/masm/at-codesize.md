# <a name="codesize"></a>@CodeSize
0 für **sehr klein**, **kleine**, **COMPACT**, und **Flatfile** Modelle und 1 für **Mittel**,  **GROßE**, und **sehr groß** Modelle (numerische entsprechen).  
  
## <a name="syntax"></a>Syntax  
  
```  
  
@CodeSize  
  
```
