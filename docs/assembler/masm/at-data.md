# <a name="data"></a>@data
Der Name des Standard-Datengruppe. Ergibt für alle Modelle außer DGROUP **Flatfile**. Ergibt **Flatfile** unter der **Flatfile** Speichermodell (Textmakro).  
  
## <a name="syntax"></a>Syntax  
  
```  
  
@data  
  
```
