# @DataSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 für **SEHR KLEIN**, **KLEIN**, **MEDIUM**und **FLAT** Modelle, 1 für **COMPACT** und **LARGE** Modelle und 2 für **HUGE** Modell \(numerisch stellen Sie mit\).  
  
## Syntax  
  
```  
  
@DataSize  
  
```
