# BYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional ein Byte Speicherplatz für jedes `initializer`.  
  
## Syntax  
  
```  
[[name]] BYTE initializer [[, initializer]] ...  
```  
  
## Hinweise  
 Kann auch verwendet werden, da ein Typspezifizierer wo ein Typ gültig ist.
