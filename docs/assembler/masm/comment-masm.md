# COMMENT (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Behandelt den gesamten *Text* zwischen dem bzw. in der gleichen Zeile wie die Kommas als Kommentar.  
  
## Syntax  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
