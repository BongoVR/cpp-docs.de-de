# DB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional ein Byte Speicherplatz für jedes `initializer`.  `DB` ist ein Synonym für [BYTE](../../assembler/masm/byte-masm.md)  
  
## Syntax  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## Hinweise  
 Kann auch verwendet werden, da ein Typspezifizierer wo ein Typ gültig ist.
