# DD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional ein Doppelwort \(4 Bytes\) des Arbeitsspeichers für jedes `initializer`.  `DD` ist ein Synonym für [DWORD](../../assembler/masm/dword.md).  
  
## Syntax  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## Hinweise  
 Kann auch verwendet werden, da ein Typspezifizierer wo ein Typ gültig ist.
