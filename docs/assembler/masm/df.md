# DF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional 6 Bytes Speicherplatz für jedes `initializer`.  `DF` ist ein Synonym für [FWORD](../../assembler/masm/fword.md).  
  
## Syntax  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## Hinweise  
 Kann als Typspezifizierer auch dazu verwendet werden, eine beliebige Stelle, das ein Typ gültig ist.
