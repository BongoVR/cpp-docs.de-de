# <a name="386"></a>.386
Ermöglicht die Assembly möglicherweise nicht berechtigte-Anweisungen für die 80386-Prozessor. deaktiviert die Assembly von Anweisungen, die mit höheren Prozessoren eingeführt wurde.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
.386  
  
```
