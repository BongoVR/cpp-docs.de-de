# .386
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aktiviert die Assembly von nonprivileged Anweisungen für den Prozessor 80386; Deaktiviert Assembly mit den Anweisungen, die mit aktuelleren Prozessoren eingegeben werden.  
  
## Syntax  
  
```  
  
.386  
  
```  
  
## Hinweise  
 Aktiviert außerdem 80387 Anweisungen.
