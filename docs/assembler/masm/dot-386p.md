# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aktiviert alle Assemblys \(einschließlich Anweisungen privilegiertes\) für den Prozessor 80386; Deaktiviert Assembly mit den Anweisungen, die mit aktuelleren Prozessoren eingegeben werden.  
  
## Syntax  
  
```  
  
.386P  
  
```  
  
## Hinweise  
 Aktiviert außerdem 80387 Anweisungen.
