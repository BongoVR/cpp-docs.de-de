# .686P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aktiviert alle Assemblys \(einschließlich Anweisungen privilegiertes\) für den Pentium pro Prozessor.  
  
## Syntax  
  
```  
  
.686P  
  
```
