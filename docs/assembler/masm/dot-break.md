# .BREAK
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert Code aus, um einen [.WHILE](../../assembler/masm/dot-while.md) oder [.REPEAT](../../assembler/masm/dot-repeat.md)\-Block zu beenden, wenn *Bedingung* true ist.  
  
## Syntax  
  
```  
  
.BREAK [[.IF condition]]   
```
