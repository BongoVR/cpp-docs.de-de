# .CONTINUE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert Code, um dem oberen Rand eines [.WHILE](../../assembler/masm/dot-while.md) oder [.REPEAT](../../assembler/masm/dot-repeat.md)\-Blocks zu springen, wenn `condition` true ist.  
  
## Syntax  
  
```  
  
.CONTINUE [[.IF condition]]  
```
