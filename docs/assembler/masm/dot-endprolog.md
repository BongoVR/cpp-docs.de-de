# <a name="endprolog"></a>.ENDPROLOG
Signalisiert das Ende der Prologdeklarationen.  
  
## <a name="syntax"></a>Syntax  
  
```  
.ENDPROLOG  
```  
  
## <a name="remarks"></a>Hinweise  
 Es ist ein Fehler der Prologdeklarationen außerhalb der Region zwischen verwenden [PROC](../../assembler/masm/proc.md) FRAME und. ENDPROLOG.  
  
 Weitere Informationen finden Sie unter [MASM für X64 (ml64.exe)](../../assembler/masm/masm-for-x64-ml64-exe.md).
