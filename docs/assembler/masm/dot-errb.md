# .ERRB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert einen Fehler, wenn *textitem* leer ist.  
  
## Syntax  
  
```  
  
.ERRB <textitem> [[, message]]  
```
