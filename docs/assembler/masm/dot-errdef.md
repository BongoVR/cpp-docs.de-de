# .ERRDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert einen Fehler, wenn der *Name* eine zuvor definierte Bezeichnung, eine Variable oder ein Symbol ist.  
  
## Syntax  
  
```  
  
.ERRDEF name [[, message]]  
```
