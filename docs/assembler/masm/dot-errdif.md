# .ERRDIF]]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert einen Fehler, wenn die Textelemente unterschiedlich sind.  
  
## Syntax  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## Hinweise  
 Wenn **I** angegeben ist, ist der Vergleich die Groß\- und Kleinschreibung nicht berücksichtigt.
