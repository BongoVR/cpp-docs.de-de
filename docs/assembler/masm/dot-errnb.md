# .ERRNB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert einen Fehler, wenn *textitem* nicht leer ist.  
  
## Syntax  
  
```  
  
.ERRNB <textitem> [[, message]]  
```
