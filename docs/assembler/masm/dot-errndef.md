# .ERRNDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert einen Fehler, wenn der *Name* nicht definiert wurde.  
  
## Syntax  
  
```  
  
.ERRNDEF name [[, message]]  
```
