# <a name="nocref"></a>.NOCREF
Unterdrückt die Liste der Symbole in der Tabelle und der Browser die Symboldatei.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```

 Wenn Spaltennamen angegeben werden, werden nur die angegebenen Namen unterdrückt. Identisch mit [. XCREF](../../assembler/masm/dot-xcref.md).
