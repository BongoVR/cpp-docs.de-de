# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unterdrückt Listen von Symbolen in der Zuordnungstabelle\- und Browserdatei.  
  
## Syntax  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## Hinweise  
 Wenn Namen angegeben werden, dann nur die angegebenen Namen werden unterdrückt.  Identisch mit [.XCREF](../../assembler/masm/dot-xcref.md).
