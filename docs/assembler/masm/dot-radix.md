# .RADIX
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Legt die Basis im Bereich von 2 bis 16, um den Wert *des Ausdrucks*fest.  
  
## Syntax  
  
```  
  
.RADIX expression  
```
