# .WHILE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Generiert Code den Block von `statements` ausgeführt wird, während `condition` auf true festgelegt ist.  
  
## Syntax  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
