# <a name="dw"></a>DW
Zum Belegen und optional ein Wort (2 Bytes) des Speichers für die einzelnen initialisiert `initializer`. Kann auch verwendet werden, als Typspezifizierer an einer beliebigen Stelle ist ein Typ zulässig. `DW`ist ein Synonym für [WORD](../../assembler/masm/word.md).  
  
## <a name="syntax"></a>Syntax  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
