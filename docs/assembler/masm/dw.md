# DW
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional ein Wort \(2 Bytes\) des Arbeitsspeichers für jedes `initializer`.  Kann auch verwendet werden, da ein Typspezifizierer wo ein Typ gültig ist.  `DW` ist ein Synonym für [WORD](../../assembler/masm/word.md).  
  
## Syntax  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
