# ECHO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Zeigt *Meldung* in das Standardausgabegerät \(standardmäßig der Monitor\).  
  
## Syntax  
  
```  
  
ECHO   
message  
  
```  
  
## Hinweise  
 Identisch mit [%OUT](../../assembler/masm/percent-out.md).
