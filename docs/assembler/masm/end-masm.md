# <a name="end-masm"></a>END (MASM)
Markiert das Ende eines Moduls und, optional, wird der Einstiegspunkt des Programms auf *Adresse*.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
END [[address]]  
```
