# ENDP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Markiert das Ende des *Namens der* zuvor mit **PROC**Prozedur gestartet.  
  
## Syntax  
  
```  
  
name  
 ENDP  
  
```

 Weitere Informationen finden Sie unter [PROC](../../assembler/masm/proc.md).
