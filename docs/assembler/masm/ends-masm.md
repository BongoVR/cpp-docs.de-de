# ENDS (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Markiert das Segmentende\-, Struktur oder Union den *Namen, der* zuvor mit [SEGMENT](../../assembler/masm/segment.md), [STRUKTUR](../../assembler/masm/struct-masm.md), [UNION](../../assembler/masm/union.md)oder vereinfachten Segment direktiven gestartet wird.  
  
## Syntax  
  
```  
  
name ENDS  
```
