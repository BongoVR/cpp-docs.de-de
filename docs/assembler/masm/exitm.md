# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Beendet die aktuelle Erweiterung Wiederholen oder eines Makros blocks, wobei Assembly der folgenden Anweisung außerhalb des Blocks.  
  
## Syntax  
  
```  
  
EXITM [[textitem]]   
```  
  
## Hinweise  
 In einer Makrofunktion ist *textitem der* zurückgegebene Wert.
