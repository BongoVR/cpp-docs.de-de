# IFIDN]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erteilt Assembly, wenn die Textelemente identisch sind.  
  
## Syntax  
  
```  
  
IFIDN[[I]] textitem1, textitem2  
```  
  
## Hinweise  
 Wenn **I** angegeben ist, ist der Vergleich die Groß\- und Kleinschreibung nicht berücksichtigt.  Weitere Informationen finden Sie unter [IF](../../assembler/masm/if-masm.md) für vollständige Syntax.
