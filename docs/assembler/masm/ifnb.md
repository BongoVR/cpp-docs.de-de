# IFNB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erteilt Assembly, wenn *textitem* nicht leer ist.  
  
## Syntax  
  
```  
  
IFNB textitem  
```
