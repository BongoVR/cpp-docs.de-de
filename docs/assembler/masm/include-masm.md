# INCLUDE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fügt dem Quellcode aus der Quelldatei, die vom *Dateinamen* in der aktuellen Quelldatei während der Assembly angegeben ist.  
  
## Syntax  
  
```  
  
INCLUDE filename  
```  
  
## Hinweise  
 Der *Dateiname* muss in spitzen Klammern eingeschlossen werden, wenn es einen umgekehrten Schrägstrich, " Kleiner\-als " Symbol " Größer\-als " Semikolons Symbol einfaches Anführungszeichen oder doppelte Anführungszeichen einschließt.
