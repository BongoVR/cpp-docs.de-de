# INSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Sucht das erste Vorkommen der *textitem2* in *textitem1*.  
  
## Syntax  
  
```  
  
name  
 INSTR [[position,]] textitem1, textitem2  
```
