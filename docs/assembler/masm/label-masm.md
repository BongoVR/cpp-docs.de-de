# LABEL (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Erstellt eine neue Bezeichnung durch Zuweisen des aktuellen Position `type` angegebenen Werts und des Indikators *zu benennen*.  
  
## Syntax  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
