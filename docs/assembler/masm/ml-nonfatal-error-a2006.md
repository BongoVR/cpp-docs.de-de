# ML Nonfatal Error A2006
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Symbol nicht definiert: Bezeichner**  
  
 Es wurde versucht, ein Symbol zu verwenden, der nicht definiert wurde.  
  
 Eines der folgenden Elemente können auf hergestellt wird:  
  
-   Ein Symbol wurde nicht definiert.  
  
-   Ein Feld ist kein Mitglied der angegebenen Struktur.  
  
-   Ein Symbol wurde in einer Includedatei definiert, die nicht eingeschlossen wurde.  
  
-   Ein externes Symbol wurde ohne [EXTERN](../../assembler/masm/extern-masm.md) oder [EXTERNDEF](../../assembler/masm/externdef.md)\-Direktiven verwendet.  
  
-   Ein Symbolname war falsch geschrieben.  
  
-   Eine Gemeindeordnungs Bezeichnung wurde außerhalb des Bereichs verwiesen.
