# ML Nonfatal Error A2008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Syntaxfehler:**  
  
 Ein Token hat einen Syntaxfehler an der aktuellen Position.  
  
 Eines der folgenden Elemente können auf hergestellt wird:  
  
-   Ein Punkt Präfix hinzugefügt wurde, oder ausgeklammert Direktive aus.  
  
-   Ein reserviertes Wort \(z **C** oder **SIZE**\) wurde als Bezeichner verwendet.  
  
-   Eine Anweisung wurde verwendet, die nicht der aktuelle Prozessor\- und Coprozessor\-Auswahl verfügbar war.  
  
-   Ein Vergleich ablauf `==`Operator \(z\) wurde in einer bedingten Anweisung der Umwandlung anstelle eines relationalen Operators verwendet \(z. B. [EQ](../../assembler/masm/operator-eq.md)\).  
  
-   Eine Anweisung oder \- Direktive wurden zu wenige Operanden angegeben.  
  
-   Veralteten Direktive verwendet wurden.
