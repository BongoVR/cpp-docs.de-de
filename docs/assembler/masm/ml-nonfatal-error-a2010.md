# <a name="ml-nonfatal-error-a2010"></a>Nicht schwerwiegender ML-Fehler A2010
**Ungültiger Ausdruck**  
  
 Der Operand für [THIS](../../assembler/masm/operator-this.md) oder [PTR](../../assembler/masm/operator-ptr.md) ist kein gültiger Typausdruck.
