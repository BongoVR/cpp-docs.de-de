# ML Nonfatal Error A2031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Index\- oder Basisregister sein müssen**  
  
 Es wurde versucht, ein Register zu verwenden, das keine Basis oder ein Indexregister in einem Speicher Ausdruck war.  
  
 Zum Beispiel führen die folgenden Ausdrücke diesen Fehler:  
  
```  
[ax]  
[bl]  
```
