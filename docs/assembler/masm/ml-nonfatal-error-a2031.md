# <a name="ml-nonfatal-error-a2031"></a>Nicht schwerwiegender ML-Fehler A2031
**Index oder Base-Register müssen sein werden.**  
  
 Es wurde versucht, ein Registers verwenden, das kein Basis- oder Indexregister in einem Ausdruck Speicher war.  
  
 Dieser Fehler wird beispielsweise die folgenden Ausdrücke verursacht:  
  
```  
[ax]  
[bl]  
```
