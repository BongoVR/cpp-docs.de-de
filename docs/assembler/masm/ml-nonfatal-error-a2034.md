# ML Nonfatal Error A2034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Segment muss in Datenbindungsausdrücken sein**  
  
 Eines der folgenden Elemente wurde außerhalb eines Segments blocks gefunden:  
  
-   Eine Anweisung  
  
-   Eine Definition der Bezeichnung  
  
-   Ein Operator [DIESES](../../assembler/masm/operator-this.md)  
  
-   Ein Operator [$](../../assembler/masm/dollar.md)  
  
-   Eine Prozedurdefinition  
  
-   [STIMMEN Sie aus](../../assembler/masm/align-masm.md)\-Direktive  
  
-   [ORG](../../assembler/masm/org.md)\-Direktive
