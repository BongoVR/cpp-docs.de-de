# ML Nonfatal Error A2037
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**zulässige innere Strukturdefinition der Anweisung nicht**  
  
 Eine Strukturdefinition enthielt eine ungültige Anweisung.  
  
 Eine Struktur kann Anweisungen, Bezeichnungen, Prozeduren, Ablaufsteuerungs Direktiven, [.STARTUP](../../assembler/masm/dot-startup.md)oder [.EXIT](../../assembler/masm/dot-exit.md)enthalten.
