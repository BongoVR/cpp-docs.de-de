# ML Nonfatal Error A2039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Zeile zu lang**  
  
 Eine Quelldatei überschreitet die Grenze von Zeile 512 Zeichen.  
  
 Wenn mehrere physische Zeilen mit dem Zeilenfortsetzungszeichen \(\\\) verkettet werden, wird die resultierende logische Zeile immer noch auf 512 Zeichen beschränkt.
