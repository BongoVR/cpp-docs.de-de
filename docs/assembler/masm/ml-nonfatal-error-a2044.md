# ML Nonfatal Error A2044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**ungültiges Zeichen in der Datei**  
  
 Die Quelldatei enthält ein Zeichen außerhalb eines Kommentars. einer Zeichenfolge oder eines Literals, das nicht als Operator oder ein anderes Zeichen zugelassenes erkannt wurde.
