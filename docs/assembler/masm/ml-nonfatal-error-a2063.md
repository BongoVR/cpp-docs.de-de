# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**darf nur zur Potenz von 2 ÜBEREINSTIMMEN: Ausdruck**  
  
 Der Ausdruck, der den [STIMMEN Sie aus](../../assembler/masm/align-masm.md)\-Direktive angegeben wurde, war ungültig.  
  
 Der **AUSRICHTEN** Ausdruck muss eine Potenz von 2 zwischen 2 und 256 sein und muss kleiner oder gleich der Ausrichtung des aktuellen Segments, Struktur oder Union sein.
