# ML Nonfatal Error A2069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Für diese Anweisung keine Operanden zulässig**  
  
 Eine oder mehrere Operanden wurden mit einer Anweisung angegeben, die keine Operanden akzeptiert.
