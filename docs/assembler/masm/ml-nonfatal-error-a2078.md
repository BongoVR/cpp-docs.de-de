# ML Nonfatal Error A2078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Anweisung nicht zulässig FAR\-indirekte Adressierung**  
  
 Ein bedingter Bounce\-Fenster bzw. eine Schleife können einen Speicher operanden nicht verwendet werden.  Sie muss angegeben sein eine relative Adresse oder eine Bezeichnung.
