# ML Nonfatal Error A2079
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Anweisung nicht zulässig FAR\-direkte Adressierung**  
  
 Ein bedingter Bounce\-Fenster bzw. eine Schleife können nicht in ein anderes Segment oder einer Gruppe sein.
