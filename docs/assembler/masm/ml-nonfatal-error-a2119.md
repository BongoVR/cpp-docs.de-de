# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Sprachen des Arrays muss angegeben werden**  
  
 Eine Prozedurdefinition oder \- prototyp Sprachen kein Typ angegeben wurden.  
  
 Ein Werttyp Sprachen muss in jeder Prozedurdefinition oder \- prototyp deklariert werden, wenn ein standardmäßiges Sprachen Typ nicht angegeben ist.  Der Standardwert ist " Sprachen entweder mit der [.MODEL](../../assembler/masm/dot-model.md)\-Direktive, **OPTION LANG**oder der ML\-Befehlszeilenoptionen **\/Gc** oder **\/Gd**festgelegt.
