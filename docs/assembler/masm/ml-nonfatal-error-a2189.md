# <a name="ml-nonfatal-error-a2189"></a>Nicht schwerwiegender ML-Fehler A2189
**Ungültige Kombination mit der Segment-Ausrichtung**  
  
 Die Ausrichtung, die gemäß der **AUSRICHTEN** oder `EVEN` Richtlinie war größer als das aktuelle Segment Ausrichtung entsprechend den Angaben von der **SEGMENT** Richtlinie.
