# ML Nonfatal Error A2189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Ungültige Kombination mit Ausrichtung von Segment**  
  
 Die Ausrichtung, die von der **AUSRICHTEN** oder `EVEN`\-Direktive angegeben wurde, war größer als die aktuelle Segment, wie durch die Ausrichtung **SEGMENT**\-Direktive angegeben.
