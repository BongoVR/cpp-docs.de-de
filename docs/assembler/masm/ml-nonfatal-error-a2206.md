# ML Nonfatal Error A2206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**fehlender Operator im Ausdruck**  
  
 Ein Ausdruck kann nicht ausgewertet werden, weil er einen Operator fehlt.  Diese Fehlermeldung kann auch ein Nebeneffekt eines vorangehenden Programm fehlgeschlagene.  
  
 Die folgende Zeile generiert diesen Fehler:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
