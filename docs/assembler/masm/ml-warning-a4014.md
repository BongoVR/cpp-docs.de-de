# ML Warning A4014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Anweisungen und initialisierte Daten in BSS\-Segmente nicht unterstützt  
  
 Es wurde versucht, initialisierte Daten innerhalb eines BSS\-Abschnitts zu definieren.  Ein BSS\-Abschnitt wird wie eine Klasse definiert, deren Name BSS ist.  Dies schließt das vereinfachte Segment `.data?`ein.
