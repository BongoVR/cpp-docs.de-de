# operator +
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Der erste Operator gibt *expression1* plus *expression2 zurück*.  Der zweite Operator gibt das Zeichen *des Ausdrucks*.  
  
## Syntax  
  
```  
  
      expression1 + expression2  
–expression  
```
