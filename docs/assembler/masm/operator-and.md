# operator AND
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt das Ergebnis der bitweisen AND\-Operation für *expression1* und *expression2*zurück.  
  
## Syntax  
  
```  
  
expression1 AND expression2  
```
