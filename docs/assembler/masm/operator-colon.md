# operator :
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Überschreibt das Standardverhalten für Segment *des Ausdrucks* mit `segment`.  `segment` kann kein Segmentregister, einen Gruppennamen, oder ein Segment des Segments ein Ausdruck sein.  Der *Ausdruck* muss eine Konstante sein.  
  
## Syntax  
  
```  
  
segment  
: expression  
```
