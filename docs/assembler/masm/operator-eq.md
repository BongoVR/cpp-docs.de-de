# <a name="operator-eq"></a>Operator EQ
Gibt "true" (-1) zurück, wenn *expression1* gleich *expression2*, oder gibt "false" (0) zurück, wenn dies nicht der Fall.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
expression1 EQ expression2  
```
