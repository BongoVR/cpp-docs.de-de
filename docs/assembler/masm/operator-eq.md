# operator EQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt true \(– 1\), wenn *expression1* *expression2*ist, zurück oder gibt false zurück \(0\).  
  
## Syntax  
  
```  
  
expression1 EQ expression2  
```
