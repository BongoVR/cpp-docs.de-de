# operator &gt;= (MASM Run Time)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ist größer als oder gleich.  Wird nur in [.IF](../../assembler/masm/dot-if.md), [.WHILE](../../assembler/masm/dot-while.md)oder [.REPEAT](../../assembler/masm/dot-repeat.md) Blöcke und zur Laufzeit ausgewertet, nicht mit der Montagezeit.  
  
## Syntax  
  
```  
  
expression1 >= expression2  
```
