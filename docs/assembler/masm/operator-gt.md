# operator GT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt true zurück, wenn \(– 1\) ist größer als *expression1* *expression2*zurück oder gibt false zurück \(0\), wenn dies nicht der Fall ist.  
  
## Syntax  
  
```  
  
expression1 GT expression2  
```
