# operator HIGH
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt die oberen 8 Bits der unteren 16 Bits *eines Ausdrucks*zurück.  MASM\-Ausdrücke sind 64\-Bit\-Werte.  
  
## Syntax  
  
```  
  
HIGH expression  
```
