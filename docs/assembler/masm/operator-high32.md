# operator HIGH32
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt die oberen 32 Bits *eines Ausdrucks*zurück.  MASM\-Ausdrücke sind 64\-Bit\-Werte.  
  
## Syntax  
  
```  
  
HIGH32 expression  
```
