# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt die oberen 16 Bits der unteren 32 Bits *eines Ausdrucks*zurück.  MASM\-Ausdrücke sind 64\-Bit\-Werte.  
  
## Syntax  
  
```  
  
HIGHWORD expression  
```
