# <a name="operator-imagerel"></a>Operator IMAGEREL
Gibt das Bild relativen Offset `expression`.  
  
## <a name="syntax"></a>Syntax  
  
```  
IMAGEREL expression  
```  
  
## <a name="remarks"></a>Hinweise  
 Der resultierende Wert wird häufig als eine RVA oder Relative virtuelle Adresse bezeichnet.  
  
 IMAGEREL ist nur mit COFF-Objekt prädikatauswertung verfügbar.
