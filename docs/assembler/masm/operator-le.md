# operator LE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt true \(– 1\), wenn *expression1* kleiner oder gleich *expression2*ist, zurück oder gibt false zurück \(0\), wenn dies nicht der Fall ist.  
  
## Syntax  
  
```  
  
expression1 LE expression2  
```
