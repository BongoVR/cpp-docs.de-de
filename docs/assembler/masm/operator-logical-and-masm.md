# operator &amp;&amp; (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ersetzt `parameter` durch ihren entsprechenden Argumentwert.  
  
## Syntax  
  
```  
  
&  
parameter  
&  
  
```
