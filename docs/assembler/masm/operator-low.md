# operator LOW
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt die unteren 8 Bits *eines Ausdrucks*zurück.  MASM\-Ausdrücke sind 64\-Bit\-Werte.  
  
## Syntax  
  
```  
  
LOW expression  
```
