# operator LT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt true \(– 1\), wenn *expression1* *expression2*ist kleiner als " zurück oder gibt false zurück \(0\), wenn dies nicht der Fall ist.  
  
## Syntax  
  
```  
  
expression1 LT expression2  
```
