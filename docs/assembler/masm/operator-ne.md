# operator NE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt true \(– 1\), wenn keine *expression1* *expression2*ist, zurück oder gibt false zurück \(0\) Wenn dies der Fall ist.  
  
## Syntax  
  
```  
  
expression1 NE expression2  
```
