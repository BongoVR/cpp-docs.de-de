# <a name="operator-opattr"></a>Operator OPATTR
Gibt ein Wort, definieren den Modus und den Bereich der *Ausdruck*. Das niedrige Byte ist identisch mit dem Byte zurückgegebenes [. Typ](../../assembler/masm/operator-dot-type.md). Das höherwertige Byte enthält zusätzliche Informationen.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
OPATTR expression  
```
