# operator PARITY?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Status des Paritäts Flags.  Wird nur in [.IF](../../assembler/masm/dot-if.md), [.WHILE](../../assembler/masm/dot-while.md)oder [.REPEAT](../../assembler/masm/dot-repeat.md) Blöcke und zur Laufzeit ausgewertet, nicht mit der Montagezeit.  
  
## Syntax  
  
```  
  
PARITY?  
  
```
