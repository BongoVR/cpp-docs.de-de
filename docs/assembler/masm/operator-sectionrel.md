# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt den Offset des Ausdrucks im Hinblick Abschnitts relativ zum Bereich zurück, der das Ziel in die endgültige ausführbare Datei enthält.  
  
## Syntax  
  
```  
SECTIONREL expression  
```  
  
## Hinweise  
 SECTIONREL ist nur mit COFF\-Objektemission verfügbar.
