# <a name="operator-"></a>Operator ;;
Behandelt *Text* als Kommentar in einem Makro, das nur in der Makrodefinition angezeigt wird. Die Liste zeigt nicht die *Text* , wo das Makro erweitert wird.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
;;  
text  
  
```
