# operator ;;
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Behandelt *Text* als Kommentar in einem Makro, das nur in Makrodefinition angezeigt wird.  Die Auflistung wird *Text* nicht an *,* wo das Makro erweitert wird.  
  
## Syntax  
  
```  
  
;;  
text  
  
```
