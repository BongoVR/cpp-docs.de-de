# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Legt die Kennsatzart um short fest.  Alle *beschriftet* Sprünge müssen kurz sein \(innerhalb des Bereichs 128 bis \+127 Bytes aus dem Sprungbefehl *beschriftet*\).  
  
## Syntax  
  
```  
  
SHORT label  
```
