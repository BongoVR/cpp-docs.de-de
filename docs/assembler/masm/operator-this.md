# operator THIS
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt einen Operanden aus dem angegebenen `type` zurück, dessen Offset\- und Segment Werte gleich den Zählerwert des aktuellen Verzeichnisses befinden.  
  
## Syntax  
  
```  
  
THIS type  
```
