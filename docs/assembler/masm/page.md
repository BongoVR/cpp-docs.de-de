# PAGE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die ersten Direktive legen *Zeilenlänge* der Zeichenbreite und Listen Programm fest.  Wenn keine Argumente angegeben werden, generiert einen Seitenumbruch.  Die zweite Direktive erhöhen die Nummer des Abschnitts und die Seitenzahl auf 1 zurück.  
  
## Syntax  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
