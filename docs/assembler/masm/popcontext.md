# POPCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Stellt den Teil oder alle aktuellen `context``context``context` \(durch die [PUSHCONTEXT](../../assembler/masm/pushcontext.md)\-Direktive\) gespeichert.  `context` kann **ASSUMES**, `RADIX`, **LISTING**, **CPU**oder **ALLE**sein.  
  
## Syntax  
  
```  
  
POPCONTEXT context  
```
