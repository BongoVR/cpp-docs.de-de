# <a name="popcontext"></a>POPCONTEXT
Die aktuelle ganz oder teilweise wiederhergestellt `context` (Einsparung/der [PUSHCONTEXT](../../assembler/masm/pushcontext.md) Richtlinie). Die `context` kann **setzt**, `RADIX`, **auflisten**, **CPU**, oder **alle**.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
POPCONTEXT context  
```
