# PUBLIC (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Legt jede Variable Bezeichnung oder absolute Symbol, das als *Name* angegeben wird, der für alle anderen Module im Programm verfügbar ist.  
  
## Syntax  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
