# QWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional 8 Bytes Speicherplatz für jedes `initializer`.  Kann als Typspezifizierer auch dazu verwendet werden, eine beliebige Stelle, das ein Typ gültig ist.  
  
## Syntax  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
