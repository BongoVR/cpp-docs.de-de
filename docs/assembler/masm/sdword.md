# SDWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional ein Doppelwort mit Vorzeichen \(4 Bytes\) des Arbeitsspeichers für jedes `initializer`.  Kann als Typspezifizierer auch dazu verwendet werden, eine beliebige Stelle, das ein Typ gültig ist.  
  
## Syntax  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
