# STRUCT (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Deklariert einen Strukturtyp, die die angegebenen *fielddeclarations*verfügt.  Jedes Feld muss eine gültige Datendefinition sein.  Identisch mit [STRUC](../../assembler/masm/struc.md).  
  
## Syntax  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
