# SUBSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Gibt eine Teilzeichenfolge der *textitem*zurück, wobei an der *Position*.  Das *textitem* eine Literalzeichenfolge kann eine Konstante, die von `%`vorangestellt werden, oder die Zeichenfolge sein, die durch eine Makrofunktion zurückgegeben wurde.  
  
## Syntax  
  
```  
  
name SUBSTR textitem, position [[, length]]  
```
