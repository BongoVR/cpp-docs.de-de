# TBYTE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ordnet zu und initialisiert optional 10 Bytes Speicherplatz für jedes `initializer`.  Kann auch verwendet werden, da ein Typspezifizierer wo ein Typ gültig ist.  
  
## Syntax  
  
```  
[[name]] TBYTE initializer [[, initializer]]...  
```
