# <a name="textequ"></a>TEXTEQU
Weist *Textitem* auf *Namen*. Die *Textitem* kann ein Zeichenfolgenliteral, eine Konstante vorangestellt werden eine `%`, oder die Zeichenfolge, die von einem Makrofunktion zurückgegeben wird.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
name TEXTEQU [[textitem]]  
```
