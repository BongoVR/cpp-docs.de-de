# TYPEDEF (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Definiert einen neuen Typ mit dem *Namen*aufgerufen wird, der zu `type`entspricht.  
  
## Syntax  
  
```  
  
name TYPEDEF type  
```
