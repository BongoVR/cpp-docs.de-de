# WHILE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Wiederholt Assembly des Blocks `statements` , solange *Ausdruck* true ist.  
  
## Syntax  
  
```  
  
   WHILE expression  
statements  
ENDM  
```
