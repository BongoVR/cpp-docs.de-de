# YMMWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Wird für Multimedia Operanden von 256\-Bit\- mit Intel\- Vektor\-Erweiterungen wechselten MSIL\-Anweisungen \(AVX\).  
  
## Syntax  
  
```  
YMMWORD  
```  
  
## Hinweise  
 `YMMWORD` muss den gleichen Typ wie `__m256` für die AVX\-systeminternen Komponenten darzustellen.
