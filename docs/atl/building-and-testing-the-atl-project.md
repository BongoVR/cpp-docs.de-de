# Building and Testing the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Wie in erwähnt, [Einfügen eines zusammengesetzten Steuerelements](../atl/inserting-a-composite-control.md) ist eine der ursprünglichen Komponenten des Projekts eine standardmäßige HTML\-Seite, die das neue zusammengesetztes Steuerelement hostet.  Nachdem Sie beenden, das zusammengesetzte Steuerelement zu ändern, klicken Sie auf **Projektmappe erstellen** oder auf **Rebuild Solution** vom Menü **Erstellen**.  Sobald laden die des Projekts Builds erfolgreich, die HTML\-Seite, im Stammverzeichnis des Projekts eines zusammengesetzten Steuerelements, in Internet Explorer oder in einen anderen Browser testen und die Funktionen des Steuerelements.  
  
 Sie können das zusammengesetzte Steuerelement mithilfe des Testcontainertools oder jeder anderen Anwendung auch testen, die ein ActiveX\-Steuerelement hosten kann.  Informationen zum Zugreifen auf den Testcontainer finden Sie unter [Testen von Eigenschaften und Ereignissen mit dem Testcontainer](../mfc/testing-properties-and-events-with-test-container.md).
