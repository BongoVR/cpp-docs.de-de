# Class Factories Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgenden Klassen implementieren oder eine Klassenfactory unterstützen:  
  
-   [CComClassFactory](../atl/reference/ccomclassfactory-class.md) stellt eine Standardklassenfactory für die Objekterstellung bereit.  
  
-   [CComClassFactory2](../atl/reference/ccomclassfactory2-class.md)\-Steuerelemente gelten Erstellung durch eine Lizenz ein.  
  
-   [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) ermöglicht die in mehreren Apartments erstellt werden, Objekte, mit Threadpool.  
  
-   [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) erstellt ein einzelnes Objekt.  
  
-   [CComCoClass](../atl/reference/ccomcoclass-class.md) definiert die Klassenfactory für das Objekt.
