# Composite Controls Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgende Klasse bietet Unterstützung für das Erstellen von zusammengesetzten Steuerelementen  
  
-   Steuerelemente [CComCompositeControl](../atl/reference/ccomcompositecontrol-class.md) ActiveX, die von `CComCompositeControl` abgeleitet werden, wird durch ein Standarddialogfeld gehostet.  Diese Typen von Steuerelementen werden zusammengesetzte Steuerelemente aufgerufen, da sie in der Lage sind, andere Steuerelemente zu hosten \(systemeigene Windows\-Steuerelement\- und ActiveX\-Steuerelemente\).
