# <a name="control-containment-classes"></a>Containment-Steuerelementklassen
Die folgenden Klassen bieten Containment-Unterstützung für das Hosten von Steuerelementen an:  
  
-   [CAxWindow](../atl/reference/caxwindow-class.md) stellt Methoden zum Bearbeiten von ein Fenster, das ein ActiveX-Steuerelement hostet.  
  
-   [CAxWindow2T](../atl/reference/caxwindow2t-class.md) bietet Methoden zum Bearbeiten von ein Fenster, das ein ActiveX-Steuerelement gehostet und bietet auch Unterstützung für das lizenzierten ActiveX-Steuerelemente hosten.  
  
-   [IAxWinAmbientDispatch](../atl/reference/iaxwinambientdispatch-interface.md) rufen die Methoden für diese Schnittstelle für eine gehostete Steuerelement ambient verfügbaren Eigenschaften fest.  
  
-   [IAxWinHostWindow](../atl/reference/iaxwinhostwindow-interface.md) rufen Sie die Methoden für diese Schnittstelle zum Erstellen und/oder fügen Sie ein Steuerelement an ein Objekt oder eine Schnittstelle von einem gehosteten Steuerelement abzurufen.
