# Control Containment Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgenden Klassen bieten Kapselungsunterstützung zum Hosten von Steuerelementen:  
  
-   [CAxWindow](../atl/reference/caxwindow-class.md) stellt Methoden zum Bearbeiten eines Fensters bereit, das ein ActiveX\-Steuerelement hostet.  
  
-   [CAxWindow2T](../atl/reference/caxwindow2t-class.md) stellt Methoden zum Bearbeiten eines Fensters bereit, das ein ActiveX\-Steuerelement hostet und auch Unterstützung für das Hosten von lizenzierten ActiveX\-Steuerelemente verfügt.  
  
-   [IAxWinAmbientDispatch](../atl/reference/iaxwinambientdispatch-interface.md) Aufruf, den die Methoden auf diesem herstellen, um die Ambient\-Eigenschaften festzulegen, die einem gehosteten Steuerelement verfügbar sind.  
  
-   [IAxWinHostWindow](../atl/reference/iaxwinhostwindow-interface.md) Aufruf die Methoden für diese Schnittstelle, um ein Steuerelement zu einem Hostobjekt zu erstellen und\/oder anzufügen, oder eine Schnittstelle aus einem gehosteten Steuerelement abzurufen.
