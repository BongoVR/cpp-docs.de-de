# <a name="data-transfer-classes"></a>Übertragung Datenklassen
Die folgenden Klassen unterstützen verschiedene Typen der Datenübertragung an:  
  
-   [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) unterstützt Uniform Data Transfer zum Abrufen und Festlegen von Daten mithilfe von Standardformaten. Handles Änderungsdaten Benachrichtigungen durch die Verwaltung von Verbindungen, um die advise-senken.  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) ermöglicht eine asynchrone Moniker zum Senden und Empfangen von Informationen über die asynchrone Datenübertragung in und aus Ihr Objekt.
