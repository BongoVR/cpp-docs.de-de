# Error Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgende Klasse gibt an, wie Fehlerinformationen bearbeitet werden:  
  
-   [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) bestimmt, ob das Objekt die [IErrorInfo](assetId:///4dda6909-2d9a-4727-ae0c-b5f90dcfa447)\-Schnittstelle unterstützt.  **IErrorInfo** ermöglicht die zurück an den Client weitergegeben werden sollen Fehlerinformationen.
