# File Handling Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Diese Klassen stellen Methoden zum Behandeln von Dateien, temporärer Dateien und von Speicherabbilddateien bereit.  
  
-   [CAtlFile](../atl/reference/catlfile-class.md) diese Klasse stellt einen dünnen Wrapper um die Windows\-DateiBehandlung API bereit.  
  
-   [CAtlFileMapping](../atl/reference/catlfilemapping-class.md) diese Klasse stellt eine Speicherabbilddatei dar und fügt einen Typumwandlungsoperator an Methoden von [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md) hinzu.  
  
-   [CAtlFileMappingBase](../atl/reference/catlfilemappingbase-class.md) diese Klasse stellt eine Speicherabbilddatei dar.  
  
-   [CAtlTemporaryFile](../atl/reference/catltemporaryfile-class.md) diese Klasse stellt Methoden für die Erstellung und Verwendung einer temporären Datei.
