# <a name="modifying-the-atl-project"></a>Ändern des ATL-Projekts
An diesem Punkt implementiert zusammengesetzte Steuerelementprojekt die notwendigen Objekte für Ihrer zusammengesetzten Steuerelements. Der nächste Schritt ist zum Hinzufügen der Steuerelemente, die zusammengesetzten Steuerelements enthalten soll, und alle notwendigen Ereignisse verarbeiten.  
  
 Um zusätzliche ActiveX oder einem Windows-Steuerelemente hinzuzufügen, fügen Sie ein neues Ressourcenskript hinzu, und verwenden Sie den Dialog-Editor. Weitere Informationen zum Hinzufügen von Steuerelementen (und Verwandte Aufgaben) finden Sie unter [Dialog-Editor](../windows/dialog-editor.md).  
  
 Behandeln Sie alle erforderlichen Ereignisse vom ActiveX-Steuerelementen finden Sie unter [Hinzufügen neuer Funktionen auf dem zusammengesetzten Steuerelements](../atl/adding-functionality-to-the-composite-control.md).
