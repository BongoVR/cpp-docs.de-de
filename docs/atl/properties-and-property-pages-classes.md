# Properties and Property Pages Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgenden Klassenstützeigenschaften und Eigenschaftenseiten:  
  
-   [CComDispatchDriver](../Topic/CComDispatchDriver.md) ruft ab oder legt die Eigenschaften eines Objekts durch einen Zeiger `IDispatch` fest.  
  
-   [CStockPropImpl](../atl/reference/cstockpropimpl-class.md) implementiert die vordefinierten Eigenschaften, die von ATL unterstützt werden.  
  
-   [IPerPropertyBrowsingImpl](../atl/reference/iperpropertybrowsingimpl-class.md) greift auf die Informationen in den Eigenschaftenseiten eines Objekts zu.  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) speichert die Eigenschaften eines Objekts in einer Client\-angegebenen Eigenschaftensammlung.  
  
-   [IPropertyPageImpl](../atl/reference/ipropertypageimpl-class.md) verwaltet eine bestimmte Eigenschaftenseite innerhalb eines Eigenschaftenblatts.  
  
-   [IPropertyPage2Impl](../atl/reference/ipropertypage2impl-class.md), das zu `IPropertyPageImpl`, lässt jedoch auch ähnelt, einem Client, um eine bestimmte Eigenschaft in einer Eigenschaftenseite auszuwählen.  
  
-   [ISpecifyPropertyPagesImpl](../atl/reference/ispecifypropertypagesimpl-class.md) erhält das CLSID für die Eigenschaftenseiten, die von einem Objekt unterstützt werden.
