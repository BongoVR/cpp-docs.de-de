# CAtlDebugInterfacesModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Klasse stellt Unterstützung für Debugschnittstellen.  
  
## Syntax  
  
```  
  
class CAtlDebugInterfacesModule  
  
```  
  
## Hinweise  
 `CAtlDebugInterfacesModule` bietet die unterstützt, die für Debugschnittstellen benötigt wird.  Es wird in jedem Projekt enthalten, das Symbol [\_ATL\_DEBUG\_QI](../Topic/_ATL_DEBUG_QI.md) definiert.  
  
## Anforderungen  
 **Header:** atlbase.h
