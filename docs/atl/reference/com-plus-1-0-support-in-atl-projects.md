# <a name="com-10-support-in-atl-projects"></a>COM+ 1.0-Unterstützung in ATL-Projekte
Sie können die [ATL-Projekt-Assistent](../../atl/reference/creating-an-atl-project.md) zum Erstellen eines Projekts mit grundlegende Unterstützung für COM+ 1.0-Komponenten.  
  
 COM+ 1.0 ist für die Anwendungsentwicklung verteilten Komponenten basierten konzipiert. Darüber hinaus eine Laufzeitinfrastruktur zum Bereitstellen und Verwalten der Anwendung.  
  
 Bei Auswahl der **COM+ 1.0-Unterstützung** Kontrollkästchen der Assistent das Buildskript während des Verknüpfungsvorgangs ändert. Insbesondere das COM+ 1.0 Verknüpfungen auf die folgenden Bibliotheken:  
  
-   comsvcs.lib hinzu:  
  
-   Mtxguid.lib  
  
 Bei Auswahl der **COM+ 1.0-Unterstützung** , Sie können auch Kontrollkästchen **Unterstützung Komponente Registrierungsstelle**. Die Registrierung der Komponente ermöglicht das COM+ 1.0-Objekt zum Abrufen einer Liste der Komponenten, Komponenten registriert oder Registrierung von Komponenten (einzeln oder alle auf einmal).
