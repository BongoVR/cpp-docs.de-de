# Composite Control Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Makros definieren Ereignissenkenzuordnungen und Einträge.  
  
|||  
|-|-|  
|[BEGIN\_SINK\_MAP](../Topic/BEGIN_SINK_MAP.md)|Markiert den Beginn der Ereignissenkenzuordnung für das zusammengesetzte Steuerelement.|  
|[END\_SINK\_MAP](../Topic/END_SINK_MAP.md)|Markiert das Ende der Ereignissenkenzuordnung für das zusammengesetzte Steuerelement.|  
|[SINK\_ENTRY](../Topic/SINK_ENTRY.md)|Eintrag zur Ereignissenkenzuordnung.|  
|[SINK\_ENTRY\_EX](../Topic/SINK_ENTRY_EX.md)|Eintrag zur Ereignissenkenzuordnung mit einem zusätzlichen Parameter.|  
|[SINK\_ENTRY\_INFORMATION](../Topic/SINK_ENTRY_INFO.md)|Eintrag zur Ereignissenkenzuordnung mit manuell angegebenen Typinformationen für die Verwendung mit [IDispEventSimpleImpl](../../atl/reference/idispeventsimpleimpl-class.md).|
