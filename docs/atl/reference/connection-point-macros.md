# Connection Point Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Makros definieren Verbindungspunktzuordnungen und \-Einträge.  
  
|||  
|-|-|  
|[BEGIN\_CONNECTION\_POINT\_MAP](../Topic/BEGIN_CONNECTION_POINT_MAP.md)|Markiert den Beginn der Verbindungspunktzuordnungseinträge.|  
|[CONNECTION\_POINT\_ENTRY](../Topic/CONNECTION_POINT_ENTRY.md)|Gibt Verbindungspunkte in die Zuordnung ein.|  
|[END\_CONNECTION\_POINT\_MAP](../Topic/END_CONNECTION_POINT_MAP.md)|Markiert das Ende der Verbindungspunktzuordnungseinträge.|
