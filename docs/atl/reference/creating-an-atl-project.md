# Erstellen eines ATL-Projekts
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die einfachste Möglichkeit, ein ATL\-Projekt zu erstellen, besteht in der Verwendung des ATL\-Projekt\-Assistenten. Dieser Assistent befindet sich im Win32\-Projektordner im Dialogfeld **Neues Projekt**.  
  
### So erstellen Sie ein ATL\-Projekt mit dem ATL\-Projekt\-Assistenten  
  
1.  Folgen Sie den Anweisungen im Hilfethema [Erstellen eines Projekts mit einem Visual C\+\+\-Anwendungs\-Assistenten](../../ide/creating-desktop-projects-by-using-application-wizards.md).  
  
2.  Wählen Sie im Bereich Vorlagen das Symbol **ATL\-Projekt** aus, um den ATL\-Projekt\-Assistenten zu öffnen.  
  
3.  Definieren Sie die Anwendungseinstellungen auf der Seite [Anwendungseinstellungen](../../atl/reference/application-settings-atl-project-wizard.md) des `ATL Project Wizard`.  
  
    > [!NOTE]
    >  Überspringen Sie diesen Schritt, um die Standardeinstellungen des Assistenten beizubehalten.  
  
4.  Klicken Sie auf **Fertig stellen**, um den Assistenten zu schließen und das neue Projekt in der Entwicklungsumgebung zu öffnen.  
  
 Nachdem das Projekt erstellt wurde, können Sie die generierten Dateien im **Projektmappen\-Explorer** anzeigen lassen.  Weitere Informationen zu den vom Assistenten erstellten Dateien im Projekt finden Sie in der projekteigenen Datei "Readme.txt".  Weitere Informationen zu den Dateitypen finden Sie unter [Für Visual C\+\+\-Projekte erstellte Dateitypen](../../ide/file-types-created-for-visual-cpp-projects.md).  Weitere Informationen zu den Konfigurationen für das neue ATL\-Projekt sowie zum Ändern dieser Konfigurationen finden sie unter [Standardmäßige ATL\-Projektkonfigurationen](../../atl/reference/default-atl-project-configurations.md).
