# IAtlAutoThreadModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Klasse stellt eine Schnittstelle zu einer `CreateInstance`\-Methode dar.  
  
> [!IMPORTANT]
>  Diese Klasse und ihre Member können in Anwendungen nicht verwendet werden, die in [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)] ausführen.  
  
## Syntax  
  
```  
  
__interface IAtlAutoThreadModule  
  
```  
  
## Hinweise  
 Die Klasse [CAtlAutoThreadModuleT](../../atl/reference/catlautothreadmodulet-class.md) wird von `IAtlAutoThreadModule` abgeleitet, mit dem sie, um Code für das Erstellen eines Objekts und das Abrufen eines Schnittstellenzeigers bereitzustellen.
