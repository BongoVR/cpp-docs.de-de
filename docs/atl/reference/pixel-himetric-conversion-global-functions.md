# Pixel/HIMETRIC Conversion Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Funktionen bieten Unterstützung für die Konvertierung von und in Pixel und HIMETRIC\-Einheiten.  
  
> [!IMPORTANT]
>  Die Funktionen, die in der folgenden Tabelle aufgeführt sind, können nicht in Anwendungen verwendet werden, die in [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)] ausführen.  
  
|||  
|-|-|  
|[AtlHiMetricToPixel](../Topic/AtlHiMetricToPixel.md)|Bekehrt\-HIMETRIC\-Einheiten \(jede Einheit ist 0,01 mm\), in Pixel.|  
|[AtlPixelToHiMetric](../Topic/AtlPixelToHiMetric.md)|Bekehrtpixel zu den HIMETRIC\-Einheiten \(jede Einheit ist 0,01 mm\).|
