# Snap-In Object Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese Makros bieten Unterstützung für Snap\-Inerweiterungen.  
  
|||  
|-|-|  
|[BEGIN\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/BEGIN_EXTENSION_SNAPIN_NODEINFO_MAP.md)|Markiert den Beginn der Snap\-In\-Erweiterungsdatenklasse\-Zuordnung für ein Snap\-Inobjekt.|  
|[BEGIN\_SNAPINTOOLBARID\_MAP](../Topic/BEGIN_SNAPINTOOLBARID_MAP.md)|Markiert den Beginn der Symbolleistenzuordnung für ein Snap\-Inobjekt.|  
|[END\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/END_EXTENSION_SNAPIN_NODEINFO_MAP.md)|Markiert das Ende der Snap\-In\-Erweiterungsdatenklasse\-Zuordnung für ein Snap\-Inobjekt.|  
|[END\_SNAPINTOOLBARID\_MAP](../Topic/END_SNAPINTOOLBARID_MAP.md)|Markiert das Ende der Symbolleistenzuordnung für ein Snap\-Inobjekt.|  
|[EXTENSION\_SNAPIN\_DATACLASS](../Topic/EXTENSION_SNAPIN_DATACLASS.md)|Erstellt einen Datenmember für die Datenklasse der Snap\-Inerweiterung.|  
|[EXTENSION\_SNAPIN\_NODEINFO\_ENTRY](../Topic/EXTENSION_SNAPIN_NODEINFO_ENTRY.md)|Gibt ein Snap\-In\-Erweiterungsdatenklasse in die Snap\-In\-Erweiterungsdatenklasse\-Zuordnung des Snap\-Inobjekts ein.|  
|[SNAPINMENUID](../Topic/SNAPINMENUID.md)|Deklariert die ID des Kontextmenüs, das vom Snap\-Inobjekt verwendet wird.|  
|[SNAPINTOOLBARID\_ENTRY](../Topic/SNAPINTOOLBARID_ENTRY.md)|Gibt eine Symbolleiste in die Symbolleistenzuordnung des Snap\-Inobjekts ein.|
