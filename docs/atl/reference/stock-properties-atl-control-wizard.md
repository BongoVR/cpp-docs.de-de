# Basiseigenschaften, ATL-Steuerelement-Assistent
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Auf dieser Seite des Assistenten werden die für das Steuerelement unterstützten Basiseigenschaften angegeben.  Standardmäßig sind keine Eigenschaften festgelegt.  
  
 **Nicht unterstützt**  
 Gibt die verfügbaren Eigenschaften an, die derzeit nicht für das Steuerelement unterstützt werden.  
  
 **Unterstützt**  
 Gibt die Eigenschaften an, die derzeit für das Steuerelement unterstützt werden.  
  
|Übertragungsschaltfläche|Description|  
|------------------------------|-----------------|  
|**\>**|Fügt der Liste **Unterstützt** die derzeit in der Liste **Nicht unterstützt** markierte Eigenschaft hinzu.|  
|**\>\>**|Fügt der Liste **Unterstützt** alle in der Liste **Nicht unterstützt** verfügbaren Eigenschaften hinzu.|  
|**\<**|Entfernt die derzeit markierte Eigenschaft aus der Liste **Unterstützt**.|  
|**\<\<**|Entfernt alle derzeit in der Liste **Unterstützt** enthaltenen Eigenschaften aus der Liste.|
