# <a name="scope-of-atl"></a>Geltungsbereich von ATL
ATL ermöglicht die einfache Erstellung von COM-Objekten, Automatisierungsserver und ActiveX-Steuerelemente. ATL stellt integrierten Unterstützung für viele grundlegenden COM-Schnittstellen.  
  
 ATL wird als Quellcode ausgeliefert, die Sie in der Anwendung einschließen. ATL stellt außerdem eine DLL zur Verfügung (atl90.dll), die Code enthält, die in Komponenten gemeinsam genutzt werden kann. Diese DLL ist jedoch nicht erforderlich.
