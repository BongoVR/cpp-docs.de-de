# Site Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgenden Klassen können einem Objekt, um die Site zu kommunizieren:  
  
-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) ruft ab und legt einen Zeiger auf die Site eines Objekts fest.  Wird für Objekte, die keine Steuerelemente sind.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) ruft ab und legt einen Zeiger auf die Site eines Objekts fest.  Wird für Steuerelemente.
