# ASCII-Zeichensatz
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.1** Member der Quell\- und Ausführungszeichensätze  
  
 Beim Quellzeichensatz handelt es sich um einen Satz zulässiger Zeichen, die in den Quelldateien auftreten können.  In Microsoft C ist der Quellzeichensatz der ASCII\-Standardzeichensatz.  
  
> [!NOTE]
>  **Warnung** Da Tastatur\- und Konsolentreiber den Zeichensatz neu zuordnen können, sollten Programme, die für den internationalen Vertrieb vorgesehen sind, den Länder\-\/Regionscode überprüfen.
