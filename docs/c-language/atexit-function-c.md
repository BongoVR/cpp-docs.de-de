# atexit-Funktion (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.3** Der von der `atexit`\-Funktion zurückgegebene Zustand, wenn der Wert des Arguments nicht Null, EXIT\_SUCCESS oder EXIT\_FAILURE lautet  
  
 Die `atexit`\-Funktion gibt Null zurück, wenn der Vorgang erfolgreich war, sonst einen Wert ungleich Null.
