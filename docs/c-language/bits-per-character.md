# Bits pro Zeichen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2.1** Anzahl von Bits in einem Zeichen  
  
 Die Anzahl von Bits in einem Zeichen wird durch die Manifestkonstante **CHAR\_BIT** dargestellt.  Die LIMITS.H\-Datei definiert **CHAR\_BIT** als 8.
