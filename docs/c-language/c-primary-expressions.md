# C-Ausdr&#252;cke (prim&#228;r)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die Operanden in Ausdrücken werden als "primäre Ausdrücke" bezeichnet.  
  
## Syntax  
 *primary\-expression*:  
 *identifier*  
  
 *constant*  
  
 *string\-literal*  
  
 **\(**  *expression*  **\)**  
  
 *expression*:  
 *assignment\-expression*  
  
 *expression*  **,**  *assignment\-expression*
