# <a name="c-unary-operators"></a>C-Operatoren (unär)
Unäre Operatoren stehen vor ihrem Operanden und werden von rechts nach links zugeordnet.  
  
## <a name="syntax"></a>Syntax  
 *unary-expression*:  
 *postfix-expression*  
  
 **++**  *unary-expression*  
  
 `--` *unary-expression*  
  
 *unary-operator cast-expression*  
  
 **sizeof**  *unary-expression*  
  
 **sizeof (**  *type-name*  **)**  
  
 *unary-operator*: eines der folgenden Zeichen  
 **& \* + -** `~` **!**
