# C-Operatoren (un&#228;r)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Unäre Operatoren stehen vor ihrem Operanden und werden von rechts nach links zugeordnet.  
  
## Syntax  
 *unary\-expression*:  
 *postfix\-expression*  
  
 **\+\+**  *unary\-expression*  
  
 `--`  *unary\-expression*  
  
 *unary\-operator cast\-expression*  
  
 **sizeof**  *unary\-expression*  
  
 **sizeof \(**  *type\-name*  **\)**  
  
 *unary\-operator*: einer von:  
 **&**  
 **\* \+ –** `~` **\!**
