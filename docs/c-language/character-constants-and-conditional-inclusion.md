# Zeichenkonstanten und bedingter Einschluss
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.8.1** Ob der Wert einer Einzelzeichen\-Zeichenkonstante in einem konstanten Ausdruck, der den bedingten Einschluss steuert, mit dem Wert der gleichen Zeichenkonstante im Ausführungszeichensatz übereinstimmt.  Ob eine solche Zeichenkonstante einen negativen Wert haben kann  
  
 Der Zeichensatz, der in Präprozessoranweisungen verwendet wird, ist mit dem Ausführungszeichensatz identisch.  Der Präprozessor erkennt negative Zeichenwerte.
