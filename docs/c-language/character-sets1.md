# Zeichens&#228;tze
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.3.4** Zuordnen von Membern des Quellzeichensatzes  
  
 Der Quell\- und Ausführungszeichensatz enthält die in der folgenden Tabelle aufgeführten ASCII\-Zeichen.  Escapesequenzen werden ebenfalls in der Tabelle dargestellt.  
  
### Escapesequenzen  
  
|Escapesequenz|Zeichen|ASCII\-Wert|  
|-------------------|-------------|-----------------|  
|\\a|Warnung\/Klingelzeichen|7|  
|\\b|Rückschritt|8|  
|\\f|Seitenvorschub|12|  
|\\n|Zeilenumbruch|10|  
|\\r|Wagenrücklauf|13|  
|\\t|Horizontaler Tabulator|9|  
|\\v|Vertikaler Tabulator|11|  
|\\"|Doppelte Anführungszeichen|34|  
|\\'|Einfache Anführungszeichen|39|  
|\\\\|Umgekehrter Schrägstrich|92|
