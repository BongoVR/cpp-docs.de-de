# clock-Funktion (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.2.1** Der Zeitraum der `clock`\-Funktion  
  
 Der Zeitraum der `clock`\-Funktion startet \(mit einem Wert von 0\), wenn das C\-Programm ausgeführt wird.  Es werden Zeiten zurückgegeben, die in 1\/**CLOCKS\_PER\_SEC** \(was 1\/1000 für Microsoft C entspricht\) gemessen werden.
