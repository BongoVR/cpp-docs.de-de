# Konvertieren von Mehrbytezeichen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.3.4** Das aktuelle Gebietsschema, mit dem Mehrbytezeichen in äquivalente Breitzeichen \(Code\) für eine Breitzeichenkonstante konvertiert werden  
  
 Das aktuelle Gebietsschema ist standardmäßig das Gebietsschema "C".  Dies kann durch [\#pragma setlocale](../preprocessor/setlocale.md) geändert werden.
