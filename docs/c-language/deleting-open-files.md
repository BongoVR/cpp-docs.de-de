# <a name="deleting-open-files"></a>Löschen von offenen Dateien
**ANSI 4.9.4.1** Der Effekt der remove-Funktion auf eine offene Datei  
  
 Mit der Funktion zum Entfernen wird eine Datei gelöscht. Wenn die Datei geöffnet ist, ist diese Funktion fehlerhaft und gibt -1 zurück.
