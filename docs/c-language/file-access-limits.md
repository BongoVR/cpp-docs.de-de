# <a name="file-access-limits"></a>Dateizugriffsgrenzen
**ANSI 4.9.3** Legt fest, ob die gleiche Datei mehrmals geöffnet werden kann.  
  
 Das Öffnen einer Datei, die bereits geöffnet ist, ist nicht zulässig.
