# Dateizugriffsgrenzen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Legt fest, ob dieselbe Datei mehrmals geöffnet werden kann.  
  
 Das Öffnen einer Datei, die bereits geöffnet ist, ist nicht zulässig.
