# Dateipufferung
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** Die Eigenschaften der Dateipufferung  
  
 Die Datenträgerdateien, auf die über Standard\-E\/A\-Funktionen zugegriffen wird, werden vollständig gepuffert.  Standardmäßig enthält der Puffer 512 Bytes.
