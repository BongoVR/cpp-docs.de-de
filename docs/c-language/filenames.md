# <a name="filenames"></a>Dateinamen
**ANSI 4.9.3** Die Regeln für das Verfassen von gültigen Dateinamen  
  
 Eine Dateispezifikation kann einen optionalen Laufwerksbuchstaben (immer gefolgt von einem Doppelpunkt), eine Reihe optionaler Verzeichnisnamen (getrennt durch umgekehrte Schrägstriche) und einen Dateinamen enthalten.  
  
 Weitere Informationen finden Sie unter [Naming Files, Paths, and Namespaces](http://msdn.microsoft.com/library/windows/desktop/aa365247) (Benennen von Dateien, Pfaden und Namespaces).
