# fmod-Funktion
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.5.6.4** Ob ein Domänenfehler auftritt oder Null zurückgegeben wird, wenn die `fmod`\-Funktion ein zweites Argument von Null aufweist  
  
 Wenn die `fmod`\-Funktion ein zweites Argument mit dem Wert 0 \(null\) aufweist, gibt die Funktion 0 \(null\) zurück.
