# Subtraktion von Zeigern
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.6, 4.1.1** Der Typ einer Ganzzahl, die erforderlich ist, um den Unterschied zwischen zwei Zeigern, die auf Elemente desselben Arrays, **ptrdiff\_t**, zeigen, zu speichern  
  
 Ein **ptrdiff\_t** ist ein **signed int**.
