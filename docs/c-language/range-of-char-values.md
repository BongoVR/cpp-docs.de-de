# Bereich der char-Werte
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.2.1.1** Ob ein "einfacher" **char**\-Wert denselben Wertebereich wie ein **signed char**\-Wert oder ein `unsigned char`\-Wert aufweist  
  
 Alle signierten Zeichenwerte liegen im Bereich von –128 bis 127.  Alle nicht signierten Zeichenwerte liegen im Bereich von 0 bis 255.  
  
 Die \/J\- Compileroption ändert den Standard von **signed** in `unsigned`.
