# Umbenennen mit einem bereits vorhandenen Namen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.4.2** Der Effekt, wenn eine Datei mit dem neuen Namen vor einem Aufruf der **rename**\-Funktion bereits vorhanden ist  
  
 Wenn Sie versuchen, eine Datei mit einem Namen umzubenennen, der vorhanden ist, schlägt die **rename**\-Funktion fehl und gibt einen Fehlercode zurück.
