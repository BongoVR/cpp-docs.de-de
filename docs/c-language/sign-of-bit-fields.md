# Vorzeichen von Bitfeldern
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.5.2.1** Ob ein "einfaches" `int`\-Feld als **signed int**\-Bitfeld oder als "unsigned int"\-Bitfeld behandelt wird  
  
 Es gibt Bitfelder mit oder ohne Vorzeichen.  Einfache Bitfelder werden als signiert behandelt.
