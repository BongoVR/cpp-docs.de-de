# Bedeutende Zeichen ohne externe Verkn&#252;pfung
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.2** Die Anzahl von signifikanten Zeichen ohne externe Verknüpfung  
  
 Bezeichner sind bis zu 247 Zeichen signifikant.  Der Compiler begrenzt nicht die Anzahl der Zeichen, die Sie in einem Bezeichner verwenden können. Er ignoriert einfach alle Zeichen über die Begrenzung hinaus.
