# Anweisungen: Einschr&#228;nkungen f&#252;r Switch-Anweisungen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.6.4.2** Die maximale Anzahl von **case**\-Werten in einer `switch`\-Anweisung  
  
 Microsoft C beschränkt nicht die Anzahl von **case** \-Werten in einer **switch** \-Anweisung.  Die Anzahl wird nur durch den verfügbaren Speicher beschränkt.
