# Speicherung von Arrays
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Der einem Arraytyp zugeordnete Speicherplatz ist der Speicher, der für seine sämtlichen Elemente erforderlich ist.  Die Elemente eines Arrays werden in aneinandergrenzenden und zunehmenden Speicherbereichen gespeichert, vom ersten bis zum letzten Element.
