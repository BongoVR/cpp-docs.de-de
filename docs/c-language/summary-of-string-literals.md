# Zusammenfassung von Zeichenfolgenliteralen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

*string\-literal*:  
 **'** *s\-char\-sequence*  opt **'**  
  
 **L'** *s\-char\-sequence*  opt             **'**  
  
 *s\-char\-sequence*:  
 *s\-char*  
  
 *s\-char\-sequence s\-char*  
  
 *s\-char*:  
 Alle Member des Quellzeichensatzes mit Ausnahme von doppelten Anführungszeichen \("\), umgekehrtem Schrägstrich \(\\\) oder Zeilenumbruchzeichen *Escapesequenz*
