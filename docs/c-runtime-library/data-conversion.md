# Datenkonvertierung
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Diese Routinen konvertieren Daten aus einer Form in eine andere.  Im Allgemeinen konvertieren diese Routinen die Daten schneller als Konvertierungen, die Sie selbst schreiben.  Jede Routine, die mit einem `to`\-Präfix beginnt, wird als Funktion und als Makro implementiert.  Weitere Informationen zum Auswählen einer Implementierung finden Sie unter [Auswählen zwischen Funktionen und Makros](../c-runtime-library/recommendations-for-choosing-between-functions-and-macros.md).
