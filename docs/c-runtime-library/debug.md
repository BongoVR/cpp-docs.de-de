# _DEBUG
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Der Compiler definiert `_DEBUG`, wenn Sie der \/MTd\- oder \/MDd\- Option angeben.  Diese Optionen Debugversionen der C\-Laufzeitbibliothek an.  
  
 Weitere Informationen finden Sie unter [CRT\-Debug\-Techniken](../Topic/CRT%20Debugging%20Techniques.md).
