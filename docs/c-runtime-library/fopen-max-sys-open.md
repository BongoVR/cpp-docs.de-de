# <a name="fopenmax-sysopen"></a>FOPEN_MAX, _SYS_OPEN
## <a name="syntax"></a>Syntax  
  
```  
  
#include <stdio.h>  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Dies ist die maximale Anzahl von Dateien, die gleichzeitig geöffnet werden können. `FOPEN_MAX` ist der ANSI-kompatible Name. `_SYS_OPEN` dient der Kompatibilität mit vorhandenem Code.
