# FOPEN_MAX, _SYS_OPEN
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Syntax  
  
```  
  
#include <stdio.h>  
  
```  
  
## Hinweise  
 Dies ist die maximale Anzahl von Dateien, die gleichzeitig geöffnet sein können.  `FOPEN_MAX` ist der ANSI\-kompatible Name.  `_SYS_OPEN` wird für die Kompatibilität mit vorhandenem Code bereitgestellt.
