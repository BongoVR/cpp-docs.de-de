# _FREEENTRY, _USEDENTRY
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## Syntax  
  
```  
#include <malloc.h>  
```  
  
## Hinweise  
 Diese Konstanten stellen die Werte dar, die von der `_heapwalk` Routinen zum Element **\_useflag** der **\_HEAPINFO**\-Struktur zugewiesen werden.  Sie geben den Fortschritt des Heapeintrags an.
