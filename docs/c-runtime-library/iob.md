# <a name="iob"></a>_iob
Das Array von stdio-Steuerungsstrukturen.  
  
## <a name="syntax"></a>Syntax  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## <a name="remarks"></a>Hinweise  
 `IOB_ENTRIES` ist als 20 in stdio.h definiert.
