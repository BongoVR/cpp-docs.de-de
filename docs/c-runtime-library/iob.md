# _iob
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Das Array von stdio Steuerungsstrukturen.  
  
## Syntax  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## Hinweise  
 `IOB_ENTRIES` wird als 20 in stdio.h definiert.
