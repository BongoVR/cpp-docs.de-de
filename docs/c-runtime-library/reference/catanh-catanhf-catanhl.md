# <a name="catanh-catanhf-catanhl"></a>catanh, catanhf, catanhl
Ruft den umgekehrten hyperbolischen Tangens einer komplexen Zahl, mit Branch Schnitte außerhalb des Intervalls [-1; + 1] auf der tatsächlichen Achse ab.  
  
## <a name="syntax"></a>Syntax  
  
```  
_Dcomplex catanh(   
   _Dcomplex z   
);  
_Fcomplex catanh(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex catanh(   
   _Lcomplex z   
);  //  C++ only  
_Fcomplex catanhf(   
   _Fcomplex z   
);  
_Lcomplex catanhl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `z`  
 Eine komplexe Zahl, die einen Winkel als Bogenmaß darstellt  
  
## <a name="return-value"></a>Rückgabewert  
 Der umgekehrte hyperbolische Tangens von `z` als Bogenmaß. Das Ergebnis ist unbounded real Achse, und klicken Sie im Intervall [-Iπ/2; + Iπ/2] der imaginäre Achse. Wenn `z` außerhalb des Intervalls [-1, +1] liegt, tritt ein Bereichsfehler auf. Wenn `z` -1 oder +1 ist, tritt ein Polstellenfehler auf.  
  
## <a name="remarks"></a>Hinweise  
 Da C++ das Überladen zulässt, können Sie Überladungen von `catanh` aufrufen, die `_Fcomplex`- und `_Lcomplex`-Werte verwenden und zurückgeben. In einem C-Programm nimmt `catanh` immer einen `_Dcomplex` -Wert an, und gibt auch einen solchen zurück.
