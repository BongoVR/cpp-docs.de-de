# <a name="ccos-ccosf-ccosl"></a>ccos, ccosf, ccosl
Ruft den Kosinus einer komplexen Zahl ab.  
  
## <a name="syntax"></a>Syntax  
  
```  
_Dcomplex ccos(   
   _Dcomplex z   
);  
_Fcomplex ccos(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex ccos(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex ccosf(   
   _Fcomplex z   
);  
_Lcomplex ccosl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>Parameter  
 `z`  
 Eine komplexe Zahl, die den Winkel als Bogenmaß darstellt  
  
## <a name="return-value"></a>Rückgabewert  
 Der Kosinus von `z` als Bogenmaß.  
  
## <a name="remarks"></a>Hinweise  
 Da C++ das Überladen zulässt, können Sie Überladungen von `ccos` aufrufen, die `_Fcomplex`- und `_Lcomplex`-Werte verwenden und zurückgeben. In einem C-Programm nimmt `ccos` immer einen `_Dcomplex` -Wert an, und gibt auch einen solchen zurück.
