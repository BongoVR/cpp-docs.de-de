# spawnvp
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Diese POSIX\-Funktion ist veraltet.  Verwenden Sie stattdessen das ISO C\+\+\-konforme [\_spawnvp](../../c-runtime-library/reference/spawnvp-wspawnvp.md)\-Funktion.  
  
> [!IMPORTANT]
