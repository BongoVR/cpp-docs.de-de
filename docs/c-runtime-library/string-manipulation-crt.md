# Zeichenfolgenbearbeitung (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Verwenden Sie die unter [Puffermanipulation](../c-runtime-library/buffer-manipulation.md) beschriebenen Puffermanipulationsroutinen, um mit Zeichenarrays zu arbeiten, die nicht mit einem NULL\-Zeichen abschließen.
