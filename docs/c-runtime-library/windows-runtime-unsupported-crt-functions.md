# <a name="windows-runtime-unsupported-crt-functions"></a>CRT-Funktionen, die nicht von Windows-Runtime unterstützt werden
Ein Großteil der CRT-APIs kann in UWP-Apps (Universelle Windows-Plattform), die in der Windows-Runtime ausgeführt werden, nicht verwendet werden. Diese Apps werden mithilfe des /ZW-Compilerkennzeichens erstellt. Eine Liste der nicht unterstützten CRT-Funktionen finden Sie unter [Nicht von /ZW unterstützte CRT-Funktionen](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).  
  
 Alle CRT-APIs werden im Abschnitt [Alphabetischer Funktionsverweis](../c-runtime-library/reference/crt-alphabetical-function-reference.md) der Dokumentation beschrieben.
