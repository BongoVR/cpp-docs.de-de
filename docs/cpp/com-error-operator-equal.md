# _com_error::operator =
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft\-spezifisch**  
  
 Weist ein vorhandenes `_com_error`\-Objekt einem anderen zu.  
  
## Syntax  
  
```  
  
      _com_error& operator = (  
   const _com_error& that   
) throw ( );  
```  
  
#### Parameter  
 `that`  
 Ein `_com_error`\-Objekt.
