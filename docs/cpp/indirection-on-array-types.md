# Dereferenzierung auf Arraytypen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Verwendung des Dereferenzierungsoperators \(**\***\) in *n*\-dimensionalem Arraytyp ergibt *n*–1\-dimensionales Array.  Wenn *n* 1 ist, ergibt sich ein Skalar \(oder Arrayelement\).
