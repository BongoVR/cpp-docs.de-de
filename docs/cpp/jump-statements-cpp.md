# Sprunganweisungen (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Eine Sprunganweisung in C\+\+ führt eine unmittelbare lokale Übertragung der Steuerung aus.  
  
## Syntax  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
