# <a name="jump-statements-c"></a>Sprunganweisungen (C++)
Eine Sprunganweisung in C++ führt eine unmittelbare lokale Übertragung der Steuerung aus.  
  
## <a name="syntax"></a>Syntax  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
