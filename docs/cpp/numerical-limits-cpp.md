# <a name="numerical-limits-c"></a>Numerische Grenzen (C++)
Die beiden Standardincludedateien LIMITS.H und FLOAT.H definieren die numerischen Grenzwerte oder die minimalen und maximalen Grenzwerte, die eine Variable eines bestimmten Typs aufnehmen kann. Diese Mindest- und Höchstwerte sind garantiert auf jeden C++-Compiler übertragbar, die die dieselbe datendarstellung wie ANSI c verwendet Die Grenzwerte. H-Includedatei definiert die [numerische Grenzwerte für ganzzahlige Typen](../cpp/integer-limits.md), und FLOAT. H definiert die [numerische Grenzwerte für Gleitkommatypen](../cpp/floating-limits.md).
