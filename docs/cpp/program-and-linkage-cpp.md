# <a name="program-and-linkage--c"></a>Programm und Verknüpfung (C++)
Ein Programm besteht aus einer oder mehreren Übersetzungseinheiten, die miteinander verknüpft sind. Die Ausführung beginnt (konzeptionell) in der Übersetzungseinheit, die die Funktion enthält **main**. (Weitere Informationen zu Übersetzungseinheiten finden Sie unter [Phasen der Übersetzung](../preprocessor/phases-of-translation.md)in der *Präprozessorreferenz*.) Weitere Informationen zu den **main** funktionieren, finden Sie unter [Programmstart: die Hauptfunktion](../cpp/main-program-startup.md).)
