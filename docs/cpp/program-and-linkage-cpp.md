# Programm und Verkn&#252;pfung (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ein Programm besteht aus einer oder mehreren Übersetzungseinheiten, die miteinander verknüpft sind.  Die Ausführung beginnt \(konzeptionell\) in der Übersetzungseinheit, die die **main**\-Funktion enthält.  \(Weitere Informationen zu Übersetzungseinheiten finden Sie unter [Phasen der Übersetzung](../preprocessor/phases-of-translation.md) in der *Präprozessorreferenz*\). \(Weitere Informationen über die **main**\-Funktion finden Sie unter [Program Startup: the main Function](../cpp/main-program-startup.md).\)
