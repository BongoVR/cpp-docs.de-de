# virtual-Spezifizierer
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Das Schlüsselwort [virtuell](../cpp/virtual-cpp.md) kann nur auf nicht statische Memberfunktionen von Klassen angewendet werden.  Es gibt an, dass die Bindung von Aufrufen der Funktion bis zur Laufzeit verzögert wird.  Weitere Informationen finden Sie unter [Virtuelle Funktionen](../cpp/virtual-functions.md).
