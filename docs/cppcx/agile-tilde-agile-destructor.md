# Agile::~Agile-Destruktor
Zerstört die aktuelle Instanz der Agile\-Klasse.  
  
## Syntax  
  
```cpp  
  
~Agile();  
```  
  
## Hinweise  
 Dieser Destruktor gibt auch das Objekt frei, das vom aktuellen Agile\-Objekt dargestellt wird.
