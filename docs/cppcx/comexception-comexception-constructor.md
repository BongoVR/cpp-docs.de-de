# COMException::COMException-Konstruktor
Initialisiert eine neue Instanz der COMException\-Klasse.  
  
## Syntax  
  
```cpp  
COMException( int hresult )  
```  
  
#### Parameter  
 HRESULT  
 Der Fehler HRESULT, der durch die Ausnahme repräsentiert wird.
