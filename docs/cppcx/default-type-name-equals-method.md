# default::(type_name)::Equals-Methode
Bestimmt, ob das angegebene Objekt mit dem aktuellen Objekt identisch ist.  
  
## Syntax  
  
```cpp  
  
bool Equals(  
    Object^ obj  
)  
```  
  
## Parameter  
 obj  
 Das zu vergleichende Objekt.  
  
## Rückgabewert  
 `true`, wenn die Objekte gleich sind, andernfalls `false`.
