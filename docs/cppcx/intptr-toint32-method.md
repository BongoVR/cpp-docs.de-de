# IntPtr::ToInt32-Methode
Konvertiert den aktuellen IntPtr\-Wert in eine 32\-Bit\-Ganzzahl.  
  
## Syntax  
  
```cpp  
int32 IntPtr::ToInt32();  
```  
  
## Rückgabewert  
 Eine 32\-Bit\-Ganzzahl
