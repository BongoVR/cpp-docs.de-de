# Map::Remove-Methode
Löscht das angegebene Schlüssel\-Wert\-Paar vom aktuellen Map\-Objekt.  
  
## Syntax  
  
```  
  
virtual void Remove(  
   K key);  
```  
  
#### Parameter  
 `key`  
 Der Schlüsselteil des Schlüssel\-Wert\-Paars. Der Typ von `key` lautet Typname *K*.  
  
## Anforderungen  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
