# Map::Size-Methode
Gibt die Anzahl der [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx)\-Elemente in der Zuordnung zurück.  
  
## Syntax  
  
```  
  
virtual property unsigned int Size;  
```  
  
## Rückgabewert  
 Die Anzahl von Elementen in der Zuordnung.  
  
## Anforderungen  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
