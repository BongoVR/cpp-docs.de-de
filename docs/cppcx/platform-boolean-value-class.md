# Platform::Boolean-Wertklasse
Stellt einen booleschen Wert dar. Das Äquivalent von `bool`.  
  
## Syntax  
  
```cpp  
public value struct Boolean  
```  
  
## Mitglieder  
 „Boolean“ enthält die Methoden Equals\(\), GetHashCode\(\) und ToString\(\), die von der [Platform::Object\-Klasse](../cppcx/platform-object-class.md) abgeleitet werden sowie die GetTypeCode\(\)\-Methode, die von der [Platform::Type\-Klasse](../cppcx/platform-type-class.md) abgeleitet wird.
