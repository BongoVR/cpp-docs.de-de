# Platform::Metadata::FlagsAttribute-Attribut
Gibt an, dass eine Enumeration als Bitfeld, d. h. als Gruppe von Flags, behandelt werden kann.  
  
## Syntax  
  
```cpp  
  
public ref class Flags abstract : Attribute  
```
