# Platform::NotImplementedException-Klasse
Wird ausgelöst, wenn ein Schnittstellenmember nicht in einem abgeleiteten Typ implementiert ist.  
  
## Syntax  
  
```cpp  
public ref class NotImplementedException : COMException,    IException,    IPrintable,    IEquatable  
```  
  
## Hinweise  
 Weitere Informationen finden Sie unter der [COMException](../cppcx/platform-comexception-class.md)\-Klasse.
