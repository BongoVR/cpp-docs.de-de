# Platform::ObjectDisposedException-Klasse
Wird ausgelöst, wenn ein Vorgang für ein verworfenes Objekt ausgeführt wird.  
  
## Syntax  
  
```cpp  
public ref class ObjectDisposedException : COMException,    IException,    IPrintable,    IEquatable  
```
