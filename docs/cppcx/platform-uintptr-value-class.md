# Platform::UIntPtr-Wertklasse
Stellt einen Zeiger ohne Vorzeichen dar, dessen Größe für die Plattform \(32\-Bit oder 64\-Bit\) geeignet ist.  
  
## Syntax  
  
```cpp  
public value struct UintPtr  
```
