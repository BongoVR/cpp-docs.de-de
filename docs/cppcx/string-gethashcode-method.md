# String::GetHashCode-Methode
Gibt den Hashcode für diese Instanz zurück.  
  
## Syntax  
  
```cpp  
  
virtual override int GetHashCode()  
```  
  
## Rückgabewert  
 Der Hashcode für diese Instanz.
