# String::IsEmpty-Methode
Gibt an, ob das aktuelle String\-Objekt leer ist.  
  
## Syntax  
  
```cpp  
  
bool IsEmpty()  
```  
  
## Rückgabewert  
 `true`, wenn das aktuelle String\-Objekt `null` oder die leere Zeichenfolge \(L ""\) ist; andernfalls `false`.
