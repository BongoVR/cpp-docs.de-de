# Vector::Append-Methode
Fügt das angegebene Element nach dem letzten Element im aktuellen Vektor ein.  
  
## Syntax  
  
```  
  
virtual void Append(  
   T item  
);  
```  
  
#### Parameter  
 `index`  
 Das Element, das in den Vektor eingefügt werden soll. Der Typ von `item` wird durch den Typnamen *T* definiert.  
  
## Anforderungen  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
