# Vector::Size-Methode
Gibt die Anzahl von Elementen im aktuellen Vector\-Objekt zurück.  
  
## Syntax  
  
```  
  
virtual property unsigned int Size;  
```  
  
## Rückgabewert  
 Die Anzahl der Elemente im aktuellen Vector.  
  
## Anforderungen  
 **Header:** collection.h  
  
 **Namespace:** Platform::Collections
