# WriteOnlyArray::Length-Eigenschaft
Gibt die Anzahl der Elemente im vom Aufrufer reservierten Array zurück.  
  
## Syntax  
  
```cpp  
property unsigned int Length{  
   unsigned int get() const;  
}  
```  
  
## Rückgabewert  
 Die Anzahl der Elemente im Array.  
  
## Anforderungen  
 **Header:** vccorlib.h  
  
 **Namespace:** Platform
