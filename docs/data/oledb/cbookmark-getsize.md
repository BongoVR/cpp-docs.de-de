# CBookmark::GetSize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ruft die Größe des Lesezeichenpuffers ab.  
  
## Syntax  
  
```  
  
virtual DBLENGTH GetSize( ) const throw( );  
  
```  
  
## Rückgabewert  
 Die Größe des Puffers in Bytes.  
  
## Anforderungen  
 **Header:** atldbcli.h
