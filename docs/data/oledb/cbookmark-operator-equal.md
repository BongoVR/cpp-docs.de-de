# CBookmark::operator =
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Weist ein `CBookmark`\-Objekt für andere zu.  
  
## Syntax  
  
```  
  
      CBookmark& operator =(   
   const CBookmark& bookmark    
) throw( );  
```  
  
## Hinweise  
 Dieser Operator wird nur in **CBookmark \<0\>** benötigt.  
  
## Anforderungen  
 **Header:** atldbcli.h
