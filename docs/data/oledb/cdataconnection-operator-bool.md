# CDataConnection::operator BOOL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Bestimmt, ob die aktuelle Sitzung oder nicht geöffnet ist.  
  
## Syntax  
  
```  
  
operator BOOL( ) throw( );  
  
```  
  
## Hinweise  
 **BOOL**\-Wert die EINGABETASTE \(MFC\-typedef\).  **TRUE** gibt an, dass die aktuelle Sitzung geöffnet ist; **FALSE** gibt an, dass die aktuelle Sitzung geschlossen wird.  
  
## Anforderungen  
 **Header:** atldbcli.h
