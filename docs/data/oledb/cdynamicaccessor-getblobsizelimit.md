# CDynamicAccessor::GetBlobSizeLimit
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ruft die maximale BLOBgröße in Bytes ab.  
  
## Syntax  
  
```  
  
const DBLENGTH GetBlobSizeLimit( ) const;  
  
```  
  
## Hinweise  
 Gibt den BLOBbehandlungswert `nBlobSize` z Satz von [SetBlobSizeLimit](../../data/oledb/cdynamicaccessor-setblobsizelimit.md) zurückgegeben.  
  
## Anforderungen  
 **Header:** atldbcli.h
