# CMultipleResults-Klasse
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Wenn Sie einen Befehl möchten, mehrere Resultsets behandeln zu können, verwenden Sie `CMultipleResults` für das `CCommand` Vorlagenargument *TMultiple*.  
  
## Syntax  
  
```  
class CMultipleResults  
```  
  
## Hinweise  
 Um mehrere Resultsets zu behandeln, muss [CCommand](../../data/oledb/ccommand-class.md) von dieser Klasse erben.  
  
## Anforderungen  
 **Header:** atldbcli
