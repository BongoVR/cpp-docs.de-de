# CNoMultipleResults-Klasse
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Wird als Vorlagenargument \(*TMultiple*\) von [CCommand](../../data/oledb/ccommand-class.md), um einen optimierten Befehl zu erstellen, der ein einzelnes Resultset behandelt.  
  
## Syntax  
  
```  
class CNoMultipleResults  
```  
  
## Hinweise  
 Wenn Sie einen Befehl möchten, mehrere Resultsets behandeln zu können, verwenden Sie stattdessen [CMultipleResults](../../data/oledb/cmultipleresults-class.md).  
  
## Anforderungen  
 **Header:**  atldbcli.h
