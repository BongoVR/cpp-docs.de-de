# END_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Markiert das Ende der Spaltenzuordnungseinträge.  
  
## Syntax  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## Hinweise  
 Sie ist mit einem einzelnen Accessor für ein Rowset verwendet.  Das Makro `BEGIN_COLUMN_MAP` wird mit dem Makro `END_COLUMN_MAP` abgeschlossen.  
  
## Beispiel  
 Siehe [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md).  
  
## Anforderungen  
 **Header:** atldbcli.h
