# END_PARAM_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Markiert das Ende der Parameterzuordnungseinträge.  
  
## Syntax  
  
```  
  
END_PARAM_MAP( )  
  
```  
  
## Beispiel  
 Im Beispiel für das [BEGIN\_PARAM\_MAP](../../data/oledb/begin-param-map.md)\-Makro.
