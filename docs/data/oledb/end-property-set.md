# END_PROPERTY_SET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Markiert das Ende eines Eigenschaftensatzes.  
  
## Syntax  
  
```  
  
END_PROPERTY_SET(  
guid   
)  
  
```  
  
#### Parameter  
 `guid`  
 \[in\] Eigenschaft die GUID.  
  
## Beispiel  
 Siehe [BEGIN\_PROPSET\_MAP](../../data/oledb/begin-propset-map.md).  
  
## Anforderungen  
 **Header:** atldb.h
