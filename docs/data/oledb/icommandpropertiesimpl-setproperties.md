# <a name="icommandpropertiesimplsetproperties"></a>ICommandPropertiesImpl::SetProperties
Legt Eigenschaften für das Command-Objekt.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
      STDMETHOD(SetProperties)(   
   ULONG cPropertySets,   
   DBPROPSET rgPropertySets[]    
);  
```  
  
#### <a name="parameters"></a>Parameter  
 Finden Sie unter [ICommandProperties:: SetProperties](https://msdn.microsoft.com/en-us/library/ms711497.aspx) in der *OLE DB Programmer's Reference*.

 **Header:** „atldb.h“
