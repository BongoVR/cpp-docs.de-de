# IDBInitializeImpl::Uninitialize
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Setzt das Datenquellenobjekt in einem nicht initialisierten Zustand durch Freigeben von internen Ressourcen wie die Eigenschaftenunterstützung.  
  
## Syntax  
  
```  
  
      STDMETHOD(Uninitialize)(   
   void    
);  
```  
  
## Hinweise  
 Siehe [IDBInitialize::Uninitialize](https://msdn.microsoft.com/en-us/library/ms719648.aspx) in der *OLE* DB\-Programmierreferenz.
