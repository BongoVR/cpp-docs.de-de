# Schemarowsets
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Informationen über die Struktur oder Organisation einer Datenbank \(z. B. Informationen über Anbieter, Rowsets, Tabellen, Spalten\) oder andere Informationen, die nicht im Zusammenhang mit dem tatsächlichen Inhalt der Datenbank stehen, werden als Schema \(oder Metadaten\) bezeichnet.  Mithilfe von Schemarowsets ist es in OLE DB möglich, diese Art von Informationen zu beziehen, ohne das Rowset zu öffnen.  
  
 Informationen über die Verwendung von Schemarowsets finden Sie unter [Abrufen von Metadaten mit Schemarowsets](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
