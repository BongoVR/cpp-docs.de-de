# generate_n (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Weist die Werte, die von ein Funktionsobjekt an einer angegebenen Anzahl des Elements generiert werden, ist ein Bereich zurückkehrt und zur Position eine Vergangenheit der letzte bestimmten Wert.  
  
## Syntax  
  
```  
template<class _OutIt, class _Diff, class _Fn0> inline  
    void generate_n(_OutIt _Dest, _Diff _Count, _Fn0 _Func);  
```  
  
## Hinweise  
 Diese Funktion verhält sich genauso wie die STL\-Funktion `generate_n`.  Weitere Informationen finden Sie unter [generate\_n](../Topic/generate_n.md).  
  
## Anforderungen  
 **Header:** \<cliext\/Algorithmus\>  
  
 **Namespace:** cliext
