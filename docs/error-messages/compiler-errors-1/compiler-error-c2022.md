# Compilerfehler C2022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Zahl': Zu groß für Zeichen  
  
 Der auf einen umgekehrten Schrägstrich \(\\\) folgende Oktalwert in einer Zeichen\- oder Zeichenfolgenkonstante ist zu groß, um ein Zeichen darzustellen.
