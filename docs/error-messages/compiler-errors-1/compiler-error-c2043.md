# Compilerfehler C2043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Schlüsselwort "break" ungültig  
  
 Ein [break](../../cpp/break-statement-cpp.md) ist nur in einer `do`\-, `for`\-, `while`\- oder `switch`\-Anweisung zulässig.
