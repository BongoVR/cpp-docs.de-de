# Compilerfehler C2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Typumwandlung in Funktionstyp nicht erlaubt  
  
 Die Typumwandlung zwischen einem Funktionszeiger und einem Datenzeiger ist in ANSI C nicht zulässig.
