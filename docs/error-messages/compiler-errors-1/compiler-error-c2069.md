# Compilerfehler C2069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Typumwandlung eines "void"\-Ausdrucks in nicht\-"void"  
  
 Der Typ `void` kann nicht in einen anderen Typ umgewandelt werden.
