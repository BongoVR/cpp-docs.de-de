# Compilerfehler C2077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Initialisierung des nicht skalaren Feldes 'Bezeichner'  
  
 Sie haben versucht, ein Bitfeld mit einem nicht skalaren Element \(Struktur, Union, Array oder Klasse\) zu initialisieren.  Verwenden Sie einen Ganzzahl\- oder Gleitkommawert.
