# Compilerfehler C2089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner': 'Klassenschlüssel' zu groß  
  
 Die angegebene Struktur oder Union ist größer als 4 GB.
