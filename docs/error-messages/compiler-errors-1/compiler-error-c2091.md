# Compilerfehler C2091
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Funktionsergebnis ist eine Funktion  
  
 Eine Funktion kann keine Funktion zurückgeben.  Stattdessen sollte ein Zeiger auf eine Funktion zurückgegeben werden.
