# Compilerfehler C2095
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion': Parameter hat den Typ 'void' : Parameter 'Nummer'  
  
 Der an die Funktion übergebene Parameter hat den nicht zulässigen Typ `void`.  Verwenden Sie stattdessen einen Zeiger auf void \( `void *`\).  
  
 `number` gibt an, welcher Parameter `void` ist.
