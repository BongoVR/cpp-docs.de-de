# Compilerfehler C2102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"&" erfordert I\-Wert  
  
 Der address\-of\-Operator \(`&`\) muss einen l\-Wert als Operanden haben.
