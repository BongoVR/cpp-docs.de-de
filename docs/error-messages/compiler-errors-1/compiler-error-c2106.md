# <a name="compiler-error-c2106"></a>Compilerfehler C2106
'Operator': linke Operand muss ein l-Wert  
  
 Der Operator muss einen l-Wert als dem linken Operanden haben.  
  
 Im folgende Beispiel wird C2106 generiert:  
  
```  
// C2106.cpp  
int main() {  
   int a;  
   1 = a;   // C2106  
   a = 1;   // OK  
}  
```
