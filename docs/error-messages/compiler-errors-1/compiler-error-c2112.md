# Compilerfehler C2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"\-": Subtraktion von Zeigern erfordert einen Ganzzahl\- oder Zeigeroperanden  
  
 Es wurde versucht, Zeiger zu subtrahieren, die auf unterschiedliche Typen zeigen.
