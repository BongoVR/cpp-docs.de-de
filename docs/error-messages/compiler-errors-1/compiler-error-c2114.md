# Compilerfehler C2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Operator': Zeiger auf der linken Seite erfordert Ganzzahl auf der rechten Seite.  
  
 Der linke Operand von `operator` war ein Zeiger, daher muss der rechte Operand ein ganzzahliger Wert sein.
