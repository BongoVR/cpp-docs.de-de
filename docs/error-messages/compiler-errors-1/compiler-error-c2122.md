# Compilerfehler C2122
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner': Prototypparameter in der Namensliste ist ungültig  
  
 Der Parameter entspricht keinem gültigen Typ.  ANSI C unterstützt keine benutzerdefinierten Typen.
