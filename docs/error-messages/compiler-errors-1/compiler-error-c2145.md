# Compilerfehler C2145
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Syntaxfehler: Fehlendes "Token" vor Bezeichner  
  
 Der Compiler hat ein `token` erwartet, jedoch einen Bezeichner gefunden.  
  
 Dieser Fehler kann dadurch verursacht sein, dass nach der letzten Deklaration in einem Block ein Semikolon fehlt.
