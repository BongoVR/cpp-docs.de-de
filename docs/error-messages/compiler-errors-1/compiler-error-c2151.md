# Compilerfehler C2151
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Mehr als ein Sprachattribut  
  
 Eine Funktion verfügt über mehr als ein Schlüsselwort \( `__cdecl`, `__stdcall` oder `__fastcall`\) zur Festlegung einer Aufrufkonvention.
