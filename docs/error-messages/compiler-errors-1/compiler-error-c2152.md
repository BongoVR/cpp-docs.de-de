# Compilerfehler C2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner': Zeiger auf Funktionen mit verschiedenen Attributen  
  
 Einem Zeiger auf eine Funktion mit einer Aufrufkonvention \(`__cdecl`, `__stdcall` oder `__fastcall`\) wurde ein Zeiger auf eine Funktion mit einer anderen Aufrufkonvention zugewiesen.
