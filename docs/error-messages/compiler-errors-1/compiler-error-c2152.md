# <a name="compiler-error-c2152"></a>Compilerfehler C2152
'Bezeichner': Zeiger auf Funktionen mit verschiedenen Attributen  
  
 Ein Zeiger auf eine Funktion mit einer Aufrufkonvention (`__cdecl`, `__stdcall`, oder `__fastcall`) wird in einen Zeiger auf eine Funktion mit einer anderen Aufrufkonvention zugewiesen.
