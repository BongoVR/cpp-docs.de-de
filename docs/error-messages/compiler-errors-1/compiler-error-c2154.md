# Compilerfehler C2154
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"Typ" : Als Argument für das systeminterne Typmerkmal "\_\_underlying\_type" des Compilers ist nur ein Enumerationstyp zulässig.  
  
 Sie können nur den zugrunde liegenden Typ eines Enumerationstyps abrufen.  
  
 Weitere Informationen finden Sie unter [Compiler Support for Type Traits](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md).
