# <a name="compiler-error-c2155"></a>Compilerfehler C2155
'?': Ungültiges linkes erwartet Operanden arithmetische oder Zeiger vom Typ  
  
 Ein Ausdruck auf der linken Seite des `?` darf nicht 0 (null) verglichen werden. Sie müssen einen arithmetische oder Zeiger-Ausdruck verwenden, der mit 0 (null) verglichen werden können.
