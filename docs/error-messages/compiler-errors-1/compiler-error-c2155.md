# Compilerfehler C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?' : linker Operand ungültig, arithmetischer Typ oder Zeigertyp erwartet  
  
 Ein Ausdruck links von `?` kann nicht mit 0 verglichen werden.  Sie müssen einen arithmetischen Ausdruck oder Zeiger verwenden, der mit 0 verglichen werden kann.
