# Compilerfehler C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion': Ist als systeminterne Funktion nicht verfügbar  
  
 Ein `intrinsic`\-Pragma oder ein `function`\-Pragma listet eine Funktion auf, die nicht in systeminterner Form verfügbar ist.  Bestimmte systeminterne Funktionen sind beispielsweise beim Kompilieren eines Programms nicht verfügbar, das \/clr\-Programmierung verwendet.
