# Compilerfehler C2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Gebietsschema': Ungültiges Gebietsschema  
  
 Das angegebene Gebietsschema ist nicht gültig.  Eine Liste der unterstützten Gebietsschemas finden Sie unter [Language and Country\/Region Strings](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) in der *Run\-Time Library Reference*.
