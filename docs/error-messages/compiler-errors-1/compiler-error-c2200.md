# Compilerfehler C2200
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"function": Funktion wurde bereits definiert.  
  
 Ein `alloc_text`\-Pragma verwendet einen bereits definierten Funktionsnamen.
