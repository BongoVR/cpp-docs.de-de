# <a name="compiler-error-c2223"></a>Compilerfehler C2223
Links von "-> Bezeichner" muss auf Struktur/Union zeigen.  
  
 Der Operand auf der linken Seite des `->` ist kein Zeiger auf eine Klasse, Struktur oder Union.  
  
 Dieser Fehler kann verursacht werden, durch einen linken Operanden, der eine Variable nicht definiert ist (Geben Sie daher `int`).
