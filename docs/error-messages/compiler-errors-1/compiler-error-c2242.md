# Compilerfehler C2242
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

typedef\-Name kann nicht auf Klasse\/Struktur\/Union folgen  
  
 Ein `typedef`\-Name tritt am Ende eines qualifizierten Namens auf.
