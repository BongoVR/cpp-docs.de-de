# Compilerfehler C2290
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die C\+\+\-Syntax "asm" wird ignoriert. Verwenden Sie "\_\_asm".  
  
 Die `asm`\-Syntax ist für die spätere Verwendung vorgesehen.
