# Compilerfehler C2322
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner': Adresse von dllimport 'dllimport' ist nicht statisch.  
  
 Ein nicht statischer Wert wird als Adresse einer Funktion angegeben, die mit `dllimport` deklariert wurde.
