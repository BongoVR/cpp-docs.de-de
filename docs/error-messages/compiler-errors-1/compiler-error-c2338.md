# Compilerfehler C2338
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

„identifier“ ATL\-Attributanbieter  
  
 Das Attribut verweist auf einen nicht deklarierten Bezeichner. Dieser Fehler wird zusammen mit dem [ATL\-Anbieterfehler ATL2010](assetId:///27a25924-b52b-4971-80d3-c3b3e9ffbe41) angezeigt.
