# Compilerfehler C2376
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"Funktion": Neudefinition; unterschiedliche Segmentzuordnung  
  
 Die Funktion wurde bereits mit einer anderen Segmentzuordnung deklariert.
