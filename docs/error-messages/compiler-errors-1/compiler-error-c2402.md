# <a name="compiler-error-c2402"></a>Compilerfehler C2402
'Bezeichner' : Das Register muss in 'Kontext' Index sein.  
  
 Das in einen indirekten Arbeitsspeicheroperanden verwendete Register muss ein Indexregister in diesem Kontext sein.
