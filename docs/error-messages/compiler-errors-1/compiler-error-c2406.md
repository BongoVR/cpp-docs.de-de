# Compilerfehler C2406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': Name 'context' undefiniert  
  
 Ein undefinierter Bezeichner wird mit dem `SIZE`\-, dem `LENGTH`\- oder dem Memberauswahl\- operator \(.\) verwendet.
