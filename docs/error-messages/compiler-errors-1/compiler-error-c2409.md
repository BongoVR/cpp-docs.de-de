# Compilerfehler C2409
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unzulässiger Typ als Operand in 'Kontext' verwendet.  
  
 Der Typ ist in diesem Kontext als Operator unzulässig.
