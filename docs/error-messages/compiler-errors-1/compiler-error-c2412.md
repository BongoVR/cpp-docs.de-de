# Compilerfehler C2412
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Marke': Marke, bei der nicht zwischen Groß\- und Kleinschreibung unterschieden wird, wurde neu definiert  
  
 Die Marke wurde innerhalb der aktuellen Funktion mehr als einmal definiert.  Ändern Sie die Schreibweise der Marke und die zugehörigen Verweise.
