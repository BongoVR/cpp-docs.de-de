# <a name="compiler-error-c2412"></a>Compilerfehler C2412
'Bezeichnung': Groß-/Kleinschreibung Bezeichnung wurde neu definiert  
  
 Die Bezeichnung ist mehr als einmal in die aktuelle Funktion definiert. Ändern Sie die Schreibweise der Bezeichnung und seinen verweisen.
