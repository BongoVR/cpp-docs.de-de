# Compilerfehler C2413
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Token': Unzulässige Ausrichtungsgröße  
  
 Die Ausrichtungsgröße, die mit der `ALIGN`\-Direktive verwendet wurde, fehlt entweder oder befindet sich außerhalb des gültigen Bereichs.
