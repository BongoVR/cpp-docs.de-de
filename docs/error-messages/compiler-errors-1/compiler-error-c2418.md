# Compilerfehler C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Löschen der Browserdatei nicht möglich: Dateiname  
  
 Der Compiler konnte die Browserdatei nicht löschen.  
  
### Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Die Datei ist schreibgeschützt.  
  
2.  Die Datei wird von einem anderen Prozess verwendet.
