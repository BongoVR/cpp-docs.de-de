# Compilerfehler C2419
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Modulo Null in "Kontext"  
  
 Der Parameter, der rechts neben dem `MOD`\-Operator steht, ist in diesem Kontext gleich null.
