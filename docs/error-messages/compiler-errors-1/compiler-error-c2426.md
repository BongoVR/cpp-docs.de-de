# Compilerfehler C2426
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Token': Unzulässiger Operator in 'Kontext'  
  
 Das Token kann in diesem Kontext nicht als Operator verwendet werden.  Indexoperatoren können z. B. nicht geschachtelt werden.
