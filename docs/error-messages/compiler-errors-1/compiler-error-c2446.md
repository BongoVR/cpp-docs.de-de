# Compilerfehler C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Operator': keine Konvertierung von 'Typ1' in 'Typ2'  
  
 Der Compiler kann `type1` nicht in `type2` konvertieren.  Eine Konvertierung macht möglicherweise keinen Sinn, da sie gegen die Regeln der C\/C\+\+\-Semantik verstößt.
