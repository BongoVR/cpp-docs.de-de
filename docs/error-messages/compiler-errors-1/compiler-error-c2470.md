# <a name="compiler-error-c2470"></a>Compilerfehler C2470

 Eine Funktionsdefinition fehlt die Argumentliste.  
  
 Im folgende Beispiel wird C2470 generiert:  
  
```  
// C2470.cpp  
int MyFunc {};  // C2470  
void MyFunc2() {};  //OK  
  
int main(){  
   MyFunc();  
   MyFunc2();  
}  
```
