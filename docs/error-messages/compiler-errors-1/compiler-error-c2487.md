# Compilerfehler C2487
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner': Member der Klasse mit DLL\-Schnittstellen darf nicht mit DLL\-Schnittstelle deklariert werden  
  
 Sie können eine ganze Klasse oder bestimmte Member einer Nicht\-DLL\-Schnittstellenklasse mit DLL\-Schnittstelle deklarieren.  Es ist jedoch nicht möglich, erst eine Klasse mit DLL\-Schnittstelle und anschließend einen Member dieser Klasse mit DLL\-Schnittstelle zu deklarieren.
