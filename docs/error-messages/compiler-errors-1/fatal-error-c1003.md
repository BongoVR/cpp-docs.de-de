# Schwerwiegender Fehler C1003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Mehr als „Anzahl“ Fehler gefunden; Kompilierung wird abgebrochen.  
  
 Die Anzahl der Fehler im Programm ist zu hoch, um eine Wiederherstellung zu gestatten. Der Compiler muss beendet werden.
