# Schwerwiegender Fehler C1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unbekannte Flagzeichenfolge in der Option  
  
 Die Befehlszeilenoption enthält eine ungültige Zeichenfolge. Überprüfen Sie die **CL**\-Befehlszeile und \-Umgebungsvariable auf Fehler.
