# Schwerwiegender Fehler C1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Keine Quelldatei angegeben  
  
 Dem Compiler wurde keine zu kompilierende C\- oder C\+\+\-Quelldatei bereitgestellt. Überprüfen Sie die **CL**\-Befehlszeile und \-Umgebungsvariable hinsichtlich Angaben zu Dateinamen.
