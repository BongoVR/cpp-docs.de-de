# <a name="fatal-error-c1035"></a>Schwerwiegender Fehler C1035
Ausdruck zu komplex. Vereinfachen Sie den Ausdruck.  
  
 Der Compiler konnte den Code für einen komplexen Ausdruck nicht generieren. Unterteilen Sie den Ausdruck in einfachere Ausdrücke, und kompilieren Sie anschließend erneut.
