# Schwerwiegender Fehler C1038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Compilerlimit: "Funktion": Ablaufsteuerung zu komplex, vereinfachen Sie die Funktion.  
  
 Die Funktion hat mehr Ablaufsteuerungsstatus, als der Compiler verarbeiten kann. Vereinfachen Sie die Ablaufsteuerung, oder teilen Sie die Funktion in kleinere Funktionen auf.
