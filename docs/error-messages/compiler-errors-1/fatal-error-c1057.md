# Schwerwiegender Fehler C1057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unerwartetes Dateiende bei der Erweiterung eines Makros  
  
 Der Compiler hat beim Abfragen von Argumenten für Makroaufrufe – möglicherweise aufgrund einer fehlenden schließenden Klammer im Makroaufruf – das Ende der Quelldatei erreicht.
