# Schwerwiegender Fehler C1064
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Compilerlimit: Überlauf des internen Puffers durch Token  
  
 Die für Bezeichner verwendete interne Pufferlänge wird von einem Bezeichner überschritten.  Kürzen Sie den Namen.
