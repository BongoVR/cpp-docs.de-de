# Schwerwiegender Fehler C1073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Interner Fehler im Zusammenhang mit der inkrementellen Kompilierung \(Compilerdatei 'Dateiname', Zeilennummer\)  
  
 Kompilieren Sie die Datei ohne inkrementelle Kompilierung neu.
