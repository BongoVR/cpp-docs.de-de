# Schwerwiegender Fehler C1075
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Das linke Token weist am Dateiende keine Entsprechung auf.  
  
 Der Compiler hat ein übereinstimmendes *Token* vor dem Erreichen des Dateiendes erwartet.  
  
 Dieser Fehler kann durch eine nicht übereinstimmende eckige Klammer, geschweifte Klammer oder ein anderes paarweise auftretendes Zeichen verursacht werden.
