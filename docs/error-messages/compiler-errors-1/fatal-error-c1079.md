# Schwerwiegender Fehler C1079
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Compilerlimit: Maximale Größe für PCH\-Datei überschritten  
  
 Die PCH\-Datei überschreitet die maximale Größe von 4 GB.
