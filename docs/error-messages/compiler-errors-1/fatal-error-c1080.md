# Schwerwiegender Fehler C1080
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Compilerlimit: Befehlszeilenoption überschreitet die Grenze von**   
 ***Anzahl* Zeichen**  
  
 Ein an den Compiler übergebenes Argument verfügt über mehr als 256 Zeichen.
