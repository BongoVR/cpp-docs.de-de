# Schwerwiegender Fehler C1085
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Schreiben in Dateityp\-Datei nicht möglich: 'Datei': Meldung  
  
### Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Das Laufwerk ist schreibgeschützt.  
  
2.  Das Laufwerk ist voll.  
  
3.  Zugriffsverletzung.  
  
4.  Wenn eine ungültige Dateinummer gemeldet wird, wurde die Datei möglicherweise im Vordergrund geschlossen, während die Kompilierung im Hintergrund lief.
