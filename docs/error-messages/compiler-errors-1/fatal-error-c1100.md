# Schwerwiegender Fehler C1100
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**OLE kann nicht initialisiert werden:**   
 ***Systemfehlermeldung***  
  
 Der Compiler kann die COM \(Component Object Model\)\-Bibliothek nicht initialisieren.  Siehe [CoInitialize](http://msdn.microsoft.com/library/windows/desktop/ms678543).
