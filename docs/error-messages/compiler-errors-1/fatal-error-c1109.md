# <a name="fatal-error-c1109"></a>Schwerwiegender Fehler C1109
Der 'einstiegspunkt' in der DLL 'dll' wurde nicht gefunden  
  
 Ein vom Compiler benötigter Einstiegspunkt in einer [verzögert geladenen DLL](../../build/reference/linker-support-for-delay-loaded-dlls.md) wurde nicht gefunden.
