# Schwerwiegender Fehler C1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#using\-Fehler bei "Datei"  
  
 Nur Dateien im Microsoft Intermediate Language \(MSIL\)\-Format können an eine [\#using](../../preprocessor/hash-using-directive-cpp.md)\-Direktive übergeben werden.  Eine MSIL\-Ausgabedatei kann mithilfe der [\/clr](../../build/reference/clr-common-language-runtime-compilation.md)\-Compileroption erstellt werden.  MSIL\-Dateien werden auch in anderen [!INCLUDE[vsprvs](../../assembler/masm/includes/vsprvs_md.md)]\-Sprachen generiert.
