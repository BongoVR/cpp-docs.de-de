# Schwerwiegender Fehler C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aufruf von CryptoAPI schlug fehl  
  
 Ein Aufruf der CryptoAPI durch den Compiler ist fehlgeschlagen.  Installieren Sie Visual Studio neu. Installieren Sie ggf. auch das Betriebssystem neu.
