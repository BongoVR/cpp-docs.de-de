# Schwerwiegender Fehler C1201
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fortfahren nach einem Syntaxfehler in der Klassenvorlagendefinition nicht möglich.  
  
 Beim Analysieren der Definition einer Klassenvorlage ist ein unerwarteter Fehler aufgetreten.  
  
 Beheben Sie alle anderen Fehler, und kompilieren Sie den Code neu. Wenn dies nicht erfolgreich war, notieren Sie die Umstände des Fehlers, versuchen Sie, das Problem zu isolieren, und erstellen Sie einen reproduzierbaren Testfall. Wenden Sie sich erst dann an den [Produktsupport von Microsoft](../Topic/Talk%20to%20Us.md).
