# Schwerwiegender Fehler C1309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nicht übereinstimmende Versionen von C2.DLL und PGODB\<version\>.DLL  
  
 Das von Ihnen verwendete Toolset zum Erstellen und Verwenden von [Profile\-Guided Optimizations](../../build/reference/profile-guided-optimizations.md) enthält nicht übereinstimmende Komponenten.  Wenn Sie diesen Fehler nicht manuell beheben können, installieren Sie Visual C\+\+ neu.
