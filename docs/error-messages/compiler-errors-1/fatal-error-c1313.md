# Schwerwiegender Fehler C1313
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Compilerlimit: \<Typ\>\-Blöcke dürfen nicht tiefer als \<Anzahl\> Ebenen geschachtelt werden  
  
 Blöcke für die Ausnahmebehandlung \(oder strukturierte Ausnahmebehandlung\) wurden zu tief geschachtelt.  Vereinfachen Sie den Code.
