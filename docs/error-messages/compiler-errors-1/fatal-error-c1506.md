# Schwerwiegender Fehler C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Weiterverarbeitung nach der Auflösung von Blöcken nicht möglich  
  
 Der Block war zu umfangreich für die Kompilierung.  
  
### Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Nicht übereinstimmende geschweifte Klammern  
  
2.  Ungewöhnlich umfangreiche Funktion oder Klasse
