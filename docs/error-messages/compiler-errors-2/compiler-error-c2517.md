# Compilerfehler C2517
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**"**   
 ***Bezeichner* ": Rechter Teil von "::" undefiniert**  
  
 Der Bezeichner auf der rechten Seite des Bereichsauflösungsoperators \(`::`\) muss ein definierter Member einer links stehenden Klasse, Struktur oder Union sein.  Wenn keine Klasse, Struktur oder Union benannt ist, muss der rechts stehende Bezeichner mit globalem Gültigkeitsbereich deklariert werden.
