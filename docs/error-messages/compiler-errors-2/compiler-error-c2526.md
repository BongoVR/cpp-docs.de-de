# Compilerfehler C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner1': Funktion mit C\-Bindung kann C\+\+\-Klasse 'Bezeichner2' nicht zurückgeben  
  
 Eine Funktion, die mit C\-Bindung definiert wurde, kann keinen benutzerdefinierten Typ zurückgeben.
