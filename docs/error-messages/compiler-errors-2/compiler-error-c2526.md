# <a name="compiler-error-c2526"></a>Compilerfehler C2526
'Bezeichner1': Funktion der C-Bindung nicht C++-Klasse "Bezeichner2" zurückgeben  
  
 Eine Funktion mit C-Bindung definiert kann nicht über einen benutzerdefinierten Typ zurück.
