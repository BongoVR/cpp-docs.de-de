# Compilerfehler C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner': ungültiger Modifizierer für Verweis  
  
 Der Verweis wurde geändert.  Es gibt keine Möglichkeit, einen Verweis auf ein anderes Objekt zu setzen.  Verwenden Sie zu diesem Zweck einen Zeiger.
