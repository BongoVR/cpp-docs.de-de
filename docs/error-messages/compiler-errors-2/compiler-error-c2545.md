# Compilerfehler C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Operator': Überladener Operator nicht gefunden  
  
 Der Operator darf nicht mit den angegebenen Operanden verwendet werden.  Sie müssen einen überladenen Operator mit den erforderlichen Operanden angeben.  
  
 Dieser Fehler kann verursacht werden, wenn Operanden vom falschen Typ sind.  
  
 Dieser Fehler kann behoben werden, wenn Sie einen Konvertierungsoperator oder Konstruktor definieren, der einen einzelnen Parameter benötigt.
