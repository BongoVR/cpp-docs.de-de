# Compilerfehler C2566
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Überladene Funktion in einem Bedingungsausdruck  
  
 Eine überladene Funktion in einem Bedingungsausdruck kann nicht ausgewertet werden.
