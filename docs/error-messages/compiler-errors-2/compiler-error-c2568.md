# Compilerfehler C2568
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner1': Auflösung der Funktionsüberladung nicht möglich  
  
 Der Compiler kann nicht eindeutig feststellen, welche überladene Funktion aufgerufen werden soll.  Die tatsächlich an die Funktion übergebenen Parameter müssen so umgewandelt werden, dass sie mit den formalen Parametern für eine der überladenen Funktionen übereinstimmen, aber keine Übereinstimmung ist eindeutig besser als alle anderen.
