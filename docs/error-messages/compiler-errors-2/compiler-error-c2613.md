# Compilerfehler C2613
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Abschließendes Komma \(,\) in der Basisklassenliste ungültig.  
  
 Nach der letzten Basisklasse in einer Basisklassenliste steht ein Komma.
