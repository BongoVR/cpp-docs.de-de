# Compilerfehler C2649
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"Bezeichner": ist kein "Klassenschlüssel"  
  
 In der Deklaration einer Klasse, Struktur oder Union wird ein unzulässiges Tag verwendet.
