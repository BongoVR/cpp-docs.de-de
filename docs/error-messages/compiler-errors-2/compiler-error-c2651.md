# <a name="compiler-error-c2651"></a>Compilerfehler Fehler C2651
"Datentyp": links von 'Operator' muss eine Klasse, Struktur oder Union sein  
  
 Um eine Template-Parameter verwenden, als ob es sich um eine Klasse ist, die Klassenvorlage mit einer Klasse anstelle eines ganzzahligen Typs spezialisiert werden.
