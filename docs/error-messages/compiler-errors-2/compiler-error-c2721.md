# Compilerfehler C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Spezifizierer': Speicherklassenbezeichner zwischen dem Schlüsselwort 'operator' und dem Typ unzulässig  
  
 Konvertierungen benutzerdefinierter Typen wirken sich auf alle Speicherklassen aus. Folglich kann in einer Typkonvertierung keine Speicherklasse festgelegt werden.
