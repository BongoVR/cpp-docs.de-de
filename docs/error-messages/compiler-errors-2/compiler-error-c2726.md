# Compilerfehler C2726
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"gcnew" kann nur zum Erstellen eines Objekts mit verwaltetem oder WinRT\-Typ verwendet werden.

```  
// C2726.cpp
```  
