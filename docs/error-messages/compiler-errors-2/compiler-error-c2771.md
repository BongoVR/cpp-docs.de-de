# Compilerfehler C2771
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import nur im globalen Gültigkeitsbereich oder im Namespace\-Gültigkeitsbereich zulässig  
  
 Die `#import`\-Direktive ist z. B. nicht in einer Funktion oder Struktur zulässig.
