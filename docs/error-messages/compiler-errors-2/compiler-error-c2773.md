# Compilerfehler C2773
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import und \#using sind nur im C\+\+\-Compiler verfügbar  
  
 Der C\-Compiler erkennt die `#import`\-Präprozessordirektive nicht.  Kompilieren Sie die Quelle als C\+\+.  Verwenden Sie, falls erforderlich, [\/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md).
