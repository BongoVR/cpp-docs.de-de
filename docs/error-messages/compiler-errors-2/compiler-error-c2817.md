# Compilerfehler C2817
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Rückgabetyp für Operator delete muss "void" sein  
  
 Eine überladene Funktion für einen [Operator delete](../Topic/operator%20delete%20\(%3Cnew%3E\).md) kann keinen Wert zurückgeben.
