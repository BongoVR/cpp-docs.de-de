# Compilerfehler C2829
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Variable Parameterliste für 'Operator Operator' nicht möglich  
  
 Variable Parameterlisten können nur mit zwei Operatoren verwendet werden:  
  
-   Funktionsaufruf `()`  
  
-   `new`
