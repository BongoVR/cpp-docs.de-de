# <a name="compiler-error-c2839"></a>Compilerfehler C2839
Ungültiger Rückgabetyp "Typ" für überladene "Operator ->"  
  
 Die `->` Operator muss eine Klasse, Struktur oder Union oder ein Verweis auf eine zurückgeben.
