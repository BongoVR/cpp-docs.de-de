# Compilerfehler C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ungültiger Rückgabetyp "Typ" für überladenen "Operator \-\>"  
  
 Der Operator `->` muss eine Klasse, Struktur oder Union bzw. einen Verweis darauf zurückgeben.
