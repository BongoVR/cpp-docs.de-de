# <a name="compiler-error-c2855"></a>Compilerfehler C2855
die Befehlszeilenoption "Option" inkonsistent mit der vorkompilierten Headerdatei  
  
 Die angegebene Befehlszeilenoption unterscheidet sich von der Option zum Erstellen des vorkompilierten Headers verwendet. Den vorkompilierten Header mit der angegebenen Befehlszeilenoption neu generiert werden.
