# Compilerfehler C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Befehlszeilenoption 'Option' steht in Konflikt mit der vorkompilierten Headerdatei  
  
 Die angegebene Befehlszeilenoption unterscheidet sich von der Option, die zum Erstellen des vorkompilierten Headers verwendet wurde.  Generieren Sie den vorkompilierten Header mit der angegebenen Befehlszeilenoption neu.
