# Compilerfehler C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Dateiname ist nicht die Typdatei, die beim Erstellen dieses vorkompilierten Headers verwendet wurde. Erstellen Sie den vorkompilierten Header neu.  
  
 Projektdatenbank\- und vorkompilierte Headerdateien müssen gemeinsam erstellt werden, um konsistente Informationen zu gewährleisten.  Erstellen Sie das Projekt neu, um den vorkompilierten Header neu zu erstellen.  
  
 Weitere Informationen zu vorkompilierten Headern finden Sie unter [\/Y \(Vorkompilierte Header\)](../../build/reference/y-precompiled-headers.md).
