# Compilerfehler C2945
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Explizite Instanziierung verweist nicht auf die Spezialisierung einer Vorlagenklasse  
  
 Sie können keine explizite Instanziierung für etwas durchführen, das nicht auf einer Vorlage basiert.
