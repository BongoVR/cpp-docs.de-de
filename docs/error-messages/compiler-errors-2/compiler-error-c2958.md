# Compilerfehler C2958
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Für das linke Trennzeichen an 'Position' gibt es keine Entsprechung  
  
 Es fehlt eine Entsprechung zu einem Trennzeichen.  
  
 Dieser Fehler kann durch eine nicht übereinstimmende Klammer in einer Funktionsvorlagendeklaration verursacht werden.
