# <a name="compiler-error-c3037"></a>Compilerfehler C3037
"var": Variable in reduction-Klausel muss im übergeordneten Kontext freigegeben sein.  
  
 Eine Variable, die im angegebenen eine [Verringerung](../../parallel/openmp/reference/reduction.md) -Klausel kann nicht für jeden Thread im Kontext privat sein.  
  
 Im folgenden Beispiel wird C3037 generiert:  
  
```  
// C3037.cpp  
// compile with: /openmp /c  
int g_i;  
  
int main() {  
   int i;  
  
   #pragma omp parallel private(g_i)  
   // try the following line instead  
   // #pragma omp parallel   
   {  
      #pragma omp for reduction(+:g_i)   // C3037  
      for (i = 0 ; i < 10 ; ++i) {  
         g_i += i;  
      }  
   }  
}  
```
