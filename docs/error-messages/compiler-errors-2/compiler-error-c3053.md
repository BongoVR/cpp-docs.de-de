# <a name="compiler-error-c3053"></a>Compilerfehler C3053
'symbol': 'threadprivate' ist nur für globale oder statische Datenelemente gültig.  
  
 Symbole an übergeben [Threadprivate](../../parallel/openmp/reference/threadprivate.md) muss entweder die global oder statisch sein.  
  
 Das folgende Beispiel generiert C3053:  
  
```  
// C3053.cpp  
// compile with: /openmp  
void Test() {  
   int x, y;  
   #pragma omp threadprivate(x, y)   // C3053  
   #pragma omp parallel copyin(x, y)  
   {  
      x = y;  
   }  
}  
```  
  
 Mögliche Lösung:  
  
```  
// C3053b.cpp  
// compile with: /openmp /LD  
int x, y;  
#pragma omp threadprivate(x, y)  
  
void Test() {  
   #pragma omp parallel copyin(x, y)  
   {  
      x = y;  
   }  
}  
```
