# Compilerfehler C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Eigenschaft': Eine Eigenschaft kann keinen Typ 'const' oder 'volatile' besitzen  
  
 Die Schlüsselwörter [const](../../cpp/const-cpp.md) und [volatile](../../cpp/volatile-cpp.md) sind für Eigenschaften nicht zulässig.
