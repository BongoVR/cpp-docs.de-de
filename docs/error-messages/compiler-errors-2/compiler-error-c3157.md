# Compilerfehler C3157
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Das ParamArray\-Attribut kann nur auf den letzten Parameter angewendet werden  
  
 Das [ParamArray](https://msdn.microsoft.com/en-us/library/system.paramarrayattribute.aspx)\-Attribut wurde auf den falschen Parameter angewendet.
