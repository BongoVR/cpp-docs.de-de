# <a name="compiler-error-c3320"></a>Compilerfehler C3320
'Typ': Typ kann nicht den gleichen Namen besitzen wie die name-Moduleigenschaft.  
  
Ein exportierter benutzerdefinierten Typ (UDT), die eine Struktur, Klasse, Enum oder Union sein k√∂nnte, kann nicht den gleichen Namen haben, wie der Parameter, die an die [Modul](../../windows/module-cpp.md) Name-Eigenschaft des Attributs.  
  
## <a name="example"></a>Beispiel  
Im folgenden Beispiel wird C3320 generiert:  
  
```cpp  
// C3320.cpp  
#include "unknwn.h"  
[module(name="xx")];  
  
[export] struct xx {   // C3320  
// Try the following line instead  
// [export] struct yy {  
   int i;  
};  
```
