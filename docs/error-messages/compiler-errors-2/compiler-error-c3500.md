# <a name="compiler-error-c3500"></a>Compilerfehler C3500
Ungültiges ProgID "progid"  
  
 Ein ungültiges progid wurde angegeben, mit der `#import` Anweisung. Überprüfen Sie die Windows-Registrierung, um sicherzustellen, dass Sie eine gültige angeben progid.
