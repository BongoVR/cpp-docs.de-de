# <a name="compiler-error-c3555"></a>Compilerfehler C3555
Falsches Argument für "decltype"  
  
 Das Argument für den `decltype(``expression``)` -Typspezifizierer ist kein gültiger Ausdruck.  
  
> [!NOTE]
>  C3555 tritt wahrscheinlich nicht auf. Wenn der Compiler C3555 ausgibt, ist es wahrscheinlich, dass ein interner Compilerfehler aufgetreten ist.
