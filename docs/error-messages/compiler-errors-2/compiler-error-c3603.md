# Compilerfehler C3603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Symbol': Typ 'Typ' wird noch nicht unterstützt  
  
 Sie haben versucht, einen Datentyp zu verwenden, der von .NET Runtime in einem verwalteten Objekt noch nicht unterstützt wird.
