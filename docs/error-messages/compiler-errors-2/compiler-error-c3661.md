# <a name="compiler-error-c3661"></a>Compilerfehler C3661
Explizites Überschreiben Liste keine Methoden zum Überschreiben nicht gefunden.  
  
 Ein explizites Überschreiben angegeben, eine oder mehrere Typnamen.  Es wurde jedoch keine Funktion mit der Signatur in den Typen, die den überschreibenden Funktionssignatur abgeglichen.  Wenn Sie versuchen, überschreiben Sie basierend auf den Typnamen, muss in den angegebenen Typen, die die Signatur der überschreibenden Funktion entsprechen eine oder mehrere virtuelle Funktionen sein.  
  
 Weitere Informationen finden Sie unter [explizite Überschreibungen](../../windows/explicit-overrides-cpp-component-extensions.md).
