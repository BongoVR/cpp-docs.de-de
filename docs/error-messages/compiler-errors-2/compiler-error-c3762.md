# Compilerfehler C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Das Attribut 'Attribut' kann nicht verarbeitet werden  
  
 Ein benutzerdefiniertes Attribut, das von `System.Security.Permissions.SecurityAttribute` erbt, wird verwendet, um ein Sicherheitsattribut zu definieren.  Ein solches Attribut kann nicht in der Assembly verwendet werden, in der es definiert wird.
