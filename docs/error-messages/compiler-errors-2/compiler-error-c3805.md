# Compilerfehler C3805
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Token': unerwartetes Token, '} ' oder Bezeichner wurde erwartet  
  
 Bei der Definition einer Eigenschaft wurde ein ungültiges Token gefunden.  Entfernen Sie das ungültige Token.
