# <a name="compiler-error-c3812"></a>Compilerfehler C3812
"Property" muss das erste Token in einer Eigenschaftendeklaration sein.  
  
 Wenn Sie eine Eigenschaft deklarieren die `__property` Schlüsselwort muss das erste Token in der Zeile sein.  
  
 C3812 ist nur über die veraltete Compileroption erreichbar **/CLR: oldSyntax**.
