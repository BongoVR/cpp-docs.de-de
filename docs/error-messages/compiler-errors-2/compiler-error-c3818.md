 Eine Überladung ist nicht möglich, dass Eigenschaften, wenn eine einen Indexer ist und der andere eine Arrayeigenschaft. 
  
 C3818 ist nur über die veraltete Compileroption erreichbar **/CLR: oldSyntax**.
