# <a name="compiler-error-c3818"></a>Compilerfehler C3818
Array-Eigenschaftsdeklaration "Eigenschaft1" sollte keine Indexeigenschaft 'Eigenschaft2' überladen  
  
 Eine Überladung ist nicht möglich, dass Eigenschaften, wenn eine ein Indexer und die andere eine Arrayeigenschaft ist. 
  
 C3818 ist nur erreichbar, mit der veralteten-Compileroption **/CLR: oldSyntax**.
