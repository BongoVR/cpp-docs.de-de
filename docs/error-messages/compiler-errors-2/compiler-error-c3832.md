# Compilerfehler C3832
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Typbibliothek': Die Typbibliothek wurde für 32\-Bit\-Zeiger erstellt. Ändern Sie den 'ptrsize'\-Qualifizierer  
  
 Mit dem `ptrsize`\-Attribut der [\#import](../../preprocessor/hash-import-directive-cpp.md)\-Direktive angegebene, explizite Informationen stimmen nicht mit dem überein, was der Compiler in der Typbibliothek gefunden hat.
