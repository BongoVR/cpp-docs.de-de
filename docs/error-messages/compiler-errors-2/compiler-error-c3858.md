# <a name="compiler-error-c3858"></a>Compilerfehler C3858
'Typ': kann im aktuellen Bereich nicht erneut deklariert werden  
  
 Der Typ kann nicht zweimal im selben GÃ¼ltigkeitsbereich deklariert werden.  
  
 Im folgende Beispiel wird C3858 generiert:  
  
```  
// C3858.cpp  
// compile with: /LD  
template <class T>  
struct Outer  
{  
   struct Inner;  
};  
  
template <class T>  
struct Outer<T>::Inner;   // C3858  
// try the following line instead  
// struct Outer<T>::Inner{};  
```
