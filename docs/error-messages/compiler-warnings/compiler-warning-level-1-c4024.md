# Compilerwarnung (Stufe 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion': Unterschiedliche Typen für formalen und übergebenen Parameter 'Nummer'  
  
 Der formale und der entsprechende tatsächlich übergebene Parameter haben unterschiedliche Typen.  Der tatsächliche Parameter wird vom Compiler unverändert übergeben.  Die empfangende Funktion wandelt den Parametertyp in den erwarteten Typ um.
