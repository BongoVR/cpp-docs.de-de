# Compilerwarnung (Stufe 1) C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof gibt 0 zurück  
  
 Der Operator `sizeof` wird auf einen Operanden der Größe 0 \(leere Struktur, Union oder Klasse bzw. Aufzählungstyp oder `void`\-Typ\) angewendet.
