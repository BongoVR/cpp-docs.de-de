# <a name="compiler-warning-level-1-c4038"></a>Compilerwarnung (Stufe 1) C4038
"Modifizierer": unzulässiger Klassenmodifizierer  
  
 Dieser Modifizierer kann nicht verwendet werden, für Klassen mit **Dllimport** oder [Dllexport](../../cpp/dllexport-dllimport.md) Attribute.
