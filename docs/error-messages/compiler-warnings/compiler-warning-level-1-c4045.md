# <a name="compiler-warning-level-1-c4045"></a>Compilerwarnung (Stufe 1) C4045
"array": Arraygrenze überschritten  
  
 Das Array hat zu viele Initialisierungen. Zusätzliche Initialisierungen werden ignoriert.  
  
 Stellen Sie sicher, dass Arrayelemente und Initialisierungen in Größe und Anzahl übereinstimmen.
