# Compilerwarnung (Stufe 1) C4048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unterschiedlich deklarierte Arrayindizes: 'Bezeichner1' und 'Bezeichner2'  
  
 Ein Ausdruck enthält Zeiger auf Arrays unterschiedlicher Größen.  Die Zeiger werden ohne Konvertierung verwendet.  
  
 Diese Warnung kann eventuell durch eine Typumwandlung der Arrays in den gleichen oder einen äquivalenten Typ behoben werden.
