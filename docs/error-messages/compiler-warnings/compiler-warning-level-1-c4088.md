# Compilerwarnung (Stufe 1) C4088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion': Zeigerkonflikt im übergebenen Parameter 'Nummer', formalen Parameter 'Nummer'  
  
 Die entsprechenden formalen und tatsächlich übergebenen Parameter weisen unterschiedliche Dereferenzierungsebenen auf.  Der tatsächliche Parameter wird unverändert übergeben.  Die aufgerufene Funktion interpretiert seinen Wert als Zeiger.
