# <a name="compiler-warning-level-1-c4113"></a>Compilerwarnung (Stufe 1) C4113
"Bezeichner1" unterscheidet sich in Parameterlisten von "Bezeichner2"  
  
 Ein Funktionszeiger wird in einem anderen Funktionszeiger zugewiesen, aber die Listen der formalen Parameter der Funktionen stimmen nicht überein. Die Zuweisung wird unverändert kompiliert.
