# Compilerwarnung (Stufe 1) C4116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unbenannte Typdefinition in runden Klammern  
  
 Eine nicht benannte Struktur oder Union bzw. ein unbenannter Aufzählungstyp wurde in einem Ausdruck definiert, der in Klammern eingeschlossen ist.  Die Typdefinition ist ohne Bedeutung.  
  
 In einem C\-Funktionsaufruf gilt die Definition global.  In einem C\+\+\-Funktionsaufruf hat die Definition denselben Gültigkeitsbereich wie die aufgerufene Funktion.
