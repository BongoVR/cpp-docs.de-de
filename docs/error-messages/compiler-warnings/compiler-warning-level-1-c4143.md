# Compilerwarnung (Stufe 1) C4143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Pragma "same\_seg" wird nicht unterstützt; reservieren Sie Segmente mit \_\_based  
  
 Das Pragma **\#pragma same\_seg** wird nicht mehr unterstützt. Verwenden Sie stattdessen das [\_\_based](../../cpp/based-pointers-cpp.md)\-Schlüsselwort.
