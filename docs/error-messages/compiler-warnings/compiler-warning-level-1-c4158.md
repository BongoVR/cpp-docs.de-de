# <a name="compiler-warning-level-1-c4158"></a>Compilerwarnung (Stufe 1) C4158
Annahme von #pragma pointers_to_members(full_generality, inheritance)  
  
 Ein **#pragma Pointers_to_members (***einzelne &#124; mehrere &#124; virtuelle***)** wurde ohne begleitendes ausgestellt **#pragma Pointers_to_ Members(full_generality)**.
