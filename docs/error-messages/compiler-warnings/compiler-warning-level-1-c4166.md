# Compilerwarnung (Stufe 1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Unzulässige Aufrufkonvention für Konstruktor\/Destruktor**  
  
 Konstruktoren und Destruktoren können keine anderen Aufrufkonventionen als den Standardwert für die Plattform besitzen \(außer wenn Sie explizit **\_\_clrcall** angeben\).
