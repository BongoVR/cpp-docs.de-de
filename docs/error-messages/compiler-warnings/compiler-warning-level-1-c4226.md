# Compilerwarnung (Stufe 1) C4226
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nicht dem Standard entsprechende Erweiterung: 'Schlüsselwort' ist ein veraltetes Schlüsselwort  
  
 In der aktuellen Version von Visual C\+\+ wird dieses Schlüsselwort nicht verwendet.
