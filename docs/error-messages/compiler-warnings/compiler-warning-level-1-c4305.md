# Compilerwarnung (Stufe 1) C4305
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner': Verkürzung von 'Typ1' in 'Typ2'  
  
 Der Bezeichner wurde in einen kleineren Typ konvertiert, was zu einem Datenverlust führte.  
  
 Wenn Sie den ATL\-Verbindungspunkt\-Assistenten verwenden, finden Sie weitere Informationen im folgenden Knowledge Base\-Artikel:  
  
-   Q250847: ATL Connection Point Wizard Generated Code for Event with VARIANT Argument Gives C4305 Warning \(nur auf Englisch verfügbar\)
