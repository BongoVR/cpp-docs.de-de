# Compilerwarnung (Stufe 1) C4340
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Wert': Umbruch für Wert von positivem zu negativem Wert  
  
 Der `enum`\-Wert ist größer als der größte positive `enum`\-Wert, der in einen negativen Wert umbrochen wurde.
