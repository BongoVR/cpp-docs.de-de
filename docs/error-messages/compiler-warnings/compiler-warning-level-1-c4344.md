# Compilerwarnung (Stufe 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Verhaltensänderung: Verwendung von expliziten Vorlagenargumenten löste einen Aufruf an 'Funktion' aus  
  
 Durch einen Funktionsaufruf unter Verwendung expliziter Vorlagenargumente wird eine andere Funktion als beim Verzicht auf explizite Argumente aufgerufen.
