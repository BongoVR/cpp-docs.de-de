# Compilerwarnung (Stufe 1) C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die nicht öffentliche 'Methode2'\-Methode überschreibt 'Methode1' nicht  
  
 Ein Typ, der einen anderen Typ implementiert, hat eine Überschreibungsmethode definiert, die Überschreibung ist jedoch nicht öffentlich.  Daher überschreibt die Methode die Basistypmethode nicht.
