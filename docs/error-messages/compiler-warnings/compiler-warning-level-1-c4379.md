# Compilerwarnung (Stufe 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Version Version der Common Language Runtime wird von diesem Compiler nicht unterstützt.Die Verwendung dieser Version kann zu unerwarteten Ergebnissen führen.  
  
 Auf Ihrem Computer ist eine frühere Version der Common Language Runtime installiert, nicht die aktuelle.  Um den Fehler C4379 zu beheben, installieren Sie die Version der Common Language Runtime, die mit dem Compiler mitgeliefert wurde.
