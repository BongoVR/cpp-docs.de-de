# Compilerwarnung (Stufe 1) C4381
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion1': Die Schnittstellenmethode wird von der nicht öffentlichen 'Funktion2'\-Methode nicht implementiert  
  
 Eine Klasse muss alle Funktionen in einer Schnittstelle implementieren.  Eine Klasse kann diese Bedingung erfüllen, wenn eine ihrer Basisklassen die Funktion implementiert.  Die Funktion muss jedoch als öffentliche Funktion implementiert werden.
