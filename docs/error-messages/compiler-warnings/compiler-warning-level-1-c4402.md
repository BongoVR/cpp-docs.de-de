# Compilerwarnung (Stufe 1) C4402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Der Operator PTR muss verwendet werden  
  
 Ein Typ wird ohne Operator PTR für einen Operanden verwendet, wenn auf einen Typ in Inlineassemblycode verwiesen bzw. eine Umwandlung in diesen Typ durchgeführt wird.  
  
 Der Compiler geht davon aus, dass der Operator PTR verwendet wird.
