# Compilerwarnung (Stufe 1) C4403
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unzulässiger Operator PTR  
  
 Der Operator PTR wurde in unzulässiger Weise im Inlineassemblercode verwendet.  
  
 Der Operator PTR wurde vom Compiler ignoriert.
