# Compilerwarnung (Stufe 1) C4406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Operand für Direktive ignoriert  
  
 Die Direktive arbeitet nicht mit Operanden, dennoch wurde ein Operand angegeben.  
  
 Der Compiler berücksichtigt den \(oder die\) angegebenen Operanden nicht.
