# Compilerwarnung (Stufe 1) C4409
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unzulässige Anweisungsgröße  
  
 Die Anweisung hat kein Format der angegebenen Größe.  Es wurde die kleinstmögliche zulässige Größe verwendet.
