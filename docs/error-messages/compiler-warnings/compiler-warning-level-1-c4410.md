# Compilerwarnung (Stufe 1) C4410
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unzulässige Größe für Operanden  
  
 Einer der für die Anweisung angegebenen Operanden hat eine unzulässige Größe.  Es wurde die kleinstmögliche zulässige Größe für den Operanden verwendet.
