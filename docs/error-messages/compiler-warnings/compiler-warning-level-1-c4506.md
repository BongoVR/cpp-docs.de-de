# Compilerwarnung (Stufe 1) C4506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Keine Definitionen für die Inlinefunktion 'Funktion'  
  
 Die angegebene Funktion wurde deklariert und für das Inlining gekennzeichnet, aber nicht definiert.  
  
 Der Compiler hat die Funktion nicht als Inlinefunktion behandelt.  
  
 Stellen Sie sicher, dass externe Inlinefunktionen mit dem Schlüsselwort `extern` deklariert werden.
