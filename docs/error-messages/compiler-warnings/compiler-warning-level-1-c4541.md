# Compilerwarnung (Stufe 1) C4541
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Bezeichner' für polymorphen Typ 'Typ' mit \/GR\- verwendet; unvorhersehbares Verhalten möglich  
  
 Sie haben versucht, ein Feature zu verwenden, das Laufzeit\-Typinformationen ohne Aktivierung derselben erfordert.  Wiederholen Sie die Kompilierung mit [\/GR](../../build/reference/gr-enable-run-time-type-information.md).
