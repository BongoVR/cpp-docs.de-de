# Compilerwarnung (Stufe 1) C4652
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Compileroption 'Option' steht in Konflikt mit der vorkompilierten Headerdatei; aktuelle Befehlszeilenoption Ã¼berschreibt die in der vorkompilierten Headerdatei definierte Option  
  
 Die angegebene Befehlszeilenoption unterschied sich von derjenigen, die beim Erstellen der vorkompilierten Headerdatei \(**.pch**\) angegeben wurde.  Die in der aktuellen Befehlszeile angegebene Option wurde verwendet.  
  
 Sie vermeiden diese Warnung, indem Sie die vorkompilierte Headerdatei mit der angegebenen Befehlszeilenoption neu generieren.
