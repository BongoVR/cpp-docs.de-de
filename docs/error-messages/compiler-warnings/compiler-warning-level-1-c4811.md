# Compilerwarnung (Stufe 1) C4811
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Wert von 'pragma conform\(forScope, show\) \=\= Wert'  
  
 Diese Warnung wird ausgegeben, wenn Sie die **show**\-Option des [conform](../../preprocessor/conform.md)\-Pragmas verwenden.*Wert* entspricht dem aktuellen conform\-Wert.
