# <a name="compiler-warning-level-1-c4811"></a>Compilerwarnung (Stufe 1) C4811
Wert von 'pragma conform(forScope, show) == Wert'  
  
 Diese Warnung wird ausgegeben, wenn Sie die **anzeigen** -Option von der [entsprechen](../../preprocessor/conform.md) Pragma. *Wert* entspricht dem aktuellen conform-Wert.
