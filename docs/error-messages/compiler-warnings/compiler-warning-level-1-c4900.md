# Compilerwarnung (Stufe 1) C4900
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Zwischensprachkonflikt zwischen „tool1“, Version „version1“, und „tool2“, Version „version2“  
  
 Die in *tool1* und *tool2* verwendete Zwischensprache stimmte nicht überein. Vergewissern Sie sich, dass jeweils die aktuellste Version der Tools installiert ist.
