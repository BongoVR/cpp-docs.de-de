# Compilerwarnung (Stufe 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Funktion': 'Attribut' muss angegeben werden  
  
 Ein für eine Funktion erforderliches Attribut wurde nicht explizit angegeben.  Beispielsweise erfordert die **main**\-Funktion das `__cdecl`\-Attribut.  Das Attribut ist für den Compiler zwingend erforderlich.
