# <a name="compiler-warning-level-2-c4948"></a>Compilerwarnung (Stufe 2) C4948
Rückgabetyp von 'Accessor' stimmt nicht mit den letzten Parametertyp des entsprechenden Setters überein.  
  
Der Compiler hat einen Konflikt zwischen Datentyp ist get und legen Sie für eine indizierte Eigenschaft gefunden.  
  
C4948 ist nur erreichbar, mit der veralteten-Compileroption **/CLR: oldSyntax**.
