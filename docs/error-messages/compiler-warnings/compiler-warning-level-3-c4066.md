# Compilerwarnung (Stufe 3) C4066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Mit Ausnahme des ersten Zeichens werden die Zeichen in einer "breiten" Zeichenkonstante ignoriert  
  
 Der Compiler verarbeitet nur das erste Zeichen einer Breitzeichenkonstante.
