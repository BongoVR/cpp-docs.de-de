# Compilerwarnung (Stufe 3) C4133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Typ': Nicht kompatible Typen \- von 'Typ1' bis 'Typ2'  
  
 Diese Warnung kann durch einen Versuch verursacht werden, zwei Zeiger verschiedener Typen zu subtrahieren.  
  
 Um diese Warnung zu vermeiden, verwenden Sie eine geeignete Typumwandlung.
