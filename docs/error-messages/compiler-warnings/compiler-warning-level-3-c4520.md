# Compilerwarnung (Stufe 3) C4520
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Klasse': Mehrere Standardkonstruktoren angegeben  
  
 Die Klasse verfügt über mehrere Standardkonstruktoren.  Der erste Konstruktor wird verwendet.
