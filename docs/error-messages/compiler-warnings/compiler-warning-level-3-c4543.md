# Compilerwarnung (Stufe 3) C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Der eingefügte Text wurde vom Attribut "no\_injected\_text" unterdrückt  
  
 Das [no\_injected\_text](../../windows/no-injected-text.md)\-Attribut kommt im Quellcode vor. Das bedeutet, dass der Compiler Attribute am Einfügen von Code hindert.  
  
 Diese Warnung weist Sie darauf hin, dass Code nicht über Attribute eingefügt werden kann.
