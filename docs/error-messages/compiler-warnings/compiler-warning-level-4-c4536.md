# Compilerwarnung (Stufe 4) C4536
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Typname': Der Typenname ist größer als das Metadatenlimit von 'Anzahl' Zeichen  
  
 Wenn es sich um einen verwalteten Typ handelte, würde ein Typname in den Metadaten verkürzt.  Weitere Informationen finden Sie unter [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md).  
  
 Diese Warnung ist standardmäßig deaktiviert.  Weitere Informationen finden Sie unter [Standardmäßig deaktivierte Compilerwarnungen](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
