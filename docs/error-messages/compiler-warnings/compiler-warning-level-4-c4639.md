# Compilerwarnung (Stufe 4) C4639
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MSXML\-Fehler, XML\-Dokumentkommentare werden nicht verarbeitet. Grund  
  
 Diese Warnmeldung kann aus beliebig vielen Gr√ºnden auftreten.  
  
 So vermeiden Sie diese Warnung:  
  
-   Kompilieren Sie erneut.  
  
-   Installieren Sie MSXML erneut, indem Sie die Common Language Runtime neu installieren.  
  
-   Bearbeiten oder entfernen Sie den Dokumentationskommentar, der die Warnung verursacht, und kompilieren Sie erneut.  
  
 Wenn C4639 ausgegeben wird, wird die weitere Verarbeitung von XML\-Kommentaren deaktiviert, und eine XDC\-Datei wird nicht erstellt.
