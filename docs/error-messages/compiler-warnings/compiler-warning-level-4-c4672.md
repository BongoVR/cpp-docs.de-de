# Compilerwarnung (Stufe 4) C4672
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1': Mehrdeutig. Zuerst als 'identifier2' aufgetreten  
  
 Das in einem **try**\-Block auszulösende angegebene Objekt ist mehrdeutig. Das Objekt kann nicht eindeutig gemacht werden, wenn es ausgelöst wird.
