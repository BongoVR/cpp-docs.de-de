# Compilerwarnung (Stufe 4) C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Es wird angenommen, dass die Typbibliothek für Anzahl\-Bit\-Pointer erstellt wurde  
  
 Da mit dem **ptrsize**\-Attribut der [\#import](../../preprocessor/hash-import-directive-cpp.md)\-Direktive keine expliziten Informationen angegeben wurden, ist der Compiler davon ausgegangen, dass die Zeigergröße der Typbibliothek ***Anzahl*** Zeiger umfasst.  
  
 Diese Warnung ist standardmäßig deaktiviert.  Weitere Informationen finden Sie unter [Standardmäßig deaktivierte Compilerwarnungen](../../preprocessor/compiler-warnings-that-are-off-by-default.md).
