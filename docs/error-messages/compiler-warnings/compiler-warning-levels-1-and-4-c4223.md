# Compilerwarnung (Stufen 1 und 4) C4223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nicht dem Standard entsprechende Erweiterung: Array, das keine L\-Werte enthält, wird in Zeiger konvertiert  
  
 In Standard C können Sie ein Array, das keinen L\-Wert enthält, nicht in einen Zeiger umwandeln.  Bei Verwendung der Microsoft\-Standarderweiterungen \([\/Ze](../../build/reference/za-ze-disable-language-extensions.md)\) ist dies jedoch möglich.
