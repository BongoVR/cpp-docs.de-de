# BSCMAKE-Fehler BK1504
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Positionieren in Datei "Dateiname" nicht möglich \[: Grund\]  
  
 BSCMAKE konnte sich nicht zu einer Position in der Datei bewegen.  
  
### Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Ein voller Datenträger.  
  
2.  Ein Hardwarefehler.  
  
3.  Dateiverkürzung aufgrund nicht genügenden Speicherplatzes oder einer Unterbrechung des Compilers während der Erstellung der SBR\-Datei.
