# <a name="bscmake-error-bk1505"></a>BSCMAKE-Fehler BK1505
kann nicht gelesen werden aus der Datei "Dateiname" [: Grund]  
  
 BSCMAKE kann nicht aus der Datei gelesen.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Die Datei beschädigt.  
  
2.  Die Datei abschneiden für den Compiler nur noch wenig Speicherplatz oder beim Erstellen der SBR-Datei unterbrochen wird.
