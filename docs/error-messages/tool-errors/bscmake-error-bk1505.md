# BSCMAKE-Fehler BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Datei 'Dateiname' kann nicht geöffnet werden \[: Grund\]  
  
 BSCMAKE kann nicht aus der Datei lesen.  
  
### Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Beschädigte Datei  
  
2.  Dateiverkürzung aufgrund nicht genügenden Speicherplatzes beim Kompilieren oder einer Unterbrechung des Compilers während der Erstellung der SBR\-Datei.
