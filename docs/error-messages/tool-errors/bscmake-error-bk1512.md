# BSCMAKE-Fehler BK1512
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**"**   
 ***Dateiname* ": Kapazität überschritten**  
  
 BSCMAKE konnte keine Browserinformationsdatei erstellen, da die Anzahl der Definitionen, Verweise, Module oder anderen Informationen das Limit überschreitet.  
  
### Beachten Sie die folgenden Vorschläge zur Problembehebung:  
  
1.  Schließen Sie mit **\/Em**, **\/Es** oder **\/Ei** einige Informationen aus.  
  
2.  Unterdrücken Sie die **\/Iu**\-Option.
