# <a name="bscmake-error-bk1512"></a>BSCMAKE-Fehler BK1512
FileName: Kapazität überschritten  
  
 BSCMAKE kann keine Browseinformationsdatei erstellen, da die Anzahl der Definitionen, Verweise, Module und andere Informationen über die Grenzen überschreitet.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>So beheben Sie den Fehler (unterschiedliche Lösungsmöglichkeiten)  
  
1.  Schließen Sie einige Informationen, die mit/em, / es oder Ei.  
  
2.  Lassen Sie die/IU-Option.
