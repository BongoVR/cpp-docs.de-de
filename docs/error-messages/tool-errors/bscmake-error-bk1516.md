# BSCMAKE-Fehler BK1516
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

BSC\-Datei beschädigt; inkrementelles Update nicht möglich  
  
 Die angegebene Browserinformationsdatei \(BSC\-Datei\) wurde beschädigt, möglicherweise durch einen Systemfehler beim Build.  Löschen Sie die BSC\-Datei, erstellen Sie erst alle SBR\-Dateien und dann die BSC\-Datei neu.
