# <a name="bscmake-warning-bk4502"></a>BSCMAKE-Warnung BK4502
abgeschnitten. SBR-Datei "Dateiname" nicht in Dateiname  
  
 Eine leere SBR-Datei, die ursprünglich nicht Teil einer BSC-Datei war, wurde während einer Aktualisierung angegeben.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Falscher Dateiname angegeben.  
  
2.  Die Datei gelöscht. (Fehler [BK1513](../../error-messages/tool-errors/bscmake-error-bk1513.md) Ergebnisse.)  
  
3.  Datei ist beschädigt, BSCMAKE, führen Sie einen vollständigen Build erfordern.
