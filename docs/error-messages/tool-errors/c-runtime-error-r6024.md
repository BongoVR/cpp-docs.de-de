# C-Laufzeitfehler R6024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nicht genügend Speicher für "\_onexit"\/"atexit"\-Tabelle  
  
 Es stand nicht genügend Speicher für die `_onexit`\-Funktion oder die `atexit`\-Funktion zur Verfügung.  Dieser Fehler wird dadurch verursacht, dass zu wenig Speicher vorhanden ist.
