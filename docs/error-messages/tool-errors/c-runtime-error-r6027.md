# C-Laufzeitfehler R6027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nicht genügend Speicher für die lowio\-Initialisierung.  
  
 Dieser Fehler tritt auf, wenn nicht genügend freier Arbeitsspeicher für die Initialisierung der E\/A\-Unterstützung auf niedriger Ebene zur Verfügung steht.
