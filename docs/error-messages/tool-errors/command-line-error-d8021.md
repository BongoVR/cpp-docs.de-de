# Befehlszeilenfehler D8021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültiges numerisches Argument 'Zahl'  
  
 Eine Zahl größer als 65.534 wurde als numerisches Argument angegeben.
