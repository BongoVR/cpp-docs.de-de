# <a name="command-line-error-d8022"></a>Befehlszeilenfehler D8022
'Nachrichtendatei' kann nicht geöffnet werden.  
  
 Die angegebene Datei wurde nicht im aktuellen Verzeichnis oder in einem Verzeichnis, in der PATH-Umgebungsvariablen angegeben. Die eingabeinstanznachrichten-Datei enthält eine kurze Zusammenfassung der Syntax für die Compilerbefehlszeile und Optionen.  
  
 Verschieben Sie diese Datei, um das aktuelle Verzeichnis oder ein Verzeichnis im aktuellen Pfad. Wenn diese Datei nicht gefunden werden kann, führen Sie das SETUP-Programm, um von den Datenträgern Verteilungspunkt zu kopieren.
