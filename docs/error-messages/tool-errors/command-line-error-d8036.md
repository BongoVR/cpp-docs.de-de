# Befehlszeilenfehler D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**"\/** *Option* **" wird für mehrere Quelldateien nicht unterstützt**  
  
 Die folgenden Compileroptionen können nicht mit mehreren Quelldateien verwendet werden:  
  
-   Assemblydateilisting benennen \(**\/Fa**\)  
  
-   Objektdatei umbenennen \(**\/Fo**\)  
  
-   Quellbrowserinformationen ohne lokale Variablen erstellen \(**\/Fr**\)  
  
-   Quellbrowserinformationen, die lokale Variablen enthalten, erstellen \(**\/FR**\)
