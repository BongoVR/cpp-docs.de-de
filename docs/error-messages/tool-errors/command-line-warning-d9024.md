# Befehlszeilenwarnung D9024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Typ der Quelldatei 'Dateiname' nicht erkannt, Objektdatei wird angenommen  
  
 Die Erweiterung der angegebenen Datei wurde nicht erkannt.  Es wurde angenommen, dass es sich bei dieser Datei um eine Objektdatei handelt, und sie wurde an den Linker Ã¼bergeben.  
  
 Folgende Erweiterungen werden erkannt:  
  
-   C \(C\-Quelldatei\)  
  
-   CXX \(C\+\+\-Quelldatei\)  
  
-   CPP \(C\+\+\-Quelldatei\)  
  
-   OBJ \(Objektdatei\)  
  
-   LIB \(Bibliotheksdatei\)  
  
-   DEF \(Moduldefinitionsdatei\)  
  
-   EXP \(Linkerexportdatei, von LINK \/LIB erstellt\)
