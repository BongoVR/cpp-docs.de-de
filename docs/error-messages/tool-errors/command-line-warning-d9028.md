# Befehlszeilenwarnung D9028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehler beim minimalen Neukompilieren, alles wird neu kompiliert  
  
 Die IDB\-Datei des Projekts ist fehlerhaft.  Löschen Sie die Datei, und erstellen Sie sie erneut.  
  
 Weitere Information über minimale Neukompilierungen und die IDB\-Datei finden Sie unter [\/Gm\-Compileroption](../../build/reference/gm-enable-minimal-rebuild.md).
