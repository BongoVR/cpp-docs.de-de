# CVTRES: Warnung CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Der Computertyp wurde nicht angegeben.**   
 ***Typ***  wird angenommen  
  
 CVTRES konnte keine Computerangaben finden.  Der angegebene Computertyp wurde angenommen.  Ist dieser falsch, starten Sie CVTRES mit der **\/MACHINE**\-Option neu.
