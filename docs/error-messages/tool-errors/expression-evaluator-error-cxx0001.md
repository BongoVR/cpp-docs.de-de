# Ausdrucksauswertungsfehler CXX0001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehler beim Ausführen einer benutzerspezifischen Funktion  
  
 Der ausgeführte Code verursachte eine allgemeine Schutzverletzung.  Dieser Fehler ist mit CAN0001 identisch.
