# Ausdrucksauswertungsfehler CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehler beim Zugriff auf den benutzerspezifischen Speicher  
  
 Vom Ausdruck wird versucht, auf Speicher zu verweisen, der nicht vom Programm belegt wurde, das gedebuggt wird.  
  
 Dieser Fehler ist mit CAN0002 identisch.
