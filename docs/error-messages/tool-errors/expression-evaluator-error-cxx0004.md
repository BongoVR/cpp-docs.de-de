# Ausdrucksauswertungsfehler CXX0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Syntaxfehler  
  
 Die Syntax des Ausdrucks ist falsch.  
  
 Geben Sie den Ausdruck in der richtigen Syntax neu ein.  
  
 Dieser Fehler ist mit CAN0004 identisch.
