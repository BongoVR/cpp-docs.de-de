# Ausdrucksauswertungsfehler CXX0008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**" am Ende der Zeichenfolge fehlt**  
  
 Das Anführungszeichen, das am Ende des Zeichenfolgenliterals erwartet wird, fehlt.  Geben Sie den Ausdruck neu ein, und schließen Sie dabei das Zeichenfolgenliteral in Anführungszeichen ein.  
  
 Dieser Fehler ist mit CAN0008 identisch.
