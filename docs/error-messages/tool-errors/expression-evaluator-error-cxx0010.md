# Ausdrucksauswertungsfehler CXX0010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Linke eckige Klammer fehlt  
  
 Eckige Klammern ohne Entsprechung sind im Ausdruck enthalten.  Geben Sie den Ausdruck mit ausgeglichenen eckigen Klammern neu ein.  
  
 Dieser Fehler ist mit CAN0010 identisch.
