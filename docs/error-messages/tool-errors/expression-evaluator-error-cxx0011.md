# Ausdrucksauswertungsfehler CXX0011
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Rechte eckige Klammer fehlt  
  
 Eckige Klammern ohne Entsprechung sind im Ausdruck enthalten.  Geben Sie den Ausdruck mit ausgeglichenen eckigen Klammern neu ein.  
  
 Dieser Fehler ist mit CAN001 identisch.
