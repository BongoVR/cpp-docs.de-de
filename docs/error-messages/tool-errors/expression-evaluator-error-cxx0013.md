# Ausdrucksauswertungsfehler CXX0013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Operator fehlt  
  
 Ein Operator wurde im Ausdruck erwartet, aber nicht gefunden.  Überprüfen Sie die Syntax des Ausdrucks.  
  
 Dieser Fehler ist mit CAN0013 identisch.
