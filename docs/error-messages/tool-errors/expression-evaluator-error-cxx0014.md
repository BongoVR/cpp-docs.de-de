# Ausdrucksauswertungsfehler CXX0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Operand fehlt  
  
 Ein Operator wurde ohne erforderlichen Operand angegeben.  Überprüfen Sie die Syntax des Ausdrucks.  
  
 Dieser Fehler ist mit CAN0014 identisch.
