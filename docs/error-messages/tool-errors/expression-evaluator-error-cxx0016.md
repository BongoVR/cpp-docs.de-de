# Ausdrucksauswertungsfehler CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Konstante zu groß  
  
 Die C\-Ausdrucksauswertung kann keine vorzeichenlose Ganzzahlkonstante, die größer als 4.294.967.295 \(0FFFFFFFF hexadezimal\) ist, und keine Gleitkommakonstante, die größer als ca. 1.8E\+308 ist, übernehmen.  
  
 Dieser Fehler ist mit CAN0016 identisch.
