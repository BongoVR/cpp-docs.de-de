# Ausdrucksauswertungsfehler CXX0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehlerhafter Registrierungsname  
  
 Das angegebene Register ist nicht vorhanden oder kann nicht angezeigt werden.  
  
 Das Überwachungsfenster kann folgende Register anzeigen:  
  
 AX    SP    DS    IP  
  
 BX    BP    ES    FL  
  
 CX    SI    SS    GS  
  
 DX    DI    CS    SS  
  
 EAX    ESP    DS    EIP  
  
 EBX    EBP    ES    EFL  
  
 ECX    ESI    FS  
  
 EDX    EDI    CS  
  
 Dieser Fehler ist mit CAN0018 identisch.
