# Ausdrucksauswertungsfehler CXX0020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Operandentypen für diese Operation ungeeignet  
  
 Ein Operator wurde auf einen Ausdruck mit einem ungültigen Typ für diesen Operator angewendet.  
  
 Es ist z. B. nicht zulässig, die Adresse eines Registers zu ermitteln oder ein Array mit einem Gleitkomma\-Ausdruck zu indizieren.  
  
 Dieser Fehler ist mit CAN0020 identisch.
