# Ausdrucksauswertungsfehler CXX0025
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Operator benötigt struct\/union  
  
 Ein Operator, durch den ein Ausdruck vom Typ `struct` oder **union** unterstützt wird, wurde auf einen Ausdruck angewendet, der nicht den Typ `struct` oder **union** aufweist.  
  
 Elemente von Klassen\-, Struktur\- oder Unionvariablen müssen über einen voll gekennzeichneten Namen verfügen.  Elemente können nicht ohne vollständige Spezifikation eingegeben werden.  
  
 Dieser Fehler ist mit CAN0025 identisch.
