# Ausdrucksauswertungsfehler CXX0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehlerhafte Formatierungszeichenfolge  
  
 Ein Zeichenfolgenformat wurde falsch angegeben.  Überprüfen Sie die Syntax des Ausdrucks.  
  
 Dieser Fehler ist mit CAN0026 identisch.  
  
 Weitere Informationen finden Sie unter [Formatbezeichner in C\+\+](../Topic/Format%20Specifiers%20in%20C++.md).
