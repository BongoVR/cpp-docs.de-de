# Ausdrucksauswertungsfehler CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Kein Struct\-Zeiger  
  
 Der Memberauswahloperator \(**\-\>**\) wurde auf einen Ausdruck angewendet, der kein Zeiger auf eine Struktur darstellt.  
  
 Stellen Sie sicher, dass der gesamte Ausdruck richtig in Klammern eingeschlossen wurde, oder wandeln Sie den Adressausdruck in den entsprechenden Zeigertyp um.  
  
 Dieser Fehler ist mit CAN0029 identisch.
