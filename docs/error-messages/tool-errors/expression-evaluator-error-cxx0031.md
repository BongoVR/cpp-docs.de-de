# Ausdrucksauswertungsfehler CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ausdruck nicht erweiterbar  
  
 Interner Fehler bei der Ausdrucksauswertung.  
  
 Wählen Sie einen äquivalenten Ausdruck, der richtig ausgewertet wird.  
  
 Dieser Fehler ist mit CAN0031 identisch.
