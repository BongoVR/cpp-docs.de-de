# Ausdrucksauswertungsfehler CXX0038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehler in Anzahl und\/oder Typenkonflikt der Funktionsargumente  
  
 Der angegebene Funktionsaufruf entspricht nicht dem Prototyp der Funktion.  
  
 Der Aufruf muss mit der richtigen Anzahl an Argumenten neu eingegeben werden.  Falls nötig muss für jedes Argument eine Typumwandlung vorgenommen werden, damit es dem Prototyp entspricht.  
  
 Dieser Fehler ist mit CAN0038 identisch.
