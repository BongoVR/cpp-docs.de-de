# Ausdrucksauswertungsfehler CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Argumentliste passt nicht zur Funktion  
  
 Eine Funktion mit einer aktuellen Parameterliste, die mit keiner formalen Parameterliste einer im Programm definierten Funktion mit dem gleichen Namen übereinstimmt, wurde von einem Ausdruck aufgerufen.  
  
 Überladene Funktionen können nur aufgerufen werden, wenn eine exakte Übereinstimmung der Parameter oder eine Übereinstimmung, für die das Erstellen eines Objekts nicht erforderlich ist, vorliegt.  
  
 Dieser Fehler ist mit CAN0047 identisch.
