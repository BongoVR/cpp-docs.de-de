# Ausdrucksauswertungsfehler CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Aufrufabfolge nicht unterstützt  
  
 Von der im Ausdruck angegebenen Funktion wurde eine Aufrufabfolge, die von der C\-Ausdrucksauswertung nicht unterstützt wird, verwendet.  Diese Funktion kann nicht in einem Überwachungsfensterausdruck aufgerufen werden.  
  
 Dieser Fehler ist mit CAN0048 identisch.
