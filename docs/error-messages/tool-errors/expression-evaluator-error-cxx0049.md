# Ausdrucksauswertungsfehler CXX0049
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Veraltetes OMF – Programm neu verknüpfen  
  
 Ein altes OMF \(Object Module Format\) wurde vom Programm verwendet.  
  
 Das Programm muss mit der aktuellen Linkerversion neu verknüpft werden.  
  
 Dieser Fehler ist mit CAN0049 identisch.
