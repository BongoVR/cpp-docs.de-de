# Ausdrucksauswertungsfehler CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Mehr als ein überladenes Symbol in Haltepunkt angegeben  
  
 Von der Ausdrucksauswertung konnte nicht ermittelt werden, welches der überladenen Symbole als Haltepunkt verwendet werden soll, da mehr als ein überladenes Symbol vorhanden ist.  
  
 Dieser Fehler ist mit CAN0051 identisch.
