# <a name="expression-evaluator-error-cxx0051"></a>Ausdrucksauswertungsfehler CXX0051
mehrere überladene Symbol im Haltepunkt angegeben  
  
 Die ausdrucksauswertung konnte nicht bestimmt die mehrere überladene-Symbol, um Sie als Breakpoint verwenden.  
  
 Dieser Fehler ist mit CAN0051 identisch.
