# Ausdrucksauswertungsfehler CXX0052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Memberfunktion nicht vorhanden  
  
 Eine Memberfunktion wurde als Haltepunkt angegeben, konnte aber nicht gefunden werden.  Dieser Fehler kann durch das Festlegen eines Haltepunktes in einer Funktion, die als Inlinefunktion deklariert wurde, verursacht werden.  
  
 Um einen Haltepunkt in dieser Funktion festzulegen, muss die Datei mit dem \/Ob0\-Schalter \(Inlinedeklaration verbieten\) neu kompiliert werden.  
  
 Eine nicht definierte Funktion wurde von einem Ausdruck aufgerufen.  
  
 Dieser Fehler ist mit CAN0052 identisch.
