# Ausdrucksauswertungsfehler CXX0054
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Registerangaben sind in Haltepunktausdrücken unzulässig  
  
 Registerangaben können in einem Haltepunktausdruck nicht verwendet werden.  
  
 Dieser Fehler ist mit CAN0054 identisch.
