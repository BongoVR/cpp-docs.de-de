# Ausdrucksauswertungsfehler CXX0055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Mehrdeutiges Symbol in Kontextoperator  
  
 Ein Symbol im Operator Kontext \(**{}**\) verweist auf mehrere Symbole im Programm.  
  
 Die Mehrdeutigkeit kann vom Bereichsauflösungsoperator \(`::`\) nicht aufgelöst werden.  
  
 Dieser Fehler ist mit CAN0055 identisch.
