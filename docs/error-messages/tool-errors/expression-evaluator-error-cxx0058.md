# Ausdrucksauswertungsfehler CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Überladenen Operator nicht gefunden  
  
 Ein Klassentyp wurde als linker Operand in einem Ausdruck angegeben, es wurde aber kein überladener Operator für diese Klasse definiert.  
  
 Dieser Fehler ist mit CAN0058 identisch.
