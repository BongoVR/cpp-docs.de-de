# Ausdrucksauswertungsfehler CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Maximale Schachtelungstiefe für Funktionen überschritten  
  
 Der Ausdruck enthält eine Funktion mit einer Schachtelungstiefe, die größer als das zulässige Limit ist.  Um die Schachtelungstiefe zu reduzieren, muss der Ausdruck geändert werden.  
  
 Dieser Fehler ist mit CAN0061 identisch.
