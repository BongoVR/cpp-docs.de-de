# Ausdrucksauswertungsfehler CXX0062
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Konstruktoraufrufe nicht unterstützt  
  
 Ein Konstruktor wurde von einem Ausdruck aufgerufen.  Es ist nicht möglich, mit Ausdrücken explizit Konstruktoren aufzurufen oder eine Konvertierung, für die der Aufruf eines Konstruktors erforderlich ist, vorzunehmen.  
  
 Dieser Fehler ist mit CAN0062 identisch.
