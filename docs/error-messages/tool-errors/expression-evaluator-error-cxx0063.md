# Ausdrucksauswertungsfehler CXX0063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Überladener Operator \- nicht \> unterstützt  
  
 Der Ausdruck verwendet Ein überladener Klassenmemberzugriffs\-Operator \(**\-\>**\).  
  
 Dieser Fehler ist mit CAN0063 identisch.
