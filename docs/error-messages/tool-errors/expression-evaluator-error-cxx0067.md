# Ausdrucksauswertungsfehler CXX0067
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Funktionsauswertung nicht unterstützt  
  
 Der Ausdruck enthält einen Funktionsaufruf.  Funktionsaufrufe werden von einigen Ausdrucksauswertungen nicht unterstützt.  
  
 Dieser Fehler ist mit CAN0067 identisch.
