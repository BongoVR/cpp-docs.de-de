# Ausdrucksauswertungsfehler CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehler: Typinformation fehlend oder unbekannt  
  
 Die PCH\-Datei wurde nicht verknüpft, oder der Code verfügt über einen Verweis auf einen Typ, der sich in einem Objektmodul, das nicht mit dem **\/Zi**\-Schalter kompiliert wurde, befindet.  
  
 Dieser Fehler ist mit CAN0067 identisch.
