# Linkertoolfehler LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültiger Computertyp Typ  
  
 Der im Objektheader angegebene Computertyp ist ungültig.  
  
 Dieser Fehler kann verursacht werden, wenn die Datei fehlerhaft ist.  Erstellen Sie das Objekt neu.
