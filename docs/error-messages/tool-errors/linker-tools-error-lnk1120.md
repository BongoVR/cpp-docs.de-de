# Linkertoolfehler LNK1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Anzahl unaufgelöste externe Verweise  
  
 In Fehler LNK1120 ist die `number` der nicht aufgelösten externen Verweise für diesen Link angegeben.  Die Bedingungen, unter denen nicht aufgelöste externe Verweise entstehen, sind unter Fehler [LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md) beschrieben, der dieser Fehlermeldung vorausgeht \(jeweils eine für jeden nicht aufgelösten externen Verweis\).  
  
 Weitere Informationen finden Sie unter [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;815650](http://support.microsoft.com/default.aspx?scid=kb;en-us;815650).
