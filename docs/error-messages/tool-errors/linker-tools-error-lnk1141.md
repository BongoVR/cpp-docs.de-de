# Linkertoolfehler LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehler beim Erstellen der Exportdatei  
  
 [LINK](../../build/reference/linker-command-line-syntax.md) konnte die Exportdatei \(**.exp**\) nicht erstellen.  
  
 Ursachen für diesen Fehler können Systemprobleme wie unzureichender Arbeitsspeicher und Syntaxfehler in Optionen oder Moduldefinitionsanweisungen sein.  
  
 Diesem Fehler geht ein anderer Fehler voraus, der zusätzliche Informationen liefert.
