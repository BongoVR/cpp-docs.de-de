# <a name="linker-tools-error-lnk1169"></a>Linkertoolfehler LNK1169
eine oder mehrere mehrfach definierte Symbole gefunden  
  
 Der Buildvorgang ist aufgrund mehrere Definitionen für eine oder mehrere Symbole fehlgeschlagen. Dieser Fehler wird Fehler vorangestellt [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).  
  
 Die [/ERZWINGEN](../../build/reference/force-force-file-output.md) oder **/Force: Multiple** Option überschreibt diesen Fehler.
