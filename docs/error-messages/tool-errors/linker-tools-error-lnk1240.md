# Linkertoolfehler LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

der IDL\-Inhalt konnte nicht kompiliert werden  
  
 Der Linker hat MIDL zum Kompilieren eingebetteter IDL erstellt, es ist jedoch ein Problem aufgetreten.  Überprüfen Sie die durch MIDL angegebenen Fehler.
