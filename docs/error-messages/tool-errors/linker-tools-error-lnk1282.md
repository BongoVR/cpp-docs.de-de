# Linkertoolfehler LNK1282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/REBASE für Datei konnte nicht durchgeführt werden; es wurde signiert  
  
 Sie haben versucht, die Basisadresse einer signierten Assembly mit der \/REBASE\-Option für [editbin](../../build/reference/editbin-reference.md) zu ändern.  Dazu müssen Sie zuerst die Basisadresse ändern und anschließend die Assembly signieren.
