# Linkertoolfehler LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehlerhafte oder ungültige COFF\-Symboltabelle \(nicht definiertes statisches oder Markierungssymbol\)  
  
 Der vom Compiler erstellte Inhalt für einen Abschnitt im betreffenden Objekt ist anscheinend beschädigt.  
  
 Dieser Fehler ist möglicherweise ein Fehler des Compilers.
