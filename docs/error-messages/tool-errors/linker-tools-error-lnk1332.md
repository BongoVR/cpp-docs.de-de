# Linkertoolfehler LNK1332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

hat \<Anzahl\> Windows Runtime verwendet werden importiert in ein Modul und in einem anderen Modul definiert  
  
 Als das aktuelle Ziel hat, gibt der Linker hat \<`count`\> Windows Runtime, von denen jeder in ein Modul importiert wird und auch in einem anderen Modul definiert.  
  
### So beheben Sie diesen Fehler  
  
-   Beheben Sie die Fehler LNK2039 im Build gemäß dem Vorschlag in der Fehlermeldung.
