# Linkertoolfehler LNK2026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unsicheres Modul für SAFESEH\-Bild.  
  
 [\/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) wurde angegeben, ein Modul war jedoch nicht mit dem Feature für sichere Ausnahmebehandlung kompatibel.  Wenn Sie dieses Modul mit **\/SAFESEH** verwenden möchten, müssen Sie das Modul unter Verwendung des Visual C\+\+ .NET 2003\-Compilers \(oder höher\) neu kompilieren.
