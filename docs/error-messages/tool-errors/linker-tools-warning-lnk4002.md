# Linkertoolwarnung LNK4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Symbol in Objekt definiert  
  
 Das in seiner ergänzten Form dargestellte Symbol wurde in `object` in seiner nicht ergänzten Form angegeben, es wurde jedoch keine eindeutige Übereinstimmung mit einem ergänzten Symbol gefunden.  Dieser Warnung geht immer die Warnung [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) voraus, und ihr folgt der schwerwiegende Fehler [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
