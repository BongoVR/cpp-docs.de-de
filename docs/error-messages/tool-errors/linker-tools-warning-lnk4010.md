# Linkertoolwarnung LNK4010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültige Versionsnummer 'Nummer'; Standardversion des Subsystems wird angenommen  
  
 Sie können eine Version für das Bild des Subsystems angeben \([\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)\).  Jedes Subsystem hat eine minimale Anforderung an die Version.  Wenn die angegebene Version niedriger als die erforderliche ist, wird diese Warnmeldung ausgegeben, und der Linker verwendet nur das Standardsubsystem.
