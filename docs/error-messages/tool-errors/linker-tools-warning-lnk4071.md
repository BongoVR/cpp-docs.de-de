# <a name="linker-tools-warning-lnk4071"></a>Linkertoolwarnung LNK4071
kann nicht auf die nachfolgenden Links inkrementell verknüpft werden  
  
 LINK hat mehrere Definitionen für eine oder mehrere Symbole gefunden, aber [/ERZWINGEN](../../build/reference/force-force-file-output.md) oder **/Force: Multiple** wurde zum Erstellen einer Ausgabedatei unabhängig von Fehlern verwendet. LINK gelöscht, die inkrementelle Statusdatei (ILK).
