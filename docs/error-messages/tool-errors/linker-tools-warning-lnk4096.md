# Linkertoolwarnung LNK4096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/BASE\-Wert 'Nummer' ist für Windows 95 und Windows 98 nicht gültig. Die Anwendung kann möglicherweise nicht ausgeführt werden.  
  
 Die von Ihnen angegebene Basisadresse ist ungültig.  Ausführbare Windows 95\- und Windows 98\-Dateien müssen über eine Basisadresse verfügen, die größer als 0x400000 ist.  Weitere Informationen über Basisadressen finden Sie unter der Linkeroption [\/BASE](../../build/reference/base-base-address.md).
