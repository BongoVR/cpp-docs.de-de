# <a name="linker-tools-warning-lnk4096"></a>Linkertoolwarnung LNK4096
/ Basiswert "Number" ist ungültig für Windows 95 und Windows 98. Abbild kann möglicherweise nicht ausgeführt.  
  
 Die Basisadresse, die Sie angegeben haben, ist ungültig. Windows 95 und Windows 98 ausführbare Dateien benötigen eine Basisadresse, die größer als 0 x 400000. Weitere Informationen über Basisadressen finden Sie unter der [/BASE](../../build/reference/base-base-address.md) (Linkeroption).
