# Linkertoolwarnung LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/DELAYLOAD:DLL\-Name wird ignoriert; keine Importe von DLL\-Name gefunden  
  
 Der `dllname` wird vom Linker ignoriert, da keine der von `dllname` exportierten Funktionen erforderlich ist.
