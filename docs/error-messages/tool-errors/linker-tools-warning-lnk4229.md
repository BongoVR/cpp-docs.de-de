# Linkertoolwarnung LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültige Direktive \/directive gefunden; ignoriert  
  
 Eine mit dem [comment](../../preprocessor/comment-c-cpp.md)\-Pragma übergebene Direktive war ungültig.  `/directive` wird vom Linker ignoriert.
