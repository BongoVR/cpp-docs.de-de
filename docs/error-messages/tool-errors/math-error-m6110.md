# Mathematischer Fehler M6110
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Stapelüberlauf  
  
 Ein Gleitkommaausdruck hat einen Gleitkomma\-Stapelüberlauf verursacht.  
  
 Stapelüberlauf\-Gleitkommaausnahmen werden bis zu einer Tiefe von 7 Ebenen aufgefangen, zusätzlich zu den 8 Ebenen, die in der Regel vom 8087\/287\/387\-Coprozessor unterstützt werden.  
  
 Das Programm wird mit Exitcode 138 beendet.
