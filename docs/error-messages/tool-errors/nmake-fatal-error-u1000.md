# NMAKE: Schwerwiegender Fehler U1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Syntaxfehler: "\)" fehlt im Makroaufruf  
  
 Eine linke Klammer **\(** ist ohne eine entsprechende rechte Klammer **\)** in einem Makroaufruf aufgetreten.  Die korrekte Form lautet $\(*Name*\). $*n* ist für Namen mit einem Zeichen zulässig.
