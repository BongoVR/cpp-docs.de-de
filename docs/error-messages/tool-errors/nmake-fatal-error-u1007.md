# NMAKE: Schwerwiegender Fehler U1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Anführungszeichen im Namen nicht zulässig  
  
 Im angegebenen Zielnamen oder Dateinamen ist ein Anführungszeichen \(**"**\) enthalten.  
  
 Anführungszeichen können einen Dateinamen umgeben, dürfen jedoch nicht in diesem enthalten sein.
