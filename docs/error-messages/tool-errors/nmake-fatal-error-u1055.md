# NMAKE: Schwerwiegender Fehler U1055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nicht genügend Umgebungsspeicher  
  
 Dem Betriebssystem steht kein Speicherplatz mehr für Umgebungsvariablen zur Verfügung.  
  
 Vergrößern Sie entweder den Umgebungsspeicher, oder legen Sie weniger Umgebungsvariablen fest.
