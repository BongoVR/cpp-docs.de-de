# NMAKE: Schwerwiegender Fehler U1065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültige Option "Option"  
  
 Bei der angegebenen Option handelt es sich nicht um eine gültige Option für NMAKE.
