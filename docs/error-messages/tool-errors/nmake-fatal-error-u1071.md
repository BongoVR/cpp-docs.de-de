# NMAKE: Schwerwiegender Fehler U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Schleife in Abhängigkeitsstruktur für Ziel "Zielname"  
  
 Für das angegebene Ziel besteht eine Ringabhängigkeit im Abhängigkeitsbaum.  Das angegebene Ziel ist von einer abhängigen Datei des angegebenen Zieles abhängig.  Ringabhängigkeiten sind nicht zulässig.
