# NMAKE: Schwerwiegender Fehler U1077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"Programm": Rückgabe\-Code "Wert"  
  
 Der angegebene Befehl oder das Programm, das von NMAKE aufgerufen wurde, konnte nicht ausgeführt werden und gab den angegebenen Exitcode zurück.  
  
 Mit der **\/I**\-Option, der **.IGNORE**\-Punkt\-Direktive oder dem Bindestrich\-Befehlsparameter \(**\-**\) kann dieser Fehler unterdrückt und die NMAKE\-Sitzung fortgesetzt werden.  Mit der **\/K**\-Option kann die NMAKE\-Sitzung für unabhängige Variablen des Abhängigkeitsbaumes fortgesetzt werden.
