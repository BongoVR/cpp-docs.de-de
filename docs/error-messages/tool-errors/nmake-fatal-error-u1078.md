# NMAKE: Schwerwiegender Fehler U1078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Konstantenüberlauf bei "Ausdruck"  
  
 Der angegebene Ausdruck enthielt eine Konstante, die den Bereich \-2.147.483.648 bis 2.147.483.647 überschritten hat.  Die Konstante trat in einer der folgenden Situationen auf:  
  
-   Einem Ausdruck, der mit einer Präprozessordirektive angegeben wurde.  
  
-   Einer Fehlerstufe, die mit dem Bindestrich\-Befehlsmodifizierer \(**\-**\) angegeben wurde.
