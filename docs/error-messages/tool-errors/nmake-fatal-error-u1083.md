# NMAKE: Schwerwiegender Fehler U1083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Zielmakro "Ziel" erweitert zu nichts  
  
 Bei dem angegebenen Ziel handelt es sich um einen Aufruf eines Makros, das nicht definiert wurde oder einen NULL\-Wert aufweist.  Es kann kein NULL\-Ziel verarbeitet werden.
