# NMAKE: Schwerwiegender Fehler U1086
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Rückschlussregel kann keine Abhängigkeiten haben**  
  
 Auf den Doppelpunkt \(:\) in einer Rückschlussregel muss eines der folgenden Zeichen folgen:  
  
-   Zeilenendemarke  
  
-   Semikolon \(**;**\), dem ein Befehl folgen kann.  
  
-   Nummernzeichen \(**\#**\), dem ein Kommentar folgen kann.
