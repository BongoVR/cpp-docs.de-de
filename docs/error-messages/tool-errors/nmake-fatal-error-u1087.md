# NMAKE: Schwerwiegender Fehler U1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Abhängige Elemente vom Typ : und :: für dasselbe Ziel nicht möglich  
  
 Ein Ziel kann nicht in einer Abhängigkeit mit einem einfachen Doppelpunkt \(**:**\) und zwei Doppelpunkten \(`::`\) angegeben werden.  
  
 Um ein Ziel in mehreren Beschreibungsblöcken anzugeben, wird in jeder Abhängigkeitszeile :: verwendet.
