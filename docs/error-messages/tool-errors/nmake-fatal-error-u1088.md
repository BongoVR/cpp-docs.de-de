# NMAKE: Schwerwiegender Fehler U1088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültiges Trennzeichen "::" in Rückschlussregel  
  
 Auf eine Rückschlussregel muss ein einfacher Doppelpunkt folgen \(**:**\).
