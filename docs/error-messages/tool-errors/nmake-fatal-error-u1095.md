# <a name="nmake-fatal-error-u1095"></a>NMAKE: Schwerwiegender Fehler U1095
Erweiterte Befehlszeile "Befehlszeile" zu lang  
  
 Nach der makroerweiterung hat die angegebene Befehlszeile die zulässige Länge von Befehlszeilen für das Betriebssystem überschritten.  
  
 MS-DOS-ermöglicht bis zu 128 Zeichen in einer Befehlszeile.  
  
 Wenn der Befehl für ein Programm, der aus einer Datei Befehlszeileneingabe akzeptieren kann ist, ändern Sie den Befehl und Bereitstellen Sie Eingaben aus einer Datei auf Datenträger oder eine Inlinedatei. LINK und LIB akzeptiert z. B. die Eingabe aus einer Antwortdatei.
