# NMAKE: Schwerwiegender Fehler U1097
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Syntax für Dateinamenteile benötigt abhängiges Element  
  
 Die aktuelle Abhängigkeit besitzt weder ein explizites abhängiges noch ein implizites abhängiges Element.  Die Syntax für Dateinamenteile, die eine Prozent \(`%`\)\-Angabe verwendet, repräsentiert die Komponenten des ersten abhängigen Elements des aktuellen Zieles.
