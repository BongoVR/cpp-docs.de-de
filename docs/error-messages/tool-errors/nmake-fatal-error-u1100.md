# NMAKE: Schwerwiegender Fehler U1100
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Das Makro "Makroname" ist im Kontext der Batchregel "Regel" unzulässig  
  
 Dieser Fehler wird generiert wenn der Befehlsblock einer Batchmodusregel direkt oder indirekt ein Gerätedateimakro, verweist das $\<nicht ist.  
  
 $\< stellt das einzige Makro für die Regeln.
