# NMAKE: Warnung U4006
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Spezialmakro undefiniert: "Makroname"  
  
 Der angegebene Spezialmakroname ist undefiniert und erweitert zu nichts.
