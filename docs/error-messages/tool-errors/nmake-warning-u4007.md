# NMAKE: Warnung U4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Dateiname "Dateiname" zu lang und wird auf 8.3 verkürzt  
  
 Der Basisname der angegebenen Datei besitzt mehr als acht Zeichen, oder die Erweiterung weist mehr als drei Zeichen auf.  Der Basisname wurde von NMAKE bis auf acht und die Erweiterung bis auf drei Zeichen abgeschnitten.  
  
 Falls lange Dateinamen vom Dateisystem unterstützt werden, muss der Name in Anführungszeichen \(**"**\) eingeschlossen werden.
