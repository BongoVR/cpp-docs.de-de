# <a name="nmake-warning-u4007"></a>NMAKE: Warnung U4007
der Dateiname 'Dateiname' zu lang. 8.3 Abschneiden  
  
 Der Basisname der angegebenen Datei verfügt über mehr als acht Zeichen lang sein, oder die Erweiterung hat mehr als drei Zeichen. NMAKE abgeschnitten, den Namen einer acht Zeichen-Basis und einer dreistelligen-Erweiterung.  
  
 Wenn vom Dateisystem lange Dateinamen unterstützt werden, schließen Sie den Namen in doppelte Anführungszeichen (**"**).
