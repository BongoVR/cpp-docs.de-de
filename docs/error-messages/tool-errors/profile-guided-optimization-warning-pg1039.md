# Warnung bei profilgesteuerter Optimierung: PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Warten auf Leerlaufzeit für Sweep ist fehlgeschlagen.  
  
 Die Anwendung hinderte [pgosweep](../../build/reference/pgosweep.md) an der Ausführung.  Führen Sie nach Möglichkeit pgosweep aus, wenn die Anwendung im Leerlauf ist.
