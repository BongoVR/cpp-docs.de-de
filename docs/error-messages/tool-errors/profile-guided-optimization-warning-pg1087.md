# Warnung bei profilgesteuerter Optimierung: PG1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die PGC\-Datei 'Datei' ist nicht gültig.  
  
 Eine PGC\-Datei ist möglicherweise aufgrund einer vorzeitigen Beendigung einer Profilerstellungssitzung fehlerhaft.  Führen Sie das Profilerstellungsszenario erneut aus, und beenden Sie dieses ordnungsgemäß.
