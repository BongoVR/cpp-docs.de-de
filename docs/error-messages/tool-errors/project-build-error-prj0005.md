# Projektbuildfehler PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Es konnte keine temporäre Datei im Verzeichnis "Verzeichnis" erstellt werden.  
  
 Fehler beim Aufruf zum Erstellen einer temporären Datei.  Folgende Gründe können vorliegen:  
  
-   Es sind keine weiteren Namen für temporäre Dateien verfügbar.  
  
-   Das temporäre Verzeichnis ist schreibgeschützt.  
  
-   Es ist kein temporäres Verzeichnis bzw. keine TMP\-Umgebungsvariable vorhanden.  
  
-   Auf dem Computer ist nicht genügend Festplattenspeicher verfügbar.
