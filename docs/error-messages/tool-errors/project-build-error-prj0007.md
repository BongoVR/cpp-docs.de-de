# Projektbuildfehler PRJ0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Das Ausgabeverzeichnis "Verzeichnis" konnte nicht erstellt werden.  
  
 Visual C\+\+ konnte kein Ausgabeverzeichnis erstellen.  Folgende Gründe können vorliegen:  
  
-   Das Verzeichnis, in dem das Ausgabeverzeichnis erstellt werden sollte, ist schreibgeschützt.  
  
-   Auf dem Computer ist nicht genügend Festplattenspeicher verfügbar.  
  
-   Es wurde ein ungültiger Verzeichnispfad angegeben.  
  
-   Die Benutzerberechtigungen reichen nicht für das Erstellen des Verzeichnisses aus.
