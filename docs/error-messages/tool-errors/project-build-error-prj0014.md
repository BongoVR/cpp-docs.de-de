# Projektbuildfehler PRJ0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Fehler beim zur Steuerung der gestarteten Prozesse verwendeten Auftragsobjekt.  Der Buildvorgang kann nicht fortgesetzt werden.  
  
 In der Entwicklungsumgebung ist ein Fehler aufgetreten.  
  
 Um diesen Fehler zu beheben, schließen Sie das Projekt, und laden Sie es erneut.  Falls erforderlich, beenden Sie Visual Studio, und starten Sie es erneut.
