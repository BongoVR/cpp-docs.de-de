# Projektbuildfehler PRJ0015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Das NULL\-Gerät ist nicht auf dem System vorhanden.Es ist nicht möglich, einen Build zu starten.  
  
 Dieser Fehler kann durch unzureichende Systemressourcen hervorgerufen werden.  Schließen Sie einige Anwendungen, oder führen Sie einen Neustart durch.  
  
 Der Fehler kann auch darin begründet sein, dass Sie auf dem Computer nicht über die erforderlichen Berechtigungen für das NULL\-Gerät verfügen.
