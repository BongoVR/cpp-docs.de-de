# <a name="project-build-error-prj0015"></a>Projektbuildfehler PRJ0015
Das NULL-Gerät ist im System nicht vorhanden. Wir können nicht zum Starten eines Builds.  
  
 Dieser Fehler kann durch unzureichende Systemressourcen verursacht werden. Schließen Sie einige Programme, oder starten Sie neu.  
  
 Er kann auch verursacht werden, wenn Sie nicht über ausreichende Berechtigungen zum NULL-Gerät auf dem Computer verfügen.
