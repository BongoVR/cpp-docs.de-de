# Projektbuildfehler PRJ0017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Das aktuelle Arbeitsverzeichnis ist ungültig.  
  
 Der Pfad zum aktuellen Arbeitsverzeichnis ist länger als durch **\_MAXPATH** vorgegeben.  Um diesen Fehler zu vermeiden, sollte zur Erstellung keine derart tiefe Projektebene verwendet werden.
