# Projektbuildfehler PRJ0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die Antwortdatei "Datei" enthält Unicodeinhalt, der nicht in die ANSI\-Codepage des Benutzers übersetzt werden konnte.  
  
 ***UNICODE\-Inhalt der Datei***  
  
 Das Projektsystem hat Unicodeinhalt in einer Antwortdatei gefunden, der nicht ordnungsgemäß in die aktuelle ANSI\-Codepage des Benutzers übersetzt werden kann.  
  
 Die Lösung dieses Problems besteht darin, dass Sie den Inhalt der Antwortdatei unter Verwendung von ANSI aktualisieren oder die Codepage auf dem Computer installieren und als Systemstandard festlegen.
