# Projektbuildfehler PRJ0027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unicodeprotokollmeldung 'Inhalt' enthält Inhalt, der nicht in die ANSI\-Codepage des Benutzers übersetzt werden konnte.  
  
 Diese Warnmeldung wird in der Regel nur zusammen mit Fehlern ausgegeben, die beim Erstellen von Batch\- und\/oder Antwortdateien generiert werden.  
  
 Die Lösung dieses Problems besteht darin, dass Sie den Inhalt des Buildprotokolls unter Verwendung von ANSI aktualisieren oder die Codepage auf dem Computer installieren und als Systemstandard festlegen.
