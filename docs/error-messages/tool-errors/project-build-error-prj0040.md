# Projektbuildfehler PRJ0040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Interner Fehler beim Erstellen.Vorgang kann nicht fortgesetzt werden.Laden Sie das Projekt erneut, und wiederholen Sie den Vorgang.  
  
 Das Projekt konnte nicht vom Projektsystem verarbeitet werden.  
  
 Versuchen Sie, das Projekt erneut zu laden.
