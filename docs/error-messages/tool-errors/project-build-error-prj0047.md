# Projektbuildfehler PRJ0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Der angehaltene Vorgang kann nicht fortgesetzt werden.  Fehler beim Buildvorgang.  
  
 In der Entwicklungsumgebung ist ein Fehler aufgetreten.  
  
 Um diesen Fehler zu beheben, schließen Sie das Projekt, und laden Sie es erneut.  Falls erforderlich, beenden Sie Visual Studio, und starten Sie es erneut.
