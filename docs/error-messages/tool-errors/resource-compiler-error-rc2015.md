# <a name="resource-compiler-error-rc2015"></a>Ressourcencompiler: Fehler RC2015
zu viele Zeichen in der Konstante  
  
 Eine Zeichenkonstante enthielt mehr als zwei Zeichen.  
  
 Zeichenkonstanten sind auf ein Zeichen (standard Zeichenkonstanten) oder zwei Schrägstriche (lange Zeichenkonstanten) beschränkt.  
  
 Beachten Sie, dass eine Escapesequenz (z. B. \t für Registerkarte ") in ein einzelnes Zeichen konvertiert wird.
