# Ressourcencompiler: Fehler RC2017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültige Escapesequenz  
  
 Eine Escape\-Sequenz tritt an einer nicht erwarteten Stelle auf.  
  
 Eine Escape\-Sequenz, ein umgekehrter Schrägstrich \(\\\), auf den eine Zahl oder ein Buchstabe folgt, darf nur innerhalb von Zeichen\- oder Zeichenfolgenkonstanten auftreten.
