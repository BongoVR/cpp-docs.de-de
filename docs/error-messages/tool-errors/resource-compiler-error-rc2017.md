# <a name="resource-compiler-error-rc2017"></a>Ressourcencompiler: Fehler RC2017
Ungültige Escapesequenz  
  
 Eine Escapesequenz angezeigt, wo eine nicht erwartet wurde.  
  
 Eine Escapesequenz – ein umgekehrter Schrägstrich (  **\\**  ) gefolgt vom Buchstaben oder eine Zahl – möglicherweise nur in Zeichens oder einer Zeichenfolge auftreten konstant.
