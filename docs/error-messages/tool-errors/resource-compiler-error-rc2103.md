# Ressourcencompiler: Fehler RC2103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unerwartetes Dateiende in Zeichenfolgenliteral  
  
 Ein Dateiende wurde vor dem Ende einer Zeichenfolge gefunden.  Der Zeichenfolge fehlt möglicherweise das schließende doppelte Anführungszeichen \(**"**\).
