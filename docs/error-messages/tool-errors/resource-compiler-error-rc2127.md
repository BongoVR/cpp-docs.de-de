# Ressourcencompiler: Fehler RC2127
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Versions\-WORD\-Angaben mit Kommas erwartet  
  
 Versionsnummern in einer Versionsressource sollten den Typ **WORD** aufweisen und durch Kommas voneinander getrennt sein.
