# <a name="resource-compiler-error-rc2167"></a>Ressourcencompiler: Fehler RC2167
Unbekanntes VERSIONINFO-Feld; BEGIN oder Komma erwartet.  
  
 Ein unbekanntes Feld wurde gefunden, der **FIXED** Teil einer **VERSIONINFO** Strukturdeklaration. Ein **VERSIONINFO** Feld muss DWORDS durch ein Komma getrennt sein.
