# Ressourcencompiler: Fehler RC2167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

VERSIONINFO\-Feld nicht erkannt; BEGIN oder Komma erwartet  
  
 Ein nicht erkanntes Feld wurde im FIXED\-Teil einer VERSIONINFO\-Strukturdeklaration gefunden.  Ein VERSIONINFO\-Feld muss den Typ `DWORD` aufweisen und durch ein Komma getrennt werden.
