# Ressourcencompiler: Fehler RC2169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Ressourcendatei "**   
 ***Dateiname* " nicht im 2.03\-Format**  
  
 Das Format einer früheren Version als Version 2.03 wurde von der angegebenen Ressource verwendet.  Die Ressourcendatei muss konvertiert oder im Format für Version 3.00 oder höher neu erstellt werden.
