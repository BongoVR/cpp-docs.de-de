# <a name="resource-compiler-error-rc2169"></a>Ressourcencompiler: Fehler RC2169
Ressourcen-Dateiname entspricht nicht 2.03-Format  
  
 Die angegebene Ressource wird ein Format älter als Version 2.03 verwendet. Die Ressourcendatei muss konvertiert werden oder neu erstellt, mit dem Format für Version 3.00 oder höher.
