# Ressourcencompiler: Fehler RC2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**Ressourcendatei "**   
 ***Dateiname* " nicht im 3.00\-Format**  
  
 Für die angegebene Ressource wurde das Format einer früheren Version als Version 3.00 verwendet.  Die Ressourcendatei muss konvertiert oder im Format für Version 3.00 oder höher neu erstellt werden.
