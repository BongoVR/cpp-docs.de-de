# Ressourcencompiler: Schwerwiegender Fehler RC1002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Kein weiterer Heap\-Speicher verfügbar  
  
### Beachten Sie die folgenden Vorschläge zur Problembehebung:  
  
1.  Vergrößern Sie den Speicher für die Windows\-Auslagerungsdatei.  Weitere Informationen über das Vergrößern der Auslagerungsdatei finden Sie unter dem Stichwort "Virtueller Speicher" in der Windows\-Hilfe.  
  
2.  Teilen Sie die aktuelle Datei in mehrere kleinere Dateien auf, und kompilieren Sie diese separat.  
  
3.  Entfernen Sie andere Programme oder Treiber, die auf dem System ausgeführt werden.
