# Ressourcencompiler: Schwerwiegender Fehler RC1004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unerwartetes Dateiende gefunden  
  
 Dieser Fehler kann durch fehlende Zeilenvorschub\- und Zeilenumschaltungszeichen in der letzten Zeile einer Textdatei verursacht werden.
