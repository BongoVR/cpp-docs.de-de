# Ressourcencompiler: Schwerwiegender Fehler RC1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Compiler\-Beschränkung: zu tiefe Schachtelung von Makros 'Makro'  
  
 Die Datei überschreitet die Ressourcencompilerbeschränkung für die Schachtelung von Makros.  Überarbeiten Sie die angegebene Quelldatei, um die Schachtelungstiefe der Makros zu verringern.
