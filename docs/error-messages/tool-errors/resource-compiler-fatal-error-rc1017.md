# Ressourcencompiler: Schwerwiegender Fehler RC1017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültiger Ausdruck für Ganzzahlkonstante  
  
 Der Ausdruck in einer \#if\-Direktive ist entweder nicht vorhanden oder lässt sich nicht zu einer Konstanten auswerten.
