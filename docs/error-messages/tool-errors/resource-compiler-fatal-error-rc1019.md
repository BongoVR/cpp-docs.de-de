# Ressourcencompiler: Schwerwiegender Fehler RC1019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unerwartetes "\#else"  
  
 Die `#else`\-Direktive wurde nicht innerhalb eines der Konstrukte `#if`, **\#ifdef** oder **\#ifndef** verwendet.  
  
 Stellen Sie sicher, dass dieser Anweisung eine der Anweisungen `#if`, **\#ifdef** oder \#ifndef vorangeht.
