# Ressourcencompiler: Schwerwiegender Fehler RC1022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"\#endif" erwartet  
  
 Eine der Direktiven `#if`, **\#ifdef** oder **\#ifndef** wurde nicht mit einer `#endif`\-Direktive beendet.  
  
 Stellen Sie sicher, dass dieser Anweisung eine der Anweisungen `#if`, **\#ifdef** oder \#ifndef vorangeht.
