# <a name="resource-compiler-fatal-error-rc1102"></a>Ressourcencompiler: Schwerwiegender Fehler RC1102
Interner Fehler: zu viele Argumente für RCPP  
  
 Zu viele Argumente wurden an den Ressourcencompiler Präprozessor übergeben. Reduzieren Sie die Anzahl der Symbole, die mit dem Definieren von Symbolen definiert (/ d) Option, indem Sie sie in der Datenquelle zu definieren. Dieser Fehler kann auch verursacht werden, durch Angabe zu viele Includedateien Search Dateipfade, die mit der Option Suchpfad einschließen (/ i).
