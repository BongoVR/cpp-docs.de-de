# Ressourcencompiler: Schwerwiegender Fehler RC1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültiger Schalter, ungültige Option: zu viele \/d\-Schalter  
  
 Es wurden zu viele Symbole mit der Option zum Definieren von Symbolen \(**\/d**\) definiert.  Definieren Sie einige Symbole in der Quelle, und kompilieren Sie neu.
