# Ressourcencompiler: Schwerwiegender Fehler RC1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

E\/A\-Fehler beim Lesen einer Datei  
  
 Eine Datei konnte nicht gelesen werden.  Überprüfen Sie, ob das Laufwerk, das die Datei enthält, zur Verfügung steht und ob die Datei gültig ist.
