# Ressourcencompiler: Schwerwiegender Fehler RC1203
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültige hexadezimale standardmäßige Sprachen\-ID angegeben  
  
 Auf die Option zur Angabe der Standardsprache \(\/l\) folgte die Angabe einer ungültigen hexadezimalen Sprach\-ID.  Eine Liste der gültigen Sprach\-IDs finden Sie unter [Language and Country\/Region Strings](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) in der *Run\-Time Library Reference*.
