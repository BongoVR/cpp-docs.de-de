# Ressourcencompiler: Schwerwiegender Fehler RC1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültige Codepage  
  
 Auf die Option zur Angabe einer Codepage \(**\/c**\) folgte eine ungültige Codepageangabe.  Weitere Informationen finden Sie unter [Code Pages](../../c-runtime-library/code-pages.md) \(nur auf Englisch verfügbar\) in der *Run\-Time Library Reference*.
