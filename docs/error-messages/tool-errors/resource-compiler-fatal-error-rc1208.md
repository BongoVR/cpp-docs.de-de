# Ressourcencompiler: Schwerwiegender Fehler RC1208
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Eingabedatei hat die Erweiterung .RES  
  
 Die RES\-Erweiterung wird für die Ausgabe des Ressourcencompilers verwendet.  Die RC\-Erweiterung sollte für Ressourcencompilerskripts verwendet werden.
