# Ressourcencompiler: Schwerwiegender Fehler RW1023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

E\/A\-Fehler beim Speichern einer Datei, Laufwerk voll  
  
 Der freie Speicherplatz muss mindestens zweimal so groß wie die ausführbare Datei sein.
