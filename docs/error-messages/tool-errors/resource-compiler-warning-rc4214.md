# Ressourcencompiler: Warnung RC4214
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Ungültige Codepage : ignoriert  
  
 Die RC\-Datei enthielt ein Codepageargument, aber die angegebene Codepage ist ungültig.  Weitere Informationen finden Sie unter [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674).
