# Ressourcencompiler: Warnung RW4003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

SHIFT oder CONTROL ohne VIRTKEY verwendet  
  
 In einer Zugriffstastentabelle ist **VIRTKEY** für **SHIFT** und **CONTROL** erforderlich.  Da **SHIFT** und **CONTROL** bei einer Zugriffstaste vom Typ **VIRTKEY** als Attributbits angezeigt werden, können diese Tasten nicht unabhängig von einem **VIRTKEY** vorhanden sein.
