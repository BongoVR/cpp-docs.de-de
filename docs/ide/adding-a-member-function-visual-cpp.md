# Hinzuf&#252;gen einer Memberfunktion (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In der **Klassenansicht** können Sie einer beliebigen Klasse eine Memberfunktion hinzufügen.  Dabei wird der Headerdatei eine Deklaration und der Implementierungsdatei der Klasse ein Stub\-Memberfunktionsrumpf hinzugefügt. Die hinzugefügten Elemente können anschließend bearbeitet werden.  
  
### So fügen Sie einer Klasse eine Memberfunktion hinzu  
  
1.  Erweitern Sie in der **Klassenansicht** den Projektknoten, um die Projektklassen anzuzeigen. \(Öffnen Sie die **Klassenansicht**, indem Sie in der Menüleiste **Ansicht** und dann **Klassenansicht** wählen.\)  
  
2.  Öffnen Sie das Kontextmenü für die Klasse, der Sie eine Memberfunktion hinzufügen möchten, und wählen Sie dann **Hinzufügen** und **Funktion hinzufügen** aus.  
  
3.  Geben Sie die entsprechenden Informationen zur Memberfunktion an.  Weitere Informationen finden Sie unter [Assistent zum Hinzufügen von Memberfunktionen](../ide/add-member-function-wizard.md).  
  
4.  Klicken Sie auf die Schaltfläche **Fertig stellen**, um den Code der Memberfunktion zu erstellen.
