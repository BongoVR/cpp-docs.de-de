# <a name="adding-a-member-function-visual-c"></a>Hinzufügen einer Memberfunktion (Visual C++)
In **Klassenansicht**, können Sie jede Klasse eine Memberfunktion hinzufügen. Dabei wird der Headerdatei eine Deklaration und der Implementierungsdatei der Klasse ein Stub-Memberfunktionsrumpf hinzugefügt. Die hinzugefügten Elemente können anschließend bearbeitet werden.  
  
### <a name="to-add-a-member-function-to-a-class"></a>So fügen Sie einer Klasse eine Memberfunktion hinzu  
  
1.  In **Klassenansicht**, erweitern Sie den Projektknoten, um die Klassen im Projekt anzuzeigen. (Öffnen **Klassenansicht**, wählen Sie in der Menüleiste **Ansicht**, **Klassenansicht**.)  
  
2.  Öffnen Sie das Kontextmenü für die Klasse, die Sie verwenden möchten, fügen Sie eine Memberfunktion hinzu, und wählen Sie dann **hinzufügen**, **Funktion hinzufügen**.  
  
3.  Geben Sie die entsprechenden Informationen zur Memberfunktion an. Weitere Informationen finden Sie unter [Assistenten zum Hinzufügen von Funktion](../ide/add-member-function-wizard.md).  
  
4.  Wählen Sie die **Fertig stellen** Schaltfläche, um den Code der Memberfunktion zu generieren.
