# Eigenschaftenseiten &quot;Befehlszeile&quot;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die meisten Ordner mit Eigenschaftenseiten enthalten eine Eigenschaftenseite **Befehlszeile**.  Auf dieser Seite wird angezeigt, welche Eigenschaften im Ordner festgelegt sind.  Die Eigenschaftenseite **Befehlszeile** enthält außerdem ein Feld **Zusätzliche Optionen**, in dem Sie Eigenschaften festlegen können, die zwar für das Tool gültig sind, für die aber keine Eigenschaft im Ordner existiert.  
  
 Alle in das Bearbeitungsfeld eingegebenen Befehle werden an das Tool des betreffenden Ordners übergeben.  Die Eingaben werden keinerlei Prüfung unterzogen, und es findet auch keine Abhängigkeitsprüfung statt.  
  
 Informationen dazu, wie Sie auf die Eigenschaftenseiten **Befehlszeile** zugreifen, finden Sie unter [Gewusst wie: Festlegen von Projekteigenschaften mit Eigenschaftenseiten](../misc/how-to-specify-project-properties-with-property-pages.md).
