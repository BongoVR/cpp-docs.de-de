# GetProjectPath
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Gibt den Verzeichnispfad des Projekts zurück.  
  
## Syntax  
  
```  
  
      function GetProjectPath(   
   oProj    
);  
```  
  
#### Parameter  
 `oProj`  
 Das ausgewählte Projekt.  
  
## Rückgabewert  
 Der Verzeichnispfad des Projekts.  
  
## Hinweise  
 Rufen Sie diese Funktion auf, um den Verzeichnispfad des Projekts zu ermitteln.
