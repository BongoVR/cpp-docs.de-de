# Makefile-Projekt-Assistent
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Mit diesem Assistenten erstellen Sie eine Befehlszeile für das auszuführende Buildmodul.  Nachdem Sie das Projekt erstellt haben, können Sie die Projekteigenschaften auf der [Eigenschaftenseite](../ide/working-with-project-properties.md) des Projekts einsehen und bearbeiten.  
  
## Übersicht  
 Auf dieser Assistentenseite werden die aktuellen Projekteinstellungen für das Projekt beschrieben, das Sie erstellen.  Das Projekt ist standardmäßig so konfiguriert, dass sowohl Debug\- als auch Releasekonfigurationen von ProjName**.exe** erstellt werden.  
  
 Um diese Standardeinstellungen zu ändern, klicken Sie in der linken Spalte des Assistenten auf [Anwendungseinstellungen](../ide/application-settings-makefile-project-wizard.md) und nehmen die gewünschten Änderungen vor.
