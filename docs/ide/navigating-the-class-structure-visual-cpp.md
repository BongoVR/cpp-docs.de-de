# <a name="navigating-the-class-structure-visual-c"></a>Navigieren in der Klassenstruktur (Visual C++)
Sie können eine Klasse Projektstruktur, einschließlich Funktionen, Vererbung und Schnittstellen, die entweder navigieren [Klassenansicht](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925) oder [Objektkatalog](http://msdn.microsoft.com/en-us/f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 Sie können auf eine Funktionsdeklaration durch Doppelklicken auf den Funktionsnamen in der Klassenansicht navigieren. Sie können auf eine Klassendefinition durch Doppelklicken auf einen Klassennamen in der Klassenansicht navigieren.
