# Navigieren in der Klassenstruktur (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Sie können in der Klassenstruktur eines Projekts navigieren und dabei Funktionen, Vererbungen und Schnittstellen entweder in der [Klassenansicht](assetId:///8d7430a9-3e33-454c-a9e1-a85e3d2db925) oder im [Objektkatalog](assetId:///f89acfc5-1152-413d-9f56-3dc16e3f0470) durchsuchen.  
  
 Indem Sie in der Klassenansicht auf einen Funktionsnamen doppelklicken, wechseln Sie zur Funktionsdeklaration.  Um zu einer Klassendefinition zu wechseln, doppelklicken Sie in der Klassenansicht auf einen Klassennamen.
