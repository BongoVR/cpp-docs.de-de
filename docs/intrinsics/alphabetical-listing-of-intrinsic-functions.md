# Alphabetische Liste von systeminternen Funktionen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgenden Abschnitte beschreiben die Microsoft\-spezifischen systeminternen Funktionen, die in einigen oder allen Architekturen verfügbar sind.  Andere unterstützten systeminternen Funktionen sind von Prozessorherstellern die in den Headerdateien oder auf ihren Websites dokumentiert.  Links zu Dokumentation von Herstellern finden Sie in den Abschnitten zu [systeminternen ARM\-Funktionen](../intrinsics/arm-intrinsics.md), [systeminternen x86\-Funktionen](../intrinsics/x86-intrinsics-list.md) und [systeminternen x64\-Funktionen](../intrinsics/x64-amd64-intrinsics-list.md).  C Run\-Time Library \(CRT\)\-Funktionen, für die systeminterne Funktionen implementiert werden, sind hier nicht dokumentiert; diese Funktionen sind unter [C\-Laufzeitbibliotheksverweis](../c-runtime-library/c-run-time-library-reference.md) dokumentiert.
