# <a name="active-document-classes"></a>Active Document-Klassen
Aktive Dokumente können angezeigt werden, in der gesamten Clientfenster eines Webbrowsers, z. B. Internet Explorer 5.5 oder in einem aktiven Container, z. B. die Microsoft Office Binder, die aktive Dokumente unterstützt.  
  
 [CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)  
 Ordnet die Active Document-Schnittstellen und initialisiert und aktiviert ein Active Document-Objekt.  
  
 [CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)  
 Implementiert OLE-Serververben speziell für Active Document-Server.  
  
 [COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)  
 Implementiert "Active Document Containment".
