# H&#228;ufig hinzugef&#252;gte Memberfunktionen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Wenn das Dialogfeld PushButtons anders OK oder Abbrechen enthält, müssen Sie Meldungshandlermemberfunktionen in der Dialogfeldklasse schreiben, um auf die Steuerelement\-Benachrichtigungen zu reagieren, die sie generieren.  Ein Beispiel finden Sie das Beispielprogramm [Scribble](../top/visual-cpp-samples.md).  Steuerelement\-Benachrichtigungen Sie können von anderen Steuerelementen im Dialogfeld auch bearbeiten.
