# <a name="creating-the-dialog-resource"></a>Erstellen der Dialogfeldressource
Entwurf der [Dialogfeld](../mfc/dialog-boxes.md) und Erstellen der Dialogfeldressource, verwenden Sie die [Dialog-Editor](../windows/dialog-editor.md). In den Dialog-Editor können Sie folgende Aktionen ausführen:  
  
-   Passen Sie die Größe und Position, die das Dialogfeld besitzt, wenn es angezeigt wird.  
  
-   Ziehen Sie verschiedene Arten von Steuerelementen aus einer Steuerelementpalette und legen Sie ab, wo sollen sie im Dialogfeld.  
  
-   Positionieren Sie die Steuerelemente mit von Ausrichtungsschaltflächen auf der Symbolleiste.  
  
-   Testen Sie das Dialogfeld, indem simuliert das Aussehen und Verhalten in Ihrem Programm haben wird. Im Testmodus können Sie das Dialogfeld Steuerelemente durch Eingabe von Text in Textfeldern ändern, die durch das Klicken auf Schaltflächen und so weiter.  
  
 Wenn Sie fertig sind, wird die Dialogfeldvorlagen-Ressource in Ihrer Anwendung Ressourcenskriptdatei gespeichert. Sie können ihn später bearbeiten, bei Bedarf. Eine vollständige Beschreibung zum Erstellen und Bearbeiten von Ressourcen finden Sie unter der [Dialog-Editor](../windows/dialog-editor.md) Themen. Diese Technik wird auch zum Erstellen der Dialogfeld-Vorlagenressourcen für [CFormView](../mfc/reference/cformview-class.md) und [CRecordView](../mfc/reference/crecordview-class.md) Klassen.  
  
 Wenn das Dialogfeld Darstellung am besten entspricht, erstellen eine Dialogfeldklasse, und ordnen Sie die Nachrichten, wie in beschrieben [Erstellen einer Dialogfeldklasse mit Code-Assistenten](../mfc/creating-a-dialog-class-with-code-wizards.md).
