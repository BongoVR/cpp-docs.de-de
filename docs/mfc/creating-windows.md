# Erstellen von Fenstern
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Das Framework stellt automatisch die meisten Fenster, die Sie in einem Rahmenprogramm benötigen.  [Dokument\/Ansichts\-Erstellung](../mfc/document-view-creation.md) zeigt, wie das Framework diese Rahmenfenster, die mit Dokumenten und Ansichten zugeordnet werden.  Zu besonderen Zwecken können Sie eigene Fenster \- einschließlich der eigenen untergeordneten Fenster Rahmenfenster oder Ansichten \- neben den Fenstern erstellen, die vom Framework angegeben werden.
