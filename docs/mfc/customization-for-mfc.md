# Anpassung f&#252;r MFC
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Dieses Thema enthält Tipps für das Anpassen einer MFC\-Anwendung bereit.  
  
## Allgemeine Anpassungen  
 Sie können den Zustand der Anwendung zur Registrierung speichern und laden.  Wenn Sie diese Option aktivieren, wird die Anwendung den Anfangszustand aus der Registrierung.  Wenn Sie das ursprüngliche Andockenlayout der Anwendung ändern, müssen Sie die Registrierungsdaten für die Anwendung deaktivieren.  Andernfalls setzen die Daten in der Registrierung alle Änderungen, die Sie für das ursprüngliche Layout vorgenommen haben.
