# <a name="drawing-tool-classes"></a>Klassen für Zeichenwerkzeuge
Diese Klassen kapseln Zeichentools an, die verwendet werden, um auf einen Gerätekontext zu zeichnen.  
  
 [CGdiObject](../mfc/reference/cgdiobject-class.md)  
 Die Basisklasse für GDI Zeichentools.  
  
 [CBrush](../mfc/reference/cbrush-class.md)  
 Kapselt einen GDI-Pinsel, der als den aktuellen Pinsel in einem Gerätekontext ausgewählt werden kann. Pinsel dienen zum Ausfüllen der Innenbereiche der Objekte, die gerade gezeichnet wird.  
  
 [CPen](../mfc/reference/cpen-class.md)  
 Kapselt einen Stift der GDI, der als der aktuelle Stift in einem Gerätekontext ausgewählt werden kann. Stifte dienen zum Zeichnen von Objekten der Rahmenlinien.  
  
 [CFont](../mfc/reference/cfont-class.md)  
 Kapselt eine GDI-Schriftart, die als die aktuelle Schriftart in einem Gerätekontext ausgewählt werden kann.  
  
 [CBitmap](../mfc/reference/cbitmap-class.md)  
 Kapselt eine GDI-Bitmap, die eine Schnittstelle für die Bearbeitung von Bitmaps bereitstellt.  
  
 [CPalette](../mfc/reference/cpalette-class.md)  
 Kapselt eine GDI-Farbpalette für die Verwendung als Schnittstelle zwischen der Anwendung und eine Farbe Ausgabegerät z. B. eine Anzeige.  
  
 [CRectTracker](../mfc/reference/crecttracker-class.md)  
 Zeigt an, und die Benutzeroberfläche für das Ändern der Größe und Verschieben von rechteckigen Objekten behandelt.
