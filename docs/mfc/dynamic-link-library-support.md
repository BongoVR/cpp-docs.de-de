# <a name="dynamic-link-library-support"></a>Dynamic Link Library-Unterstützung
Die NAFXCW.lib und NAFXCWD.lib-Bibliotheken (in der Static Link Library-Benennungskonventionen Tabelle aufgeführten [Bibliotheks-Benennungskonventionen](../mfc/library-naming-conventions.md)) erstellen Sie das Projekt als Dynamic Link Library, eine "normale MFC-DLL" aufgerufen (früher eine "USRDLL") kann mit Anwendungen, die nicht mit der Klassenbibliothek verwendet werden. Diese DLL-Unterstützung ist nicht identisch mit MFCx0.DLL und MFCx0D.DLL (bekannt als AFXDLL), die die gesamte Klassenbibliothek in einer DLL enthalten. Weitere Informationen finden Sie unter [DLLs](../build/dlls-in-visual-cpp.md). Eine Tabelle von DLL-Namen finden Sie unter [Namenskonventionen für MFC-DLLs](../build/naming-conventions-for-mfc-dlls.md).
