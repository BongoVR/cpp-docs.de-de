# Editing Graphical Resources (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Grafische Ressourcen sind Bilder, die Sie für die Anwendung definieren.  Es können Freihandobjekte oder Formen gezeichnet werden.  Teile eines Bildes können markiert und bearbeitet, gekippt oder in ihrer Größe verändert werden. Außerdem kann von einem markierten Teil eines Bildes ein benutzerdefinierter Pinsel erstellt und mit diesem Pinsel dann gezeichnet werden.  Weiterhin können Bildeigenschaften definiert, Bilder in verschiedenen Formaten gespeichert bzw. in andere Formate konvertiert werden.  
  
 Zusätzlich zur Erstellung neuer grafischer Ressourcen können Sie [vorhandene Bilder zu Bearbeitungszwecken importieren](../windows/how-to-import-and-export-resources.md) und diese anschließend dem Projekt hinzufügen.  Darüber hinaus können Sie Bilder, die keinem Projekt angehören, öffnen und [unabhängig bearbeiten](../mfc/editing-an-image-outside-of-a-project-image-editor-for-icons.md).
