# Verarbeiten von Windows-Meldungen in einem Dialogfeld
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Dialogfelder sind Fenster, sodass sie Windows\-Meldungen bearbeiten, wenn Sie die entsprechenden Handlerfunktionen stellen.  Wenn Sie die Dialogfeldklasse mit dem Assistenten zum Hinzufügen von Klassen erstellen, fügt der Assistent eine leere Meldungszuordnung der Klasse hinzu.  Verwenden Sie das Eigenschaftenfenster, um allen möglichen Windows\-Meldungen oder Befehlen zuzuordnen, die Sie der Klasse behandeln soll.  
  
 Weitere Informationen finden Sie unter [Zuordnungs\-Windows\-Meldungen zu der Dialogfeldklasse](../mfc/mapping-windows-messages-to-your-class.md).
