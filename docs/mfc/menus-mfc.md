# <a name="menus-mfc"></a>Menüs (MFC)
MFC stellt zwei Elemente, um die Arbeit mit Menüs zu erleichtern:  
  
-   Klasse [CMenu](../mfc/reference/cmenu-class.md) zum Bearbeiten von Menüs des Programms zur Laufzeit. Verwenden Sie die Dokumentation für `CMenu` und der Stichprobe, die Informationen zum Verwenden `CMenu` effektiv.  
  
-   Einen Mechanismus zum Aktualisieren von Menüs und Symbolleisten-Schaltflächen: Aktivieren oder deaktivieren sie bei Bedarf entsprechend der aktuellen programmbedingungen.  
  
 Visual C++ bietet auch eine [Menü-Editor](../windows/menu-editor.md) zum Erstellen und bearbeiten Menüressourcen des Programms.
