# <a name="mfc-technical-notes"></a>Technische Hinweise zu MFC
Ein technischer betrachtet ist ein Dokument, das für Programmierer, die von Programmierern geschrieben.  
  
 Jede technische Hinweis beschreibt ein Problem oder eine Funktion, die Gegenstand der Rest der MFC-Dokumentation. Die technische Hinweise angegeben wider Anforderungen für Informationen von Benutzern als auch spezielle Informationen, die die MFC-Entwickler erwarten erfahrene Benutzer benötigen.  
  
 Es gibt zwei Möglichkeiten, die technische Hinweise zu durchsuchen:  
  
-   [Technische Hinweise – nach Anzahl](../mfc/technical-notes-by-number.md)  
  
-   [Technische Hinweise – nach Kategorie](../mfc/technical-notes-by-category.md)
