# <a name="ready-to-use-array-classes"></a>Klassen für gebrauchsfertige Arrays
Die folgenden Klassen sind Arrayklassen kann jetzt zu verwendende.  
  
 [CByteArray](../mfc/reference/cbytearray-class.md)  
 Speichert die Elemente des Typs **BYTE** in einem Array.  
  
 [CDWordArray](../mfc/reference/cdwordarray-class.md)  
 Speichert die Elemente des Typs `DWORD` in einem Array.  
  
 [CObArray](../mfc/reference/cobarray-class.md)  
 Speichert Zeiger auf Objekte der Klasse `CObject` oder auf Objekte von abgeleiteten Klassen `CObject` in einem Array.  
  
 [CPtrArray](../mfc/reference/cptrarray-class.md)  
 Speichert Zeiger zu `void` (generische Zeiger) in einem Array.  
  
 [CUIntArray](../mfc/reference/cuintarray-class.md)  
 Speichert die Elemente des Typs **"uint"** in einem Array.  
  
 [CWordArray](../mfc/reference/cwordarray-class.md)  
 Speichert die Elemente des Typs **WORD** in einem Array.  
  
 [CStringArray](../mfc/reference/cstringarray-class.md)  
 Speichert `CString` Objekte in einem Array.
