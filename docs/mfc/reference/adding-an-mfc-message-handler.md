# Hinzuf&#252;gen eines MFC-Meldungshandlers
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Sie verwenden das [Eigenschaftenfenster für das Hinzufügen eines Meldungshandlers](../../mfc/reference/mapping-messages-to-functions.md) \(eine Memberfunktion, die Windows\-Meldungen verarbeitet\) zu einer Klasse und für das Zuordnen von Windows\-Meldungen zum Meldungshandler.  Außerdem können Sie einen [Ereignishandler für ein beliebiges Dialogfeld\-Steuerelement](../../mfc/adding-event-handlers-for-dialog-box-controls.md) hinzufügen  
  
 Wenn Sie Meldungs\- und Ereignisbehandlungsfunktionen im Eigenschaftenfenster definieren, können Sie die Meldungsdispatchtabelle \(oder Meldungszuordnung\) sowie die Klassenheaderdatei automatisch aktualisieren.  
  
> [!NOTE]
>  Grundsätzlich können Sie einer ATL\-Klasse einen Meldungshandler im Eigenschaftenfenster hinzufügen, die Resultate sind jedoch nicht immer genau vorhersehbar.  Weitere Informationen finden Sie unter dem ATL\-Thema [Hinzufügen eines ATL\-Meldungshandlers](../../atl/adding-an-atl-message-handler.md).
