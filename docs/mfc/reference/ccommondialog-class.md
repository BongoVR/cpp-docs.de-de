# CCommonDialog Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die Basisklasse für Klassen, die Funktionen des Windows\-Common kapseln, Dialogfelder.  
  
## Syntax  
  
```  
class CCommonDialog : public CDialog  
```
