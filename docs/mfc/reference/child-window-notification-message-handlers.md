|Kategorie|Beschreibung|  
|--------------|-----------------|  
|[Generischer Steuerelementhandler](../../mfc/reference/generic-control-handler.md)|Handler für generische Steuerungscodes-Benachrichtigung.|  
|[Handler für Benutzerschaltflächen](../../mfc/reference/user-button-handlers.md)|Handler für den Benutzer auf die Schaltfläche Benachrichtigungscodes.|
