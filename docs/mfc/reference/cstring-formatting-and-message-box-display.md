# CString-Formatierung und Meldungsfeldanzeige
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Einige Funktionen werden bereitgestellt, um `CString`\-Objekte zu formatieren und zu analysieren.  Sie können diese Funktionen verwenden, wenn Sie `CString`\-Objekte bearbeiten müssen, sind aber bei Formatierungszeichenfolgen besonders nützlich, die im Meldungsfeldtext werden.  
  
 Diese Gruppe Funktionen enthält auch eine globale Routine für das Anzeigen eines Meldungsfelds.  
  
### CString\-Funktionen  
  
|||  
|-|-|  
|[AfxExtractSubString](../Topic/AfxExtractSubString.md)|Extrahiert die Teilzeichenfolgen, die ein einzelnes Zeichen aus einer angegebenen Quellzeichenfolge getrennt werden.|  
|[AfxFormatString1](../Topic/AfxFormatString1.md)|Ersetzt eine angegebene Zeichenfolge für die Formatzeichen "%1 in einer Zeichenfolge, die in der Zeichenfolgentabelle enthalten ist.|  
|[AfxFormatString2](../Topic/AfxFormatString2.md)|Ersetzung zwei Zeichenfolgen für die Formatzeichen "%1 "oder "%2" in einer Zeichenfolge enthalten in der Zeichenfolgentabelle.|  
|[AfxMessageBox](../Topic/AfxMessageBox.md)|Zeigt ein Meldungsfeld an.|
