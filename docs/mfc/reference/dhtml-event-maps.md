# DHTML-Ereigniszuordnungen
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die folgenden Makros können verwendet werden, um DHTML\-Ereignisse zu behandeln.  
  
## DHTML\-Ereigniszuordnungs\-Makros  
 Die folgenden Makros können verwendet werden, um DHTML\-Ereignisse in [CDHtmlDialog](../../mfc/reference/cdhtmldialog-class.md) abgeleitete Klassen zu behandeln.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_EVENT\_MAP](../Topic/BEGIN_DHTML_EVENT_MAP.md)|Markiert den Beginn der DHTML\-Ereigniszuordnung.|  
|[BEGIN\_DHTML\_EVENT\_MAP\_INLINE](../Topic/BEGIN_DHTML_EVENT_MAP_INLINE.md)|Markiert den Beginn der DHTML\-Ereigniszuordnung.|  
|[DECLARE\_DHTML\_EVENT\_MAP](../Topic/DECLARE_DHTML_EVENT_MAP.md)|Deklariert die DHTML\-Ereigniszuordnung.|  
|[DHTML\_EVENT](../Topic/DHTML_EVENT.md)|Wird verwendet, um ein Ereignis auf der Dokumentebene für ein einzelnes HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_AXCONTROL](../Topic/DHTML_EVENT_AXCONTROL.md)|Wird verwendet, um ein Ereignis zu behandeln ausgelöst durch ein ActiveX\-Steuerelement.|  
|[DHTML\_EVENT\_CLASS](../Topic/DHTML_EVENT_CLASS.md)|Wird verwendet, um ein Ereignis auf der Dokumentebene für alle HTML\-Elemente mit einer Klasse eine bestimmte CSS zu behandeln.|  
|[DHTML\_EVENT\_ELEMENT](../Topic/DHTML_EVENT_ELEMENT.md)|Wird verwendet, um ein Ereignis auf Elementebene zu behandeln.|  
|[DHTML\_EVENT\_ONAFTERUPDATE](../Topic/DHTML_EVENT_ONAFTERUPDATE.md)|Wird verwendet, um das **onafterupdate**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONBEFOREUPDATE](../Topic/DHTML_EVENT_ONBEFOREUPDATE.md)|Wird verwendet, um das **onbeforeupdate**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONBLUR](../Topic/DHTML_EVENT_ONBLUR.md)|Wird verwendet, um das **onblur**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONCHANGE](../Topic/DHTML_EVENT_ONCHANGE.md)|Wird verwendet, um das `onchange`\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONCLICK](../Topic/DHTML_EVENT_ONCLICK.md)|Wird verwendet, um das **onclick**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONDATAAVAILABLE](../Topic/DHTML_EVENT_ONDATAAVAILABLE.md)|Wird verwendet, um das **ondataavailable**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONDATASETCHANGED](../Topic/DHTML_EVENT_ONDATASETCHANGED.md)|Wird verwendet, um das **ondatasetchanged**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONDATASETCOMPLETE](../Topic/DHTML_EVENT_ONDATASETCOMPLETE.md)|Wird verwendet, um das **ondatasetcomplete**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONDBLCLICK](../Topic/DHTML_EVENT_ONDBLCLICK.md)|Wird verwendet, um das **ondblclick**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONDRAGSTART](../Topic/DHTML_EVENT_ONDRAGSTART.md)|Wird verwendet, um das **ondragstart**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONERRORUPDATE](../Topic/DHTML_EVENT_ONERRORUPDATE.md)|Wird verwendet, um das **onerrorupdate**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONFILTERCHANGE](../Topic/DHTML_EVENT_ONFILTERCHANGE.md)|Wird verwendet, um das **onfilterchange**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONFOCUS](../Topic/DHTML_EVENT_ONFOCUS.md)|Wird verwendet, um das **onfocus**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONHELP](../Topic/DHTML_EVENT_ONHELP.md)|Wird verwendet, um das `onhelp`\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONKEYDOWN](../Topic/DHTML_EVENT_ONKEYDOWN.md)|Wird verwendet, um das **onkeydown**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONKEYPRESS](../Topic/DHTML_EVENT_ONKEYPRESS.md)|Wird verwendet, um das **onkeypress**\-Ereignis von einem HTML\-Element zu behandeln.|  
|[DHTML\_EVENT\_ONKEYUP](../Topic/DHTML_EVENT_ONKEYUP.md)|Wird verwendet, um das **onkeyup**\-Ereignis von einem HTML\-Element zu behandeln.|

|[DHTML\_EVENT\_ONMOUSEOVER](../Topic/DHTML_EVENT_ONMOUSEOVER.md)|Wird verwendet, um das **onmouseover**\-Ereignis von einem HTML\-Element zu behandeln.|

|[DHTML\_EVENT\_ONRESIZE](../Topic/DHTML_EVENT_ONRESIZE.md)|Wird verwendet, um das **onresize**\-Ereignis von einem HTML\-Element zu behandeln.|

|[DHTML\_EVENT\_ONROWEXIT](../Topic/DHTML_EVENT_ONROWEXIT.md)|Wird verwendet, um das **onrowexit**\-Ereignis von einem HTML\-Element zu behandeln.|

## URL\-Ereigniszuordnungs\-Makros  
 Die folgenden Makros können verwendet werden, um DHTML\-Ereignisse in [CMultiPageDHtmlDialog](../../mfc/reference/cmultipagedhtmldialog-class.md) abgeleitete Klassen zu behandeln.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_URL\_EVENT\_MAP](../Topic/BEGIN_DHTML_URL_EVENT_MAP.md)|Markiert den Beginn von paginierten DHTML und der URL\-Ereigniszuordnung.|  
|[BEGIN\_EMBED\_DHTML\_EVENT\_MAP](../Topic/BEGIN_EMBED_DHTML_EVENT_MAP.md)|Markiert den Beginn einer eingebetteten DHTML\-Ereigniszuordnung.|  
|[BEGIN\_URL\_ENTRIES](../Topic/BEGIN_URL_ENTRIES.md)|Markiert den Beginn einer URL\-Ereigniseintragszuordnung.|  
|[DECLARE\_DHTML\_URL\_EVENT\_MAP](../Topic/DECLARE_DHTML_URL_EVENT_MAP.md)|Deklariert das mehrseitige DHTML und die URL\-Ereigniszuordnung.|  
|[END\_DHTML\_URL\_EVENT\_MAP](../Topic/END_DHTML_URL_EVENT_MAP.md)|Markiert das Ende von paginierten DHTML und der URL\-Ereigniszuordnung.|  
|[END\_EMBED\_DHTML\_EVENT\_MAP](../Topic/END_EMBED_DHTML_EVENT_MAP.md)|Markiert das Ende einer eingebetteten DHTML\-Ereigniszuordnung.|  
|[END\_URL\_ENTRIES](../Topic/END_URL_ENTRIES.md)|Markiert das Ende einer URL\-Ereigniseintragszuordnung.|  
|[URL\_EVENT\_ENTRY](../Topic/URL_EVENT_ENTRY.md)|Ordnet eine URL oder eine HTML\-Ressource auf eine Seite in einem paginierten Dialogfeld zu.|
