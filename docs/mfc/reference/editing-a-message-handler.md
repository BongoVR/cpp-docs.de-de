# Bearbeiten eines Meldungshandlers
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Nachdem Sie einen Meldungshandler definiert haben, können Sie zur Definition der Memberfunktion wechseln, um Code hinzuzufügen oder zu ändern.  
  
 Um vom [Dialog\-Editor](../../mfc/dialog-editor.md) zur Definition einer Memberfunktion zu wechseln, doppelklicken Sie auf ein Steuerelement, für das bereits ein Handler definiert wurde.  Auf diese Weise gelangen Sie zu der Datei, in der der Meldungshandler des ausgewählten Steuerelements definiert ist.  
  
 Um von der [Klassenansicht](assetId:///8d7430a9-3e33-454c-a9e1-a85e3d2db925) zur Definition einer Memberfunktion zu wechseln, doppelklicken Sie in der Klassenansicht auf den Funktionsnamen.
