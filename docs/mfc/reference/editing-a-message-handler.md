# <a name="editing-a-message-handler"></a>Bearbeiten eines Meldungshandlers
Nachdem Sie einen Meldungshandler definiert haben, gelangen Sie zur Definition der Member-Funktion hinzufügen oder Ändern von Code.  
  
 Wechseln Sie zur Definition einer Memberfunktion aus der [Dialog-Editor](../../windows/dialog-editor.md), doppelklicken Sie auf ein Steuerelement für die bereits ein Handler definiert wird. Damit gelangen Sie auf die Datei, die in der Meldungshandler des ausgewählten Steuerelements definiert ist.  
  
 Wechseln Sie zur Definition einer Memberfunktion aus [Klassenansicht](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), doppelklicken Sie auf den Namen der Funktion in der Klassenansicht.
