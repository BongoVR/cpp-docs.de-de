# Interne Klassen
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die folgenden Klassen sind in MFC intern verwendet.  Vollständigkeitshalber werden diese internen Klassen in diesem Abschnitt beschreiben. Sie sollen allerdings nicht direkt im Code verwendet werden.  
  
## In diesem Abschnitt  
  
|Klasse|**Beschreibung**|  
|------------|----------------------|  
|`CCommandManager Class`|Die `CCommandManager`\-Klasse verwaltet Befehle und ihre Zuordnung mit Bildern.|
