# Globale Variablen zur Analyse von Internet-URLs
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Wenn ein Client eine Abfrage zum Internet\-Server sendet, können Sie einen der URL\-Analyseglobalen werte verwenden, um Informationen über den Client zu extrahieren.  
  
### Internet Analyse\-globaleWerte URL  
  
|||  
|-|-|  
|[AfxParseURL](../Topic/AfxParseURL.md)|Analysiert eine URL\-Zeichenfolge und gibt den Typ des Diensts und seiner Komponenten zurück.|  
|[AfxParseURLEx](../Topic/AfxParseURLEx.md)|Analysiert eine URL\-Zeichenfolge und gibt den Typ des Diensts und seiner Komponenten zurück und stellt den Benutzernamen und das Kennwort bereit.|
