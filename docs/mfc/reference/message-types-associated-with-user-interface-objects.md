# Mit Benutzeroberfl&#228;chenobjekten verkn&#252;pfte Meldungstypen
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

In der folgenden Tabelle sehen Sie die Objekttypen, mit denen Sie arbeiten, sowie die Meldungstypen, mit denen diese verbunden sind.  
  
### Benutzeroberflächenobjekte und verknüpfte Meldungen  
  
|Objekt\-ID|Meldungen|  
|----------------|---------------|  
|Klassenname, der auf das enthaltende Fenster verweist.|Windows\-Meldungen, die für eine von [CWnd](../../mfc/reference/cwnd-class.md) abgeleitete Klasse geeignet sind: Dialogfeld, Fenster, untergeordnetes Fenster, untergeordnetes MDI\-Fenster oder Rahmenfenster der obersten Ebene.|  
|Menü\- oder Zugriffstasten\-ID|-   **COMMAND**\-Meldung \(führt die Programmfunktion aus\)<br />-   **UPDATE\_COMMAND\_UI**\-Meldung \(aktualisiert die Menüoption dynamisch\)|  
|Steuerelement\-ID|Steuerelement\-Benachrichtigungen für den ausgewählten Steuerelementtyp|
