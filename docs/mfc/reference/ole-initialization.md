# OLE-Initialisierung
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Bevor eine Anwendung OLE\-Systemdienste verwenden kann, muss sie die OLE\-System\-DLLs initialisieren und überprüfen, ob die DLLs der richtigen Version sind.  Die Funktion **AfxOleInit** initialisiert die OLE\-System\-DLLs.
