# Threadingmodelle der Remoteautomatisierung
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Remote\- Automatisierung unterstützt der Nicht\-Debugversion und die ApartmentThreadmodelle von COM.  Auf den 32\-Bit\- und 64\-Bit\-Plattformen die alle sind, die Visual C\+\+ unterstützt, sollte das Threadingmodell auf "Apartment festgelegt werden." Weitere Informationen dazu, wie und wo dies, finden Sie unter [Programme erstellen, die die Automatisierung verwenden](../mfc/creating-programs-that-use-remote-automation.md).
