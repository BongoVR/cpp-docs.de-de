# Verwenden des Dialog-Editors zum Hinzuf&#252;gen von Steuerelementen
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Wenn Sie die Dialogfeldvorlagen\-Ressource mit dem [Dialog\-Editor](../mfc/dialog-editor.md) erstellen, ziehen Sie Steuerelemente von einer Kontrollenpalette und legen sie im Dialogfeld ab.  Dies fügt der Spezifikation für diesen Steuerelementtyp der Dialogfeldvorlagen\-Ressource hinzu.  Wenn Sie ein Dialogfeldobjekt erstellen und seine **Erstellen** oder `DoModal`\-Memberfunktion aufrufen, erstellt das Framework ein Windows\-Steuerelement und platziert ihn im Dialogfeld auf dem Bildschirm.  
  
 Sie können stattdessen [Erstellen Sie Steuerelemente manuell](../mfc/adding-controls-by-hand.md), wenn Sie möchten.  Dies ist mehr Arbeit.
