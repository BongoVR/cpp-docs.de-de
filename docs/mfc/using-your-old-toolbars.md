# Verwenden fr&#252;herer Symbolleisten
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Wenn Sie vorherige Versionen von Visual C\+\+ verwendet haben, um benutzerdefinierte Symbolleisten zu erstellen, könnte die neue Implementierung der Klasse [CToolBar](../mfc/reference/ctoolbar-class.md) Sie Probleme verursachen.  Damit Sie die alten Symbolleisten nicht aufgeben müssen, um die neuen Features zu verwenden, wird die alte Implementierung immer noch unterstützt.  
  
 Das DOCKTOOL\-Beispiel verwendet nicht die alten Symbolleisten, nur die neuen Symbolleisten.  
  
 Sie können Symbolleisten alte mit dem Symbolleistenressourcen\-editor nicht bearbeiten.
