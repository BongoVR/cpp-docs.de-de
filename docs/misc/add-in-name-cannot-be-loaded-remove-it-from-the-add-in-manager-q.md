# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
Das ausgewählte Add\-In konnte nicht gefunden werden.  Dieser Fehler tritt i. A. auf, wenn eine DLL oder eine sonstige erforderliche Komponente für das ausgewählte Add\-In fehlt.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie das Add\-In neu.
