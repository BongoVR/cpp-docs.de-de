# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
Dieses Add\-In wurde nach einer Fehlfunktion beendet. Dies kann u. U. weitere Probleme in der integrierten Entwicklungsumgebung nach sich ziehen.  
  
### So beheben Sie diesen Fehler  
  
1.  Wählen Sie **Ja** aus, um das Add\-In aus der Liste der verfügbaren Add\-Ins zu entfernen.  
  
     \- oder \-  
  
     Wählen Sie Nein aus.  
  
    > [!NOTE]
    >  Wenden Sie sich an den Ersteller des Add\-Ins, und melden Sie das Problem.
