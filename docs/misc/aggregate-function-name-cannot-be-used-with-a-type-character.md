# Der Name der Aggregatfunktion darf nicht mit einem Typzeichen verwendet werden.
Ein Typzeichen befindet sich am Ende des Namens einer Aggregatfunktion.  
  
 **Fehler\-ID:** BC36617  
  
### So beheben Sie diesen Fehler  
  
1.  Entfernen Sie das Typzeichen aus dem Namen der Aggregatfunktion.
