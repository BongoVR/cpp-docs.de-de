# An invalid cookie was passed to UnregisterExtenderProvider().
Sie haben ein Cookie festgelegt, das nicht mehr gültig ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie denselben Cookiewert zurück, den Sie beim Aufruf von RegisterExtenderProvider\(\) erhalten haben.
