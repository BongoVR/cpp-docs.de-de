# Argument darf keine leere Zeichenfolge sein.
Ein Argument wurde als eine leere Zeichenfolge \(""\) festgelegt.  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie einen gültigen Wert für das Argument an.
