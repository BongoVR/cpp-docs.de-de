# Argument is missing in the pattern.
Dieser Fehler tritt normalerweise auf, wenn die Syntax eines in die Suchzeichenfolge eingegebenen regulären Ausdrucks unvollständig ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie die Zeichenfolge erneut ein.  Weitere Informationen zur Syntax finden Sie unter [Reguläre Ausdrücke](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
