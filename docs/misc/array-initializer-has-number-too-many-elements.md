# Der Arrayinitialisierer hat &lt;Anzahl&gt; Elemente zu viel.
Ihr Arrayinitialisierer enthält zu viele Elemente.  
  
 **Fehler\-ID:** BC30568  
  
### So beheben Sie diesen Fehler  
  
-   Verwenden Sie `ReDim`, um die Größe des Arrays zu ändern.
