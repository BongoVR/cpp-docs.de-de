# Das Automatisierungsobjekt hat keinen Standardwert.
[!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] kann den Standardmember für das angegebene Objekt nicht ermitteln.  
  
### So beheben Sie den Fehler  
  
-   Lesen Sie die Dokumentation des Objekts, und geben Sie eine explizite Spezifikation für die Eigenschaft oder Methode an.
