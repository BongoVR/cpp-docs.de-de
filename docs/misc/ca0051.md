# CA0051
Es wurden keine Regeln ausgewählt.  
  
 Dieser Fehler gibt an, dass das Analysetool für verwalteten Code zwar aktiviert ist, aber keine Regeln ausgewählt wurden.  Um diesen Fehler zu beheben, wählen Sie entsprechende Regeln aus.
