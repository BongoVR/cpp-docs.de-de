# CA0057
Der aktuellen Gruppe von Zielen und Regeln können keine Verstöße zugeordnet werden.  
  
 Dieser Fehler tritt auf, wenn FxCopCmd.exe manuell mit einer Projektdatei oder einem importierten XML\-Bericht aufgerufen wird, die bzw. der Verstöße aufweist, die der aktuellen Gruppe von Zielen und Regeln nicht zugeordnet werden können.  
  
 Tritt nur auf, wenn die Analyse mit der Befehlszeilenversion des Tools durchgeführt wird.
