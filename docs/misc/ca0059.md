# CA0059
Fehler bei Befehlszeilenschaltern.  
  
 Dieser Fehler tritt auf, wenn ein erforderlicher Befehlszeilenschalter fehlt, z. B.  
  
 \/out oder \/console switch is not specified.  Dieser Fehler tritt auch bei ungültigen Schaltern auf, z. B. bei einem ungültigen Timeoutwert.
