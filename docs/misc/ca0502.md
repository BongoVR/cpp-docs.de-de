# CA0502
Nicht unterstützte Sprache.  
  
 Dieser Fehler tritt auf, wenn der Benutzer versucht, ein ASP.NET\-Projekt zu analysieren, das mindestens eine Seite oder Klasse enthält, die nicht in C\# oder Visual Basic geschrieben wurde.  Um diesen Fehler zu beheben, können Sie die Datei aus dem Projekt entfernen oder die Codeanalyse deaktivieren.
