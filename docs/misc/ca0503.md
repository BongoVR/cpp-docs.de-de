# CA0503
Die Eigenschaft ist veraltet.  Verwenden Sie die Ersatzeigenschaft.  
  
 Die Warnung "CA0503" wird generiert, wenn eine veraltete Codeanalyseeigenschaft im Build oder in der Projektkonfigurationsdatei einen Wert enthält.  Verwenden Sie die angegebene Ersatzeigenschaft, um dieses Problem zu beheben.
