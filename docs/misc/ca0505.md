# CA0505
Die veraltete Eigenschaft wird ignoriert, da die ersetzende Eigenschaft definiert ist.  
  
 Warnung CA505 wird generiert, wenn ein Projekt oder eine Buildkonfigurationsdatei Werte für die angegebene veraltete und für die ersetzende Codeanalyseeigenschaft enthalten.  Zum Beheben dieses Problems entfernen Sie die veraltete Eigenschaft.
