# Cannot find component &#39;file&#39; of multifile assembly &#39;assembly&#39;
Eine Datei in einer Mehrfachdateiassembly wurde nicht gefunden.  Eine Mehrfachdateiassembly verweist auf eine oder mehrere Dateien, z. B. Modul\- oder Ressourcendateien.  
  
 Bei der kopierten Assembly kann es sich um einen direkten Verweis \(siehe den Knoten **Verweise** im Projektmappen\-Explorer\) oder um eine Assembly handeln, die eine Abhängigkeit von einem direkten Verweis aufweist.  
  
 **So beheben Sie diesen Fehler**  
  
-   Installieren Sie die Komponente neu.  
  
     Das Projekt kann eventuell erstellt, jedoch nicht ausgeführt werden.
