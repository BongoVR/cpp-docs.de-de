# Die Klasse ist auf dem lokalen Computer nicht registriert.
Dieser Fehler tritt in der Regel aufgrund einer beschädigten Datei aus. Sie müssen [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] möglicherweise neu installieren.  
  
### So beheben Sie diesen Fehler  
  
-   Installieren Sie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] neu.
