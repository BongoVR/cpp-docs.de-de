# Dialogfeld &quot;&#220;berschriebene Eigenschaften l&#246;schen&quot;
> [!NOTE]
>  Dieses Dialogfeld wird unter [!INCLUDE[vs_dev10_long](../build/includes/vs_dev10_long_md.md)] nicht unterstützt.  
  
 Wenn Sie einer Projektkonfiguration ein benutzerdefiniertes Eigenschaftenblatt \(eine PROPS\-Datei\) hinzufügen, erbt diese Projektkonfiguration standardmäßig alle in diesem Eigenschaftenblatt definierten Eigenschaften.  In diesem Dialogfeld können Sie für eine Projektkonfiguration nicht gewünschte geerbte Eigenschaften deaktivieren.
