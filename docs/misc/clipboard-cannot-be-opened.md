# Clipboard cannot be opened.
Im Allgemeinen tritt dieser Fehler auf, wenn Sie ein Element ausschneiden, kopieren oder einfügen möchten, der Zugriff auf die Zwischenablage jedoch nicht möglich ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie andere Programme, die möglicherweise die Zwischenablage verwenden, und versuchen Sie es erneut.
