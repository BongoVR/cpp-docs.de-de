# Command does not accept switches.
Im Allgemeinen tritt dieser Fehler auf, wenn ein Schalter für einen Befehl angegeben wurde, der nur Argumente erkennt.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die Syntaxanforderungen für den Befehl.  Weitere Informationen finden Sie unter [Visual Studio\-Befehle mit Argumenten](../Topic/Visual%20Studio%20Commands.md).
