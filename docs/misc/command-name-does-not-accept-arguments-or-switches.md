# Command &lt;name&gt; does not accept arguments or switches.
Im Allgemeinen tritt dieser Fehler auf, wenn ein Schalter oder ein Argument für einen Befehl oder Alias eingegeben wurde, der keine Argumenten oder Schaltern akzeptiert.  Wenn ein Alias eingegeben wurde, wird in der Fehlermeldung der vollständige Name des Befehls anstelle des Aliasnamens angezeigt.  Eine Liste mit gültigen Befehlen, bei denen Schalter und Argumente verwendet werden, finden Sie unter [Visual Studio\-Befehle](../Topic/Visual%20Studio%20Commands.md).  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie den Befehl und die Argumente neu ein, oder geben Sie einen neuen Befehl ein.
