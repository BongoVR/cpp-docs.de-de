# Command &lt;name&gt; is not available.
Je nach aktuellem Zustand von Visual Studio tritt dieser Fehler im Allgemeinen auf, wenn im **Befehlsfenster** im Feld **Suchen** ein ungültiger Befehl eingegeben wird.  Beispielsweise ist der `File.CloseSolution` \-Befehl ungültig, wenn eine Projektmappe in Visual Studio gegenwärtig nicht geöffnet ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Wählen Sie einen anderen Befehl aus.  
  
     – oder –  
  
     Treffen Sie eine andere Auswahl für den Befehl, mit dem gearbeitet werden soll.
