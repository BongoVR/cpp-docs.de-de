# Command &lt;name&gt; is not valid.
Im Allgemeinen tritt dieser Fehler auf, wenn ein Befehl oder ein Alias des Befehls, der von Visual Studio nicht erkannt wird, im **Befehlsfenster** im Feld **Suchen**  eingegeben wurde.  Wenn ein Alias eingegeben wurde, wird in der Fehlermeldung der vollständige Name des Befehls anstelle des Alias angezeigt.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie den Namen des Befehls erneut ein, oder wählen Sie einen gültigen Befehlsnamen aus.
