# Command requires at least one argument.
Im Allgemeinen tritt dieser Fehler auf, wenn keine Argumente für einen Befehl eingegeben wurden, der mindestens ein Argument benötigt.  
  
### So beheben Sie diesen Fehler  
  
1.  Schlagen Sie die korrekte Befehlssyntax in der Dokumentation nach, und versuchen Sie es erneut.
