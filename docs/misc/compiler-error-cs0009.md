# Compilerfehler CS0009
Die Metadatendatei 'datei' konnte nicht geöffnet werden – 'beschreibung'  
  
 Die mit der Compileroption [\/reference](../Topic/-reference%20\(C%23%20Compiler%20Options\).md) angegebene Datei enthält keine gültigen Metadaten.
