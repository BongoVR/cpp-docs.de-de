# Compilerfehler CS0587
Interner Compilerfehler: Stufe 'STAGE'  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil unerwartete Syntax nicht analysiert werden kann. Wenn dies nicht der Fall ist, versuchen Sie Folgendes: [Sprechen Sie mit uns](../Topic/Talk%20to%20Us.md).
