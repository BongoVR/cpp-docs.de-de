# Compilerfehler CS1007
Der Eigenschaftenaccessor ist bereits definiert.  
  
 Wenn Sie eine [Eigenschaft](../Topic/Using%20Properties%20\(C%23%20Programming%20Guide\).md) deklarieren, m√ºssen Sie auch ihre Accessormethoden deklarieren. Eine Eigenschaft kann jedoch nicht mehr als eine `get`\-Accessormethode oder nicht mehr als eine `set`\-Accessormethode aufweisen.  
  
## Beispiel  
 Im folgenden Beispiel wird CS1007 generiert:  
  
```  
// CS1007.cs public class clx { public int MyProperty { get { return 0; } get   // CS1007, this is the second get method { return 0; } } public static void Main() {} }  
```
