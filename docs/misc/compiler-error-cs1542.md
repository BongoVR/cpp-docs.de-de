# Compilerfehler CS1542
„dll“ kann nicht zu dieser Assembly hinzugefügt werden, da es sich bereits um eine Assembly handelt. Verwenden Sie stattdessen die Option \/R.  
  
 Die Datei, auf die mit der Compileroption [\/addmodule](../Topic/-addmodule%20\(C%23%20Compiler%20Options\).md) verwiesen wurde, wurde nicht mit [\/target:module](../Topic/-target:module%20\(C%23%20Compiler%20Options\).md) erstellt. Verwenden Sie [\/reference](../Topic/-reference%20\(C%23%20Compiler%20Options\).md), um auf die Datei in dieser Kompilierung zu verweisen.
