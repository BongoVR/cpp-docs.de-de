# Compilerfehler CS1555
"Klasse", angegeben für die Main\-Methode, konnte nicht gefunden werden.  
  
 Für die [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md)\-Compileroption wurde eine Klasse angegeben, aber der Name der Klasse wurde nicht im Quellcode gefunden.
