# Compilerfehler CS1556
"Konstrukt", angegeben für die Main\-Methode, muss eine gültige Klasse oder Struktur sein.  
  
 An die [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md)\-Compileroption wurde ein Bezeichner übergeben, der kein Klassenname war.
