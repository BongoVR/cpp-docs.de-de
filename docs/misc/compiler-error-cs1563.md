# Compilerfehler CS1563
Die Ausgabe "Ausgabedatei" hat keine Quelldateien.  
  
 Die Compileroption [\/out](../Topic/-out%20\(C%23%20Compiler%20Options\).md) wurde angegeben, ohne dass danach Quellcodedateien folgen. Sie sollten **\/out** entweder nicht verwenden oder nach **\/out** die Quellcodedateien angeben.
