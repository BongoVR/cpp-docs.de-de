# Compilerfehler CS1565
Die angegebenen Optionen führen zu einem Konflikt: Win32\-Ressourcendatei; Win32\-Symbol  
  
 Es ist nicht zulässig, die beiden Compileroptionen [\/win32res](../Topic/-win32res%20\(C%23%20Compiler%20Options\).md) und [\/win32ico](../Topic/-win32icon%20\(C%23%20Compiler%20Options\).md) für dieselbe Kompilierung anzugeben.
