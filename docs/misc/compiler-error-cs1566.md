# Compilerfehler CS1566
Fehler beim Lesen der Ressourcendatei 'datei' – 'ursache'  
  
 Der Compiler hatte ein Problem mit dem an die Compileroption [\/resource](../Topic/-resource%20\(C%23%20Compiler%20Options\).md) übergebenen Dateinamen.
