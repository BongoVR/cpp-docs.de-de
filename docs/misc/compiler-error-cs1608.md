# Compilerfehler CS1608
Das Required\-Attribut ist in C\#\-Typen unzulässig.  
  
 <xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> ist für in C\#\-Code definierte Typen unzulässig.
