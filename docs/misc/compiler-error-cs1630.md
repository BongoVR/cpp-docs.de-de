# Compilerfehler CS1630
Ungültige Option "Option" für \/errorreport. Muss "prompt", "send", "queue" oder "none" sein.  
  
 Auf die Befehlszeilenoption [\/errorreport](../Topic/-errorreport%20\(C%23%20Compiler%20Options\).md) muss **prompt**, **send**, **queue** oder **none** folgen. Dadurch wird angeben, welche Aktion ergriffen sollen, wenn ein interner Compilerfehler auftritt.
