# Compilerfehler CS2006
Befehlszeilen\-Syntaxfehler: Fehlender 'Text' für die Option 'Option'.  
  
 Die Syntax für *Option* erfordert zusätzlichen Text. Informationen finden Sie unter [Compileroptionen](../Topic/C%23%20Compiler%20Options.md).
