# Compilerfehler CS2012
"Datei" kann nicht zum Schreiben geöffnet werden  
  
 Beim Verwenden der Compileroption [\/bugreport:](../Topic/-bugreport%20\(C%23%20Compiler%20Options\).md)`file` konnte die Datei nicht zum Schreiben geöffnet werden. Vergewissern Sie sich, dass Sie einen gültigen Dateinamen angegeben haben und die Datei nicht schreibgeschützt ist.
