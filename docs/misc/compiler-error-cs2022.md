# Compilerfehler CS2022
Die Optionen \/out und \/target müssen vor den Quelldateinamen stehen.  
  
 Die Compileroptionen [\/out \(Ausgabedateinamen festlegen\)](../Topic/-out%20\(C%23%20Compiler%20Options\).md) und [\/target \(Ausgabedateiformat festlegen\)](../Topic/-target%20\(C%23%20Compiler%20Options\).md) müssen, wenn sie in der Befehlszeile angegeben werden, den Quellcodedateien vorangestellt sein.
