# Compilerwarnung (Stufe 1) CS2014
Die Compileroption "alte Option" ist veraltet, verwenden Sie stattdessen "neue Option".  
  
 Die Form der Compileroption ist veraltet. Weitere Informationen zu Compileroptionen finden Sie unter [C\#\-Compileroptionen](../Topic/C%23%20Compiler%20Options.md).
