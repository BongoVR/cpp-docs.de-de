# Compilerwarnung (Stufe 1) CS2023
Die \/noconfig\-Option wird ignoriert, da sie in einer Antwortdatei angegeben wurde.  
  
 Die Compileroption [\/noconfig](../Topic/-noconfig%20\(C%23%20Compiler%20Options\).md) wurde in einer Antwortdatei angegeben, was nicht zulässig ist.  
  
 Diese Warnung kann nicht durch die [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md)\-Option unterdrückt werden.
