# Compilerwarnung (Stufe 1) CS2029
Ungültiger Wert für '\/define'; 'Bezeichner' ist kein gültiger Bezeichner.  
  
 Diese Warnung tritt auf, wenn der in der [\/define](../Topic/-define%20\(C%23%20Compiler%20Options\).md)\-Option verwendete Wert ungültige Zeichen aufweist.  
  
 Diese Warnung kann nicht durch die [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md)\-Option unterdrückt werden.
