# Compilerwarnung (Stufe 4) CS0422
Die Option '\/incremental' wird nicht mehr unterstützt.  
  
 Inkrementelle Kompilierung \(**\/incr** oder **\/incremental**\) wird in Visual C\# 2005 nicht unterstützt.
