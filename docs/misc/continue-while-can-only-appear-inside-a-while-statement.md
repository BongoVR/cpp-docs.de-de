# &#39;Continue While&#39; darf nur innerhalb einer While-Anweisung verwendet werden.
Eine `Continue While`\-Anweisung kann nur innerhalb einer `For...Next`\-Schleife verwendet werden.  
  
 **Fehler\-ID:** BC30784  
  
### So beheben Sie diesen Fehler  
  
1.  Wenn sich die `Continue While`\-Anweisung in einer `Do...Loop`\-Schleife befindet, m체ssen Sie die Anweisung in `Continue Do` 채ndern.  
  
2.  Wenn sich die `Continue While`\-Anweisung in einer `For...Next`\-Schleife befindet, m체ssen Sie die Anweisung in `Continue For` 채ndern.  
  
3.  Entfernen Sie andernfalls die `Continue While`\-Anweisung.
