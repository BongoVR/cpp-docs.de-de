# Custom Tool Error
Ein benutzerdefiniertes Tool hat einen Fehler oder eine Warnung generiert.  Benutzerdefinierte Tools sind Komponenten, mit denen Dateien zur Entwurfszeit von einem Typ in einen anderen umgewandelt werden können.  Ein benutzerdefiniertes Tool kann z. B. ein Datasetcodegenerator sein, der in einer XML\-Schemadatei \(XSD\-Datei\) liest und Klassen in einer Codedatei generiert, die Tabellen und Spalten programmgesteuert offen legt.  
  
 **So beheben Sie diesen Fehler**  
  
-   Entnehmen Sie die erforderlichen Informationen der Dokumentation, die vom Anbieter der benutzerdefinierten Tools bereitgestellt wird.
