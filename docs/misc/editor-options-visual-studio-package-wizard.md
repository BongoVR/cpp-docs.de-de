# Editor-Optionen, Assistent f&#252;r das Visual Studio-Paket
In diesem Schritt können Sie die dem Editor zugeordnete Dateierweiterung definieren, den Dateinamen, der als Standardname für mit dem Editor erstellte Dokumente verwendet werden soll, festlegen und das Symbol angeben, das dem Editor und dem Dateityp zugeordnet ist.  
  
 **Editorname**  
 Geben Sie den Namen des neuen Editors ein. Dieser Name wird im Dialogfeld **Öffnen mit** für den Editor verwendet.  
  
 **Dateierweiterung**  
 Definieren Sie die Dateierweiterung, die dem Editor zugeordnet ist. Sie können mehrere Erweiterungen angeben, indem Sie eine durch Komma getrennte Liste erstellen. Der Editor ist der [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]\-Standardeditor für Dateien mit der definierten Dateierweiterung.  
  
 **Standarddateiname**  
 Geben Sie den Namen ein, der als Basis für den Standardnamen für die vom Editor erstellten Dokumente verwendet werden soll.  
  
 **Symbol ändern**  
 Klicken Sie, um das Dialogfeld **Symbol ändern** zu öffnen. In diesem Dialogfeld können Sie ein Symbol für den Editor auswählen oder zum Speicherort einer anderen Datei navigieren. Bei der Datei muss es sich um eine ICO\-Datei handeln. Das ausgewählte Symbol wird im Editorfenster angezeigt und dem vom Editor generierten Dateien zugeordnet.  
  
 **Abbrechen**  
 Bricht den Vorgang ab, und beendet den Assistenten.  
  
 **Hinten**  
 Wechselt zur vorherigen Seite des Assistenten.  
  
 **Fertig stellen**  
 Erstellt das VSPackage.
