# Ausdrucksende erwartet
Am Ende eines ansonsten vollständigen Ausdrucks wurden zusätzliche Zeichen gefunden.  
  
 **Fehler\-ID:** BC30710  
  
### So beheben Sie diesen Fehler  
  
-   Überprüfen Sie die Syntax des Objekts oder der Anweisung.
