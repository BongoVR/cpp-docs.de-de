# Error getting Codezone Community sites.
Dieser Fehler tritt auf, wenn keine Verbindung zum Onlinewebdienst gefunden wird.  Die Webdienste liefern Informationen für die Seite **Online**, die im Dialogfeld **Optionen** über **Umgebung**, **Hilfe** aufgerufen wird.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass Sie über eine aktive Internetverbindung verfügen.  
  
     \- oder \-  
  
     Wenn die Internetverbindung über einen Proxyserver hergestellt wird, überprüfen Sie die Einstellungen für den Proxyserver.  
  
2.  Versuchen Sie erneut, auf die Onlineseite zuzugreifen.
