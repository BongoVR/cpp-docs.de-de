# Error getting information on languages supported by online search providers.
Dieser Fehler tritt im Allgemeinen auf, wenn Sie versuchen, eine Sprache in der Liste **Onlinethemensprache** unter [Help, Environment, Options Dialog Box](assetId:///a2a5dff5-70a5-4f14-bf4c-66793894de6a) auszuwählen, jedoch keine Onlinedienste verfügbar sind.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass Sie über eine aktive Internetverbindung verfügen.  
  
2.  Vergewissern Sie sich bei Verwendung eines Proxyservers, dass der Proxyserver ordnungsgemäß konfiguriert ist.  
  
3.  Wählen Sie die Dropdownliste **Onlinethemensprache** erneut aus.
