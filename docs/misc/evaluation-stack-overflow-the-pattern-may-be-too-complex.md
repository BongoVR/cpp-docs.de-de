# Evaluation stack overflow. The pattern may be too complex.
Dieser Fehler tritt normalerweise auf, wenn in einer Suchzeichenfolge ein komplexer regulärer Ausdruck verwendet wurde, den die integrierte Entwicklungsumgebung \(IDE\) nicht verarbeiten kann.  
  
### So beheben Sie diesen Fehler  
  
1.  Reduzieren Sie die Anzahl der Einschlüsse und Klauseln, und führen Sie den Vorgang erneut aus.
