# Exit-Anweisungen sind im Direktfenster nicht g&#252;ltig
`Exit`\-Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler\-ID:** BC30713  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Exit`\-Anweisungen aus dem Debugcode.
