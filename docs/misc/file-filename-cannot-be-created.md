# File &lt;filename&gt; cannot be created.
Dieser Fehler tritt normalerweise auf, wenn versucht wird, einen Webbericht über Codekommentare zu erstellen, der Benutzer jedoch keinen Schreibzugriff hat.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie, ob Sie Schreibzugriff haben.  
  
2.  Überprüfen Sie, ob genügend Platz zum Speichern der Seiten zur Verfügung steht.
