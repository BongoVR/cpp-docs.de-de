# File is currently in use and cannot be overwritten.
Die aktuelle Datei kann nicht gespeichert werden.  Im Allgemeinen tritt dieser Fehler auf, wenn Sie eine Datei unter dem Namen einer Datei speichern möchten, die derzeit ebenfalls geöffnet ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie die Datei mit demselben Namen, und versuchen Sie es erneut.  
  
     – oder –  
  
     Benennen Sie die Datei um, und speichern Sie sie.
