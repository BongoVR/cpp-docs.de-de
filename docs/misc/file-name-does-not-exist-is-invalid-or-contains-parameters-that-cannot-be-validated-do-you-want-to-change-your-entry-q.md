# Der Dateiname ist nicht vorhanden, ist ung&#252;ltig oder enth&#228;lt Parameter, die nicht &#252;berpr&#252;ft werden k&#246;nnen. M&#246;chten Sie Ihre Eingabe &#228;ndern?
Im Allgemeinen tritt dieser Fehler auf, wenn ein falscher Pfad oder Dateiname in das Dialogfeld **Programm hinzufügen** eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Wählen Sie **Durchsuchen**, um zur richtigen Datei zu navigieren.  
  
     – oder –  
  
2.  Geben Sie den Pfad und den Dateinamen erneut ein.
