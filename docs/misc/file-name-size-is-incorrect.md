# File &lt;name&gt; size is incorrect.
Diese Datei enthält weniger Bytes als erwartet. Die Datei ist vermutlich abgeschnitten oder beschädigt.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie Visual Studio, und starten Sie es neu.  
  
2.  Wenn der Fehler weiterhin auftritt, installieren Sie Visual Studio erneut, um die Datei zu reparieren.
