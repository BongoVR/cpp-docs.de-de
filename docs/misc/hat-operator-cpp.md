# ^-Operator (C++)
Der Operator `^` hat in Visual C\+\+ je nach Kontext zwei Bedeutungen.  Weitere Informationen finden Sie unter folgenden Themen:  
  
-   [Bitweiser exklusiver OR\-Operator: ^](../cpp/bitwise-exclusive-or-operator-hat.md), für die Nutzung in allen Arten von C\+\+.  
  
-   [Handle für Objekt \(^\)](../windows/handle-to-object-operator-hat-cpp-component-extensions.md), für die Nutzung in [!INCLUDE[cppwrt_short](../build/reference/includes/cppwrt_short_md.md)] und [!INCLUDE[cppcli](../build/reference/includes/cppcli_md.md)]
