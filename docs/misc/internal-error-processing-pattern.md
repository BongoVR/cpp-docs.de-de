# Internal error processing pattern.
Dieser Fehler tritt normalerweise auf, wenn ein regulärer Ausdruck zu einer Suchzeichenfolge hinzugefügt wurde, die Syntax für den regulären Ausdruck jedoch Fehler enthält.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie die Suchzeichenfolge erneut unter Verwendung der korrekten Syntax für reguläre Ausdrücke ein.  Weitere Informationen finden Sie unter [Reguläre Ausdrücke](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md).
