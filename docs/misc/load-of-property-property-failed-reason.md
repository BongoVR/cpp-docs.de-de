# Load of property &#39;property&#39; failed. &lt;reason&gt;
Ein Wert für eine Eigenschaft befindet sich außerhalb des gültigen Bereichs.  
  
 Dieser Fehler wird meistens durch das manuelle Bearbeiten der Projektdatei verursacht.  
  
 **So beheben Sie diesen Fehler**  
  
-   Bearbeiten Sie die Projektdatei erneut, und beheben Sie den Fehler manuell.  
  
     Für die betroffenen Eigenschaften wird der Standardwert festgelegt, d. h. der Wert, den die Eigenschaft hatte, als das Projekt erstellt wurde.
