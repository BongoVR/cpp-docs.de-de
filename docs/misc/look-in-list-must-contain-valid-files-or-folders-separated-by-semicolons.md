# Look In list must contain valid files or folders, separated by semicolons.
Dieser Fehler tritt normalerweise auf, wenn das Verzeichnis oder die Dateien im Listenfeld **Suchen in** ungültige Zeichen enthalten.  
  
### So beheben Sie diesen Fehler  
  
1.  Verwenden Sie das Dialogfeld **Suchen in**, um das Verzeichnis oder die Datei, das bzw. die durchsucht werden soll, mithilfe von **Durchsuchen** anzuzeigen, ohne dass der Pfad\- oder Dateiname eingegeben werden muss.  Zum Öffnen des Dialogfelds **Suchen in** klicken Sie neben dem Listenfeld **Suchen in** auf die Schaltfläche mit dem Auslassungszeichen.
