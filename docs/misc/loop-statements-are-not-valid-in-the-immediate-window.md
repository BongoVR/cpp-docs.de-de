# Schleifenanweisungen sind im Direktfenster nicht g&#252;ltig.
`Loop`\-Anweisungen können im **Direktfenster** nicht verwendet werden.  
  
 **Fehler\-ID:** BC30708  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Loop`\-Anweisungen aus dem Debugcode.
