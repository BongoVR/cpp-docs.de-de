# &quot;&lt;LSet-Anweisung&gt;&quot; ist nicht deklariert
"\<LSet\-Anweisung\>" ist nicht deklariert. LSet\-Anweisungen werden nicht mehr unterstützt. Verwenden Sie stattdessen Microsoft.VisualBasic.LSet.  
  
 Einige systeminterne Funktionen von [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] in früheren Versionen wurden in den <xref:Microsoft.VisualBasic?displayProperty=fullName>\-Namespace verschoben. Dadurch wird ihre Funktionalität allgemeiner verfügbar für alle Programmiersprachen.  
  
 **Fehler\-ID:** BC30820  
  
### So beheben Sie diesen Fehler  
  
-   Verwenden Sie stattdessen die `LSet`\-Funktion im `Microsoft.VisualBasic`\-Namespace.
