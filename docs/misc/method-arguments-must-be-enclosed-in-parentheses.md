# Methodenargumente m&#252;ssen in Klammern stehen.
Die Regeln, die Prozeduraufrufe steuern, sind in neueren Versionen von [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] einfacher. Alle Argumente müssen in Klammern stehen.  
  
 **Fehler\-ID:** BC30800  
  
### So beheben Sie diesen Fehler  
  
-   Schließen Sie die Argumentliste in Klammern ein. Beispiel:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
