# Eine Methodendeklarationsanweisung muss die erste Anweisung in einer logischen Zeile sein
Einem Doppelpunkt \(:\) als Trennzeichen folgt eine `Function`\-Anweisung oder `Sub`\-Anweisung.`Function` oder `Sub` muss die einzige Anweisung auf der jeweiligen Quellzeile sein.  
  
 **Fehler\-ID:** BC32009  
  
### So beheben Sie diesen Fehler  
  
-   Setzen Sie jede Anweisung in eine eigene Zeile.
