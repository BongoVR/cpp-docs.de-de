# MSBuild Error MSB1009
**Die Projektdatei existiert nicht.**  
  
 Diese Projektdatei ist im angegebenen Verzeichnis nicht vorhanden oder, wenn kein Verzeichnis angegeben wurde, ist die Datei im aktuellen Arbeitsverzeichnis nicht vorhanden.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die Schreibweise des Dateinamens und \-pfads.
