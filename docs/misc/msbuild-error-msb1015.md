# MSBuild Error MSB1015
**MSBuild kann auf dieser Version des Betriebssystems nicht ausgeführt werden.  Nur Windows 2000, Windows XP und Folgeversionen werden unterstützt.**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] wird unter dieser Betriebssystemversion nicht unterstützt.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren und führen Sie [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] unter einer der unterstützten Betriebssystemversionen aus.
