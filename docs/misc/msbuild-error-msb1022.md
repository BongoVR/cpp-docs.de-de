# MSBuild Error MSB1022
**Die Antwortdatei existiert nicht.**  
  
 Die Antwortdatei ist am angegebenen Speicherort nicht vorhanden.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie, ob die Antwortdatei im angegebenen Verzeichnis vorhanden ist, falls Sie den absoluten oder relativen Speicherort der Antwortdatei angegeben haben.
