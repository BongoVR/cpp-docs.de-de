# MSBuild Error MSB1026
**Die Schemadatei existiert nicht.**  
  
 Die Schemadatei ist am angegebenen Speicherort nicht vorhanden.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie, ob die Schemadatei im angegebenen Verzeichnis vorhanden ist, falls Sie den absoluten oder relativen Speicherort der Schemadatei angegeben haben.
