# MSBuild Error MSB2007
**Visual Studio findet die Projektdatei "\<Datei\>" nicht.**  
  
 Die angegebene Projektdatei konnte von [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] nicht gefunden werden.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die Schreibweise des Projektdateinamens.
