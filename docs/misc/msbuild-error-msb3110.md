# MSBuild Error MSB3110
**MSB3110: Konflikt bei Identität \<Assemblyidentität\> der Assembly \<Assembly\>. Erwarteter Dateiname: \<Datei\>.**  
  
 Dieser Fehler wird generiert, wenn der aus der Assemblyidentität abgerufene Name der Assemblydatei nicht mit dem tatsächlichen Dateinamen übereinstimmt.  
  
### So beheben Sie diesen Fehler  
  
-   Korrigieren Sie den Namen der Assemblydatei, sodass er der Assemblyidentität entspricht.
