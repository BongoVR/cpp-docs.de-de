# MSBuild Error MSB3111
**MSB3111: Für die Verwendung der BindingRedirect\-Einträge in "app.config" ist volle Vertrauenswürdigkeit erforderlich.**  
  
 Diese Warnung wird generiert, wenn beim Buildprozess erkannt wird, dass die Anwendung einige der zugehörigen an eine andere Version in der Datei "app.config" umgeleitet hat.  Dies funktioniert bei teilweise vertrauenswürdigen Anwendungen nicht.
