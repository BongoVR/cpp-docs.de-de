# MSBuild Error MSB3166
**MSB3166: Die erforderliche Datei \<Datei\> konnte für \<Pakte\> nicht gefunden werden.**  
  
 Dieser Fehler wird generiert, wenn ein Endbenutzer\-Lizenzvertrag \(EULA\) oder eine externe Prüfdatei fehlt.  Vergewissern Sie sich, dass die Datei auf dem Datenträger vorhanden ist.  
  
### So beheben Sie diesen Fehler  
  
-   Installieren Sie das Visual Studio SDK neu.  
  
     \- oder \-  
  
-   Kopieren Sie die erforderlichen Dateien an den entsprechenden Speicherort.
