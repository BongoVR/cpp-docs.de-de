# MSBuild Error MSB3173
**MSB3172: Das Manifest \<Datei\> kann nicht geschrieben werden. '  \<Fehler\>'**  
  
 Dieser Fehler wird generiert, wenn beim Buildprozess ein unspezifisches Problem beim Schreiben einer Manifestdatei auftritt.  
  
### So beheben Sie diesen Fehler  
  
-   Weitere Informationen zur Ursache und zu möglichen Lösungen eines MSBuild\-Fehlers finden Sie unter [Additional Resources](../Topic/Additional%20MSBuild%20Resources.md).
