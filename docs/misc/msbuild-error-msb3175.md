# MSBuild Error MSB3175
**MSB3175: Ungültiger Wert für \<Datei\> von Element \<Buildaufgabe\>.**  
  
 Diese Warnung wird generiert, wenn beim Buildprozess der Wert verschiedener enumerationsbasierter Aufgabenelementeigenschaften wie `AssemblyType`, `DependencyType`, `FileType` oder `TargetZone` nicht erkannt wird.  Dies bezieht sich sowohl auf Anwendungs\- als auch auf Bereitstellungsmanifeste.
