# MSBuild Error MSB3178
**MSB3178: Die \<Assembly\>\-Assembly ist fälschlicherweise als Datei angegeben.**  
  
 Diese Warnung wird generiert, wenn bei der Generierung des Anwendungsmanifests erkannt wird, dass es sich bei einem Dateiverweis um eine \(verwaltete oder systemeigene\) Assembly handelt.  
  
### So beheben Sie diesen Fehler  
  
-   Weitere Informationen zur Ursache und zu möglichen Lösungen eines MSBuild\-Fehlers finden Sie unter [Additional Resources](../Topic/Additional%20MSBuild%20Resources.md).
