# MSBuild-Fehler MSB4136
**MSB4136: Fehler beim Lesen der Konfigurationsinformationen.**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] hat bei dem Versuch, die Toolsetinformationen in der [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]\-Konfigurationsdatei \(msbuild.exe.config\) zu lesen, einen Fehler erhalten.  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass die Konfigurationsdatei richtig und wohlgeformt ist.  Wenn Sie die .config\-Datei beispielsweise durch das Hinzufügen eines Toolsets angepasst haben, überprüfen Sie die Toolsetdefinition.
