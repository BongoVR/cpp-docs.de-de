# MSBuild-Fehler MSB4138
**MSB4138: Am Registrierungsspeicherort "'\<Schlüssel\>'" wurden Daten angegeben, die keine Zeichenfolge sind.**  
  
 Der in einer benutzerdefinierten Toolsetkonfiguration angegebene Registrierungsspeicherort ist ungültig.  
  
### So beheben Sie diesen Fehler  
  
-   Stellen Sie sicher, dass die Toolsetdefinition richtig ist und dass der angegebene Registrierungsschlüssel vorhanden ist.
