# MSBuild-Fehler MSB8011
Fehler beim Registrieren der Ausgabe. Versuchen Sie, die Umleitung pro Benutzer zu aktivieren, oder registrieren Sie die Komponente an einer Eingabeaufforderung mit erweiterten Berechtigungen.
