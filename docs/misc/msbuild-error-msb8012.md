# MSBuild-Fehler MSB8012
*{0}\(\\{1\\}\)* entspricht nicht dem OutputFile\-Eigenschaftswert \(*\\{3\\}*\) von *\\{2\\}*. Das Projekt wird daher möglicherweise nicht ordnungsgemäß erstellt. Damit dieses Problem behoben werden kann, müssen die Eigenschaftswerte für $\(OutDir\), $\(TargetName\) und $\(TargetExt\) dem in %\(\\{4\\}.OutputFile\) angegebenen Wert entsprechen.
