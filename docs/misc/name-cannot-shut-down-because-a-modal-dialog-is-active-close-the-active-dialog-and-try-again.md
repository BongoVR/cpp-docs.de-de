# &lt;name&gt; cannot shut down because a modal dialog is active. Close the active dialog and try again.
Im Allgemeinen tritt dieser Fehler auf, wenn das Betriebssystem heruntergefahren wird und in einer Anwendung noch ein modales Dialogfeld geöffnet ist, wodurch das Herunterfahren verhindert wird.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie zunächst das modale Dialogfeld und anschließend die Anwendung, oder fahren Sie das Betriebssystem herunter.
