# &quot;&lt;Name&gt;&quot; ist in den Anwendungsobjekten &quot;&lt;Liste&gt;&quot; nicht eindeutig.
Sie haben einen mehrdeutigen Namen angegeben, der einen Konflikt mit einem anderen Namen verursacht. Der [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)]\-Compiler enthält keine Konfliktlösungsregeln. Sie müssen die Mehrdeutigkeit der Namen selbst aufheben. Die in der Fehlermeldung angegebene Liste enthält die Anwendungsobjekte, deren Name mehrdeutig ist.  
  
 **Fehler\-ID:** BC30563  
  
### So beheben Sie diesen Fehler  
  
-   Kennzeichnen Sie den Namen vollständig.
