# &lt;name&gt; is not valid editor for this file type.
Im Allgemeinen tritt dieser Fehler auf, wenn im Befehlsfenster oder im Feld Suchen ein falscher Wert für das Argument \/e:*editorname* eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Schlagen Sie die richtige Syntax für \/e:*editorname* in der Dokumentation nach, und geben Sie den Befehl erneut ein.
