# Der Namespace oder Typ &#39;&lt;name&gt;&#39; wurde bereits importiert.
Es wurde versucht, einen Namespace mehrmals zu importieren.  
  
 **Fehler\-ID:** BC31051  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie redundante `Imports`\-Anweisungen.
