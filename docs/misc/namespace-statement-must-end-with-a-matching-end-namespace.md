# „Namespace“ muss mit einem entsprechenden „End Namespace“ abgeschlossen werden.
`Namespace`\-Blöcke müssen mit einer `End Namespace`\-Anweisung enden.  
  
 **Fehler\-ID:** BC30626  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie eine `End Namespace`\-Anweisung ans Ende des `Namespace`\-Blocks hinzu.
