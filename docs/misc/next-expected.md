# &quot;Next&quot; erwartet
Ein `On Error Resume`\-Konstrukt tritt ohne abschließendes `Next` auf.  
  
 **Fehler\-ID:** BC30003  
  
### So beheben Sie diesen Fehler  
  
-   Fügen Sie das `Next`\-Schlüsselwort am Ende der `On Error`\-Anweisung hinzu.
