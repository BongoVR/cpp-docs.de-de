# No suitable target for find was specified or another find error occurred.
Dieser Fehler tritt normalerweise auf, wenn eine Suche mit dem Gültigkeitsbereich **Aktuelles Dokument** gestartet wurde, jedoch keine Datei geöffnet ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Öffnen Sie eine Datei, und führen Sie die Suche erneut durch, oder ändern Sie den Gültigkeitsbereich für die Suche.
