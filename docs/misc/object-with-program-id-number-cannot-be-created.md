# Object with program ID &lt;number&gt; cannot be created.
Die von Ihnen verwendete Zeichenfolge für die Klasse ist ungültig.  Hierbei handelt es sich um einen Fehler auf Betriebssystem\-Ebene.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie Visual Studio oder die betroffene Anwendung neu, und versuchen Sie es erneut.
