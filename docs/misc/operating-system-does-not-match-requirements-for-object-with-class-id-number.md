# Operating system does not match requirements for object with class ID &lt;number&gt;.
Im Allgemeinen tritt dieser Fehler auf, wenn ein nicht kompatibles Betriebssystem bzw. eine nicht kompatible Version des Betriebssystems erkannt wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Überprüfen Sie die Systemanforderungen für die Anwendung.  
  
2.  Installieren Sie ein kompatibles Betriebssystem, und versuchen Sie es erneut.
