# Operator, der Funktionszeiger ist
Dieser Inhalt wurde entfernt. Weitere Informationen finden Sie unter [Überladen des Funktionsaufrufoperators](../cpp/function-call-cpp.md).  
  
```  
// operator_that_is_pointer_to_function.cpp  
// function style call on object will invoke user-defined conversion   
// if there is one. See secion 13.3.1.1.2   
typedef void(*ptf)();  
void func()  
{  
}  
struct S  
{  
   operator ptf()  
   {  
      return func;  
   }  
};  
  
int main()  
{  
   S s;  
   s();//operates as s.operator ptf()()  
}  
```
