# Package &lt;name&gt; failed to load.
Das Paket konnte nicht geöffnet werden.  Im Allgemeinen tritt dieser Fehler auf, wenn das Paket nicht richtig registriert wurde, eine DLL für das Paket fehlt bzw. beschädigt ist oder eine für das Paket erforderliche DLL einer anderen Komponente fehlt.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie das Produkt neu, das das Paket bereitstellt.
