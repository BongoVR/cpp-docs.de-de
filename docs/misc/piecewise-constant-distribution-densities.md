# piecewise_constant_distribution::densities
Gibt die gespeicherten Dichten für die Verteilung zurück.  
  
## Syntax  
  
```  
std::vector<double> densities() const;  
```  
  
## Hinweise  
 Die Memberfunktion gibt die gespeicherten Dichten zurück, die anhand der Gewichtsparameter berechnet werden.  
  
## Anforderungen  
 **Header:** \<random\>  
  
 **Namespace:** std
