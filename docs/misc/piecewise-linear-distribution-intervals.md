# piecewise_linear_distribution::intervals
Gibt die gespeicherten Intervalle für die Verteilung zurück.  
  
## Syntax  
  
```  
std::vector<RealType> intervals() const;  
```  
  
## Hinweise  
 Die Memberfunktion gibt die gespeicherten Intervallwerte zurück.
