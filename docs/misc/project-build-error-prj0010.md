# Projektbuildfehler PRJ0010
Die erforderliche Abhängigkeit konnte nicht gefunden werden: Datei  
  
 Eine Datei verfügt über eine angegebene Abhängigkeit, aber die abhängige Datei wurde nicht gefunden.
