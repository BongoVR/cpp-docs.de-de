# Projektbuildfehler PRJ0043
Die Outputs\-Eigenschaft für die der Datei „file“ zugewiesen benutzerdefinierte Buildregel „rule“ ist ungültig. Die Eigenschaft enthält „string“, die zu „value“ ausgewertet wird.  
  
 Die Outputs\-Eigenschaft wurde zu einer leeren Zeichenfolge ausgewertet. Benutzerdefinierte Buildregeln benötigen die Ausgabe des Buildvorgangs.
