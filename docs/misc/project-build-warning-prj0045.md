# Projektbuildwarnung PRJ0045
Die Outputs\-Eigenschaft für die der Datei 'file' zugewiesen benutzerdefinierte Buildregel 'rule' ist nicht festgelegt. Die Ausführung der Regel wird übersprungen.  
  
 Die Outputs\-Eigenschaft ist für eine benutzerdefinierte Buildregel nicht definiert. Benutzerdefinierte Buildregeln benötigen die Ausgabe des Buildvorgangs.
