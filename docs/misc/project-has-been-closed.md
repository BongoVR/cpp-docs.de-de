# Das Projekt wurde geschlossen.
Innerhalb des [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)]\-Compilers ist beim Zugriff auf das Codemodell ein Fehler aufgetreten.  
  
 **Fehler\-ID:** BC32301  
  
### So beheben Sie diesen Fehler  
  
1.  Kompilieren Sie das Programm erneut, um festzustellen, ob der Fehler erneut auftritt.  
  
2.  Wenn der Fehler erneut auftritt, installieren Sie den [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)]\-Compiler neu.  
  
3.  Wenn der Fehler nach der Neuinstallation weiterhin besteht, tragen Sie Informationen zu den Umst√§nden zusammen, und benachrichtigen Sie den Produktsupport von Microsoft.
