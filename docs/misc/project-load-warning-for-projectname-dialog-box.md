# Projektladewarnung f&#252;r &lt;Projektname&gt; (Dialogfeld)
Dieses Dialogfeld wird angezeigt, wenn die Einstellungen für in der IDE geöffnete Projekte nicht richtig geladen werden.  
  
 **Angegebene Fehler korrigieren und Projekt neu laden**  
 Zeigt Informationen zu Problemen wie einem ungültigen Pfad oder Dateinamen an.  Um Fehler zu beheben, entfernen oder ersetzen Sie alle ungültigen Verweise aus den Eigenschaftenseiten des Projekts.  Speichern Sie das Projekt, und laden Sie es dann in der IDE erneut.  
  
 **Dieses Dialogfeld in dieser Sitzung nicht mehr anzeigen**  
 Bei Aktivierung wird das Dialogfeld mit der Ladewarnung für \<Projektname\> erst beim nächsten Öffnen der IDE wieder angezeigt.
