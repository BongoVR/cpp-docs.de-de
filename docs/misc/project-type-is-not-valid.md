# Project type is not valid.
Im Allgemeinen tritt dieser Fehler auf, wenn ein falscher Wert für das *templatename*\-Argument für den Befehl **Neues Projekt hinzufügen** eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Schlagen Sie die korrekte Syntax für den Befehl **Neues Projekt hinzufügen** in der Dokumentation nach, und versuchen Sie es erneut.
