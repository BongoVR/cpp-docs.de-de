# Eigenschaften in einem Modul k&#246;nnen nicht als &lt;Spezifizierer&gt; deklariert werden.
Sie haben in einer Eigenschaft innerhalb eines Moduls versucht, einen Spezifizierer, z.Â B. `MustOverride`, `NotOverridable` oder `Overridable`, zu verwenden.
