# Ressourcencompiler: Fehler RC2018
Unbekanntes Zeichen 'hexzahl'  
  
 Das der angegebenen Hexadezimalzahl entsprechende ASCII\-Zeichen kommt in der Quelldatei vor, ist jedoch unzulässig.  
  
 Dieser Fehler kann eine Beschädigung der Quelldatei zur Ursache haben.
