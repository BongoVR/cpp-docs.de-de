# Ressourcencompiler: Fehler RC2019
Präprozessordirektive erwartet, 'Zeichen' gefunden.  
  
 Auf das angegebene Zeichen folgt ein Nummernzeichen \(**\#**\), aber es war nicht der erste Buchstabe einer Präprozessordirektive.
