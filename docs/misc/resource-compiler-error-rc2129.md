# Ressourcencompiler: Fehler RC2129
BEGIN in VERSIONINFO\-Ressource erwartet  
  
 Das **BEGIN**\-Schlüsselwort muss direkt auf das **VERSIONINFO**\-Schlüsselwort folgen.
