# Ressourcencompiler: Fehler RC2132
VALUE\-, BLOCK\- oder END\-Schlüsselwort erwartet.  
  
 Ein Block in einer **VERSION**\-Ressource endet nicht ordnungsgemäß, oder ein neuer Block beginnt nicht ordnungsgemäß. Dies führt zu einer **VERSION**\-Anweisung, die keinen gültigen Block darstellt.
