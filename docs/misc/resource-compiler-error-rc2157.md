# Ressourcencompiler: Fehler RC2157
Komma in Zugriffstastentabelle erwartet  
  
 Der Ressourcencompiler erfordert ein Komma zwischen dem Feld `event` und dem Feld *idvalue* in der **ACCELERATORS**\-Anweisung.
