# Ressourcencompiler: Schwerwiegender Fehler RC1049
Ungültiges numerisches Argument "Zeichenfolge"  
  
 Der Ressourcencompiler hat ein numerisches Argument erwartet, aber die angegebene Zeichenfolge empfangen.  
  
 Dieser Fehler kann durch die Angabe einer Hexadezimalzahl ohne das erforderliche \\x\-Präfix oder durch eine fehlerhaft formatierte Gleitkommazahl verursacht werden.
