# Ressourcencompiler: Schwerwiegender Fehler RC1122
E\/A\-Fehler beim Schreiben der Datei.  
  
 Der Ressourcencompiler konnte nicht in eine Datei schreiben.  
  
### Dieser Fehler kann eine der folgenden Ursachen haben:  
  
1.  Es ist nicht genügend Speicherplatz verfügbar. Der freie Speicherplatz muss mehr als doppelt so groß sein wie die ausführbare Datei, die Sie erstellen.  
  
2.  Das Volume ist schreibgeschützt.  
  
3.  Es liegt ein beschädigter Sektor vor.  
  
4.  Freigabeverletzung.
