# Ressourcencompiler: Schwerwiegender Fehler RC1202
Ungültiger Schalter – fehlende Standardsprach\-ID nach \/l  
  
 Auf die Option zur Angabe der Standardsprache \(\/I\) folgte keine Angabe einer hexadezimalen Sprachen\-ID.
