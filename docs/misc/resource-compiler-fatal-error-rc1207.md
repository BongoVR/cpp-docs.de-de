# Ressourcencompiler: Schwerwiegender Fehler RC1207
Standardcodepage ist ungültig  
  
 Die durch die Umgebungsvariable **RCCODEPAGE** angegebene Codepage ist ungültig. Weitere Informationen hierzu finden Sie unter [Codepages](../c-runtime-library/code-pages.md) in der *Laufzeitbibliotheksreferenz*.
