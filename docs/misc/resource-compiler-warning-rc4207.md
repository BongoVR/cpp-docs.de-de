# Ressourcencompiler: Warnung RC4207
Typzeichenfolge zu lang – auf 256 Zeichen gekürzt  
  
 Eine Typzeichenfolge hat den Grenzwert von 256 Zeichen überschritten. Sie wurde auf 256 Zeichen gekürzt.
