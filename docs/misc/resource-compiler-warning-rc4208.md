# Ressourcencompiler: Warnung RC4208
Titelzeichenfolge zu lang – auf 256 Zeichen gekürzt  
  
 Eine Titelzeichenfolge hat den Grenzwert von 256 Zeichen überschritten. Sie wurde auf 256 Zeichen gekürzt.
