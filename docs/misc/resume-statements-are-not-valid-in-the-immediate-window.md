# Resume-Anweisungen sind im Direktfenster nicht g&#252;ltig.
Resume\-Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler\-ID:** BC30714  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Resume`\-Anweisungen aus dem Debugcode.
