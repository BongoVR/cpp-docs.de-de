# Doppelter Spezifizierer.
Ein Schlüsselwort tritt in derselben Deklaration mehrmals auf.  
  
 **Fehler\-ID:** BC30178  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie doppelte Vorkommen des Schlüsselworts aus der Deklaration.
