# System registry could not be accessed.
Im Allgemeinen tritt dieser Fehler auf, wenn das Betriebssystem die Registrierungsdatei nicht finden kann, die Datei beschädigt ist oder ein Lese\- oder Schreibvorgang für die Datei fehlgeschlagen ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie Visual Studio neu.
