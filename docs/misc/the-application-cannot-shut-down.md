# The application cannot shut down.
Im Allgemeinen wird diese Fehlermeldung angezeigt, wenn ein Programm von außerhalb \(z. B. der Task\- Manager\) versucht, Visual Studio zu schließen, während noch ein modales Dialogfeld in Visual Studio geöffnet ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie das modale Dialogfeld, und versuchen Sie es erneut.
