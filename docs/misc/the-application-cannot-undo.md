# The application cannot undo.
Im Allgemeinen tritt dieser Fehler auf, wenn eine Rückgängig\-Operation für eine Datei ausgeführt wird, die über eine Rückgängig\-Verknüpfung zu einer anderen Datei verfügt.  Wenn nach der verknüpften Änderung ein Vorgang ausgeführt wurde, kann Visual Studio diese Änderung nicht zurücksetzen.  
  
### So beheben Sie diesen Fehler  
  
1.  Machen Sie die Änderung manuell rückgängig.
