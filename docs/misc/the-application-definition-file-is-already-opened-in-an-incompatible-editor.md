# The application definition file is already opened in an incompatible editor
**Die Anwendungsdefinitionsdatei wurde bereits in einem nicht kompatiblen Editor geöffnet.  Schließen Sie den anderen Editor, und laden Sie die Projekteigenschaftenseite erneut.**  
  
 Dieser Fehler tritt auf, wenn Sie versuchen, die Anwendungsdefinitionsdatei zu öffnen, die jedoch bereits in einem anderen Editor geöffnet ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie den anderen Editor.  
  
2.  Schließen Sie den Projekt\-Designer.  
  
3.  Klicken Sie im Menü **Projekt** auf **Eigenschaften**, um den Projekt\-Designer erneut zu öffnen.
