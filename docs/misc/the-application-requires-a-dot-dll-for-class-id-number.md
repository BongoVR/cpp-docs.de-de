# The application requires a .dll for class ID &lt;number&gt;.
Die DLL für die angegebene Klasse wurde nicht gefunden.  Dies ist ein Fehler auf Betriebssystem\-Ebene.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie Visual Studio oder die betroffene Anwendung neu, und versuchen Sie es erneut.
