# Der Verbindungstimeout-Wert muss gr&#246;&#223;er als 0 sein.
Wenn Dateien mit dem [My.Computer.Network Object](../Topic/My.Computer.Network%20Object.md) hochgeladen und heruntergeladen werden, müssen Sie einen `connectionTimeout`\-Wert angeben, der größer ist als `0`.  
  
### So beheben Sie diesen Fehler  
  
-   Geben Sie einen `connectionTimeout`\-Wert an, der größer ist als `0`.
