# Die .NET Framework-Debuggingdienste wurden nicht ordnungsgem&#228;&#223; installiert. Die wahrscheinlichste Ursache ist, dass „mscordbi.dll „ nicht ordnungsgem&#228;&#223; registriert ist. Klicken Sie auf „Hilfe“, wenn Sie weitere Informationen dazu w&#252;nschen, wie Sie Setup f&#252;r .NET Framework reparieren k&#246;nnen.
Dieser Fehler tritt üblicherweise auf, wenn Visual Studio die Datei „mscordbi.dll“ nicht finden kann, die mit den Debuggingdiensten der Common Language Runtime installiert wird.  
  
### So beheben Sie diesen Fehler  
  
-   Reparieren Sie die Installation von .NET Framework auf Ihrem Computer.
