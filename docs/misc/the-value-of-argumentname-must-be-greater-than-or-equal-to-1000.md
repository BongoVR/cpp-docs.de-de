# Der Wert von &lt;Argumentname&gt; muss gr&#246;&#223;er oder gleich 1000 sein.
Der Wert der <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize*>\-Eigenschaft muss größer oder gleich 1000 sein.  
  
 Die <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize*>\-Eigenschaft gibt die maximale Byteanzahl an, die in einer Protokolldatei zulässig sind, bevor eine neue Protokolldatei erstellt wird.  
  
### So beheben Sie diesen Fehler  
  
-   Legen Sie die <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize*>\-Eigenschaft auf eine Zahl größer oder gleich 1000 fest.
