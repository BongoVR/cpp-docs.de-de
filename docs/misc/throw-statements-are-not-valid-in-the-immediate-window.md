# Throw-Anweisungen sind im Direktfenster nicht g&#252;ltig.
`Throw`\-Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler\-ID:** BC30725  
  
### So beheben Sie diesen Fehler  
  
-   Entfernen Sie `Throw`\-Anweisungen aus Debugcode.
