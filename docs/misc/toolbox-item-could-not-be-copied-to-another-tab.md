# Toolbox item could not be copied to another tab.
Dieser Fehler tritt normalerweise auf, wenn das Element beschädigt ist oder nicht mehr genügend Speicherplatz zur Verfügung steht.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie andere geöffnete Anwendungen, und versuchen Sie dann erneut, das Element zu verschieben.  
  
     \- oder \-  
  
2.  Wählen Sie ein anderes Element aus.
