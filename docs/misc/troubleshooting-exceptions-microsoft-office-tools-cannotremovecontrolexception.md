# Problembehandlung bei Ausnahmen: Microsoft.Office.Tools.CannotRemoveControlException
Eine <xref:Microsoft.Office.Tools.CannotRemoveControlException>\-Ausnahme wird ausgelöst durch den Versuch, ein Steuerelement programmgesteuert zu entfernen, das nicht programmgesteuert hinzugefügt wurde.  
  
## Tipps  
 Sie können ein Steuerelement nur dann programmgesteuert entfernen, wenn es zuvor programmgesteuert hinzugefügt wurde.  
 -   Sie können ein Steuerelement nur dann programmgesteuert entfernen, wenn es zuvor programmgesteuert hinzugefügt wurde. Wenn das Steuerelement dem Dokument nicht zur Laufzeit, sondern während des Entwurfs hinzugefügt wurde, kann es nicht entfernt werden.
