# Problembehandlung bei Ausnahmen: Microsoft.Office.Tools.ControlNameAlreadyExistsException
Eine <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException>\-Ausnahme wird beim Versuch ausgelöst, programmgesteuert ein Steuerelement zu erstellen, dessen Name schon für ein anderes Steuerelement im Projekt vergeben ist.  
  
## Tipps  
 **Verwenden Sie für eines der Steuerelemente einen anderen Namen.**  
 Steuerelementnamen müssen eindeutig sein.
