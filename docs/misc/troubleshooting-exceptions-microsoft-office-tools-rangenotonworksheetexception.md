# Problembehandlung bei Ausnahmen: Microsoft.Office.Tools.RangeNotOnWorksheetException
Eine <xref:Microsoft.Office.Tools.RangeNotOnWorksheetException>\-Ausnahme wird ausgelöst, wenn ein Ansichtssteuerelement programmgesteuert erstellt wird, dessen Bereich nicht dem Arbeitsblatt entstammt, das der Steuerelementeauflistung entspricht.  
  
## Tipps  
 Stellen Sie sicher, dass der angegebene Bereich dem korrekten Arbeitsblatt entspricht.
