# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.CachedDataTypeMismatchException
Diese Ausnahme wird ausgelöst, wenn der mithilfe von XML serialisierte Zustand des Objekts im Cache nicht mit den Typinformationen im zwischengespeicherten Datenmanifest übereinstimmt.
