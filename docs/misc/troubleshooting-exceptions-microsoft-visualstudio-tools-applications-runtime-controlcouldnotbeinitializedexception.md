# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.ControlCouldNotBeInitializedException
Diese Ausnahme wird ausgelöst, wenn ein mit der Steuerelementauflistung hinzugefügtes Steuerelement nicht initialisiert werden konnte.
