# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.InvalidNewApplicationManifestException
Diese Ausnahme wird ausgelöst, wenn das Bereitstellungsmanifest auf ein ungültiges Anwendungsmanifest verweist.
