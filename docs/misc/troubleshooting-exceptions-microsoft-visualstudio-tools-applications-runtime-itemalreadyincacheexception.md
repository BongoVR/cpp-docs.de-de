# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.ItemAlreadyInCacheException
Diese Ausnahme wird ausgelöst, wenn ein im Cachespeicher eines Dokuments bereits vorhandenes Element dem Cachespeicher erneut hinzugefügt wird.
