# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.NotAvailableOnlineException
Diese Ausnahme wird ausgelöst, wenn eine Assembly aufgerufen wird, die nicht im Cache verfügbar ist.
