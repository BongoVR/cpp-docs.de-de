# Problembehandlung bei Ausnahmen: System.ComponentModel.InvalidEnumArgumentException
Diese Ausnahme wird ausgelöst, wenn Sie einen nicht gültigen Enumerationswert an eine Methode übergeben oder beim Festlegen einer Eigenschaft verwenden.
