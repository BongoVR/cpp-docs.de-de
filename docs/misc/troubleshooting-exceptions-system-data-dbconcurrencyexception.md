# Problembehandlung bei Ausnahmen: System.Data.DBConcurrencyException
Diese Ausnahme wird während der Aktualisierungsoperation durch den `DataAdapter` ausgelöst, wenn die Anzahl der betroffenen Zeilen gleich 0 ist.
