# Problembehandlung bei Ausnahmen: System.Data.DeletedRowInaccessibleException
Diese Ausnahme wird bei dem Versuch ausgelöst, eine Aktion in einer <xref:System.Data.DataRow> auszuführen, die gelöscht wurde.
