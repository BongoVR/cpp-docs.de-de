# Problembehandlung bei Ausnahmen: System.Data.InvalidExpressionException
Eine <xref:System.Data.InvalidExpressionException>\-Ausnahme wird bei dem Versuch ausgelöst, einer <xref:System.Data.DataColumn> eine <xref:System.Data.DataColumn.Expression*> mit einem ungültigen <xref:System.Data.DataColumnCollection> hinzuzufügen.  
  
## Tipps  
 **Überprüfen Sie die Syntax des Ausdrucks.**  
 Die <xref:System.Data.DataColumn.Expression*>\-Eigenschaft wird verwendet, um den Wert einer Spalte zu berechnen oder um eine Aggregatsspalte zu erstellen.
