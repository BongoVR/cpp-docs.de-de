# Problembehandlung bei Ausnahmen: System.Data.MissingPrimaryKeyException
Diese Ausnahme wird bei dem Versuch ausgelöst, auf eine Tabellenzeile zuzugreifen, die über keinen Primärschlüssel verfügt.
