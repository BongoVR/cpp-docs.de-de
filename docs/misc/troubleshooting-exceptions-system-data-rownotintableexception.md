# Problembehandlung bei Ausnahmen: System.Data.RowNotInTableException
Diese Ausnahme wird bei dem Versuch ausgelöst, eine Operation für eine <xref:System.Data.DataRow> durchzuführen, die sich nicht in einer <xref:System.Data.DataTable> befindet.
