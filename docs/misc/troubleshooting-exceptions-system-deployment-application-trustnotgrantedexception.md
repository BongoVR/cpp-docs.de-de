# Problembehandlung bei Ausnahmen: System.Deployment.Application.TrustNotGrantedException
Die Ausnahme wird ausgelöst, wenn die Anwendung nicht über die erforderliche Vertrauensebene verfügt, um auf dem lokalen Computer ausgeführt zu werden.
