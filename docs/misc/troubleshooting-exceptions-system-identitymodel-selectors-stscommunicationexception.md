# Problembehandlung bei Ausnahmen: System.IdentityModel.Selectors.StsCommunicationException
Eine <xref:System.IdentityModel.Selectors.StsCommunicationException>\-Ausnahme wird ausgelöst, wenn ein Problem bei der Kommunikation mit dem Sicherheitstokendienst auftritt.  
  
 Das Problem kann folgende Ursachen haben:  
  
-   Netzwerkfehler.  
  
-   MEX\-Endpunkt wurde nicht erreicht.  
  
-   Sicherheitstokendienst verfügt über keine gültige Endpunktidentität.
