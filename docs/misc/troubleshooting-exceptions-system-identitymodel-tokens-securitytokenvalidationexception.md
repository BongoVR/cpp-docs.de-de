# Problembehandlung bei Ausnahmen: System.IdentityModel.Tokens.SecurityTokenValidationException
Eine <xref:System.IdentityModel.Tokens.SecurityTokenValidationException>\-Ausnahme wird ausgelöst, wenn ein erhaltener Sicherheitstoken ungültig ist. Diese Ausnahme wird normalerweise von der <xref:System.IdentityModel.Selectors.SecurityTokenAuthenticator.ValidateToken*>\-Methode ausgelöst.
