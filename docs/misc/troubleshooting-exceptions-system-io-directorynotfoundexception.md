# Problembehandlung bei Ausnahmen: System.IO.DirectoryNotFoundException
Eine <xref:System.IO.DirectoryNotFoundException>\-Ausnahme wird ausgelöst, wenn ein Dateipfad oder ein Verzeichnis nicht vollständig gefunden wird.  
  
## Tipps  
 **Stellen Sie sicher, dass das Verzeichnis am angegebenen Speicherort vorhanden ist.**  
 Überprüfen Sie, ob das Verzeichnis am angegebenen Speicherort vorhanden ist.  
  
 **Wenn Sie relative Pfade verwenden, müssen Sie überprüfen, ob das aktuelle Verzeichnis korrekt ist.**  
 Der Pfad ist möglicherweise falsch, wenn Sie von einem fehlerhaften aktuellen Verzeichnis ausgehen.
