# Problembehandlung bei Ausnahmen: System.Net.Mail.SmtpFailedRecipientException
Diese Ausnahme wird ausgelöst, wenn eine E\-Mail mit einem <xref:System.Net.Mail.SmtpClient> versendet wird und die Auslieferung an einen Empfänger nicht möglich ist.
