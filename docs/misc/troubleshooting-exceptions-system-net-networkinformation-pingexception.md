# Problembehandlung bei Ausnahmen: System.Net.NetworkInformation.PingException
Diese Ausnahme wird ausgelöst, wenn eine <xref:System.Net.NetworkInformation.Ping.Send*>\-Methode oder eine <xref:System.Net.NetworkInformation.Ping.SendAsync*>\-Methode eine Methode aufruft, die eine Ausnahme auslöst. Dies kann z. B. der Fall sein, wenn ein Server Ping nicht unterstützt oder eine Firewall die Operation verhindert.
