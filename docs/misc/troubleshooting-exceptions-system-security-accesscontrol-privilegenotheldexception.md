# Problembehandlung bei Ausnahmen: System.Security.AccessControl.PrivilegeNotHeldException
Diese Ausnahme wird ausgelöst, wenn eine Methode in dem `AccessControl`\-Namespace versucht, eine für diese Methode nicht verfügbare Berechtigung zu aktivieren.
