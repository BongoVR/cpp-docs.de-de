# Problembehandlung bei Ausnahmen: System.ServiceModel.AddressAlreadyInUseException
Eine <xref:System.ServiceModel.AddressAlreadyInUseException>\-Ausnahme wird ausgelöst, wenn eine Adresse nicht verfügbar ist, da sie bereits in Gebrauch ist.
