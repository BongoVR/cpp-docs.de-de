# Problembehandlung bei Ausnahmen: System.ServiceModel.Dispatcher.MultipleFilterMatchesException
Eine <xref:System.ServiceModel.Dispatcher.MultipleFilterMatchesException>\-Ausnahme wird ausgelöst, wenn mehrere Filter übereinstimmen, jedoch nur eine Übereinstimmung mit einem Filter erwartet wurde.
