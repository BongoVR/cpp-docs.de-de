# Problembehandlung bei Ausnahmen: System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException
Eine <xref:System.ServiceModel.Dispatcher.NavigatorInvalidBodyAccessException>\-Ausnahme wird ausgelöst, wenn ein <xref:System.Xml.XPath.XPathNavigator> den Text einer nicht zwischengespeicherten Nachricht prüfen soll.
