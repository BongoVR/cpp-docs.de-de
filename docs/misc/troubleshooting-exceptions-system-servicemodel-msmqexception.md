# Problembehandlung bei Ausnahmen: System.ServiceModel.MsmqException
Eine <xref:System.ServiceModel.MsmqException>\-Ausnahme kapselt die von Message Queuing zurückgegebenen Fehler. Diese Ausnahme wird vom Message Queuing\-Transport und vom Message Queuing\-Integrationschannel ausgelöst.
