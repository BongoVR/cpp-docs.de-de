# Problembehandlung bei Ausnahmen: System.ServiceModel.Persistence.InstanceLockException
Die <xref:System.ServiceModel.Persistence.InstanceLockException>\-Ausnahme wird für Klassen ausgelöst, die vom <xref:System.ServiceModel.Persistence.LockingPersistenceProvider> erben, wenn der Vorgang aufgrund des Status der Instanzsperre nicht ausgeführt werden kann.
