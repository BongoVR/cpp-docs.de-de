# Problembehandlung bei Ausnahmen: System.ServiceModel.Security.SecurityAccessDeniedException
Eine <xref:System.ServiceModel.Security.SecurityAccessDeniedException>\-Ausnahme stellt die Sicherheitsausnahme dar, die ausgelöst wird, wenn eine Sicherheitsautorisierungsanforderung fehlschlägt.
