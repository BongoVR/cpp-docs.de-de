# Problembehandlung bei Ausnahmen: System.Threading.SemaphoreFullException
Diese Ausnahme wird ausgelöst, wenn die <xref:System.Threading.Semaphore.Release*>\-Methodenüberladung auf einem Semaphore aufgerufen wird, dessen Zähler bereits den Maximalwert erreicht hat.
