# Problembehandlung bei Ausnahmen: System.Threading.ThreadStateException
Diese Ausnahme wird ausgelöst, wenn ein Thread einen <xref:System.Threading.ThreadState> aufweist, der für den Methodenaufruf nicht gültig ist.
