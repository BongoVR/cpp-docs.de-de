# Problembehandlung bei Ausnahmen: System.Threading.WaitHandleCannotBeOpenedException
Diese Ausnahme wird bei dem Versuch ausgelöst, ein nicht vorhandenes Systemmutex oder ein nicht vorhandenes Semaphore zu öffnen.
