# Problembehandlung bei Ausnahmen: System.Transactions.TransactionInDoubtException
Diese Ausnahme wird ausgelöst, wenn versucht wird, eine Operation auf einer zweifelhaften Transaktion durchzuführen.  
  
## Hinweise  
 Eine Transaktion ist zweifelhaft, wenn zu einem registrierten Ressourcen\-Manager kein Kontakt hergestellt werden kann.
