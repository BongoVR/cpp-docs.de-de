# Problembehandlung bei Ausnahmen: System.TypeInitializationException
Die Ausnahme, die als Wrapper um die von dieser Klasseninitialisierung ausgelöste Ausnahme ausgelöst wird.  
  
## Hinweise  
 Wenn die Initialisierung eines Typs durch eine Klasseninitialisierung fehlschlägt, wird eine <xref:System.TypeInitializationException> erstellt. Dieser Ausnahme wird dann ein Verweis auf die Ausnahme übergeben, die durch die Klasseninitialisierung des Typs ausgelöst wurde. Die <xref:System.Exception.InnerException*>\-Eigenschaft der <xref:System.TypeInitializationException> enthält die zugrunde liegende Ausnahme.
