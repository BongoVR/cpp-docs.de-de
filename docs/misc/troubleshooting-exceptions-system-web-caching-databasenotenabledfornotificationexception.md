# Problembehandlung bei Ausnahmen: System.Web.Caching.DatabaseNotEnabledForNotificationException
Diese Ausnahme wird ausgelöst, wenn eine Microsoft SQL Server\-Datenbank nicht dafür eingerichtet ist, die mit der <xref:System.Web.Caching.SqlCacheDependency>\-Klasse verknüpften Abhängigkeiten zu unterstützen.
