# Problembehandlung bei Ausnahmen: System.Web.Caching.TableNotEnabledForNotificationException
Die Ausnahme, die ausgelöst wird, wenn eine <xref:System.Web.Caching.SqlCacheDependency>\-Klasse für eine Datenbanktabelle verwendet wird, für die keine Änderungsbenachrichtigungen aktiviert sind.
