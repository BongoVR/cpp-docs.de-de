# Problembehandlung bei Ausnahmen: System.Windows.Automation.NoClickablePointException
Eine <xref:System.Windows.Automation.NoClickablePointException>\-Ausnahme enthält Informationen zu der Ausnahme, die ausgelöst wird, wenn <xref:System.Windows.Automation.AutomationElement.GetClickablePoint*> für ein UI\-<xref:System.Windows.Automation.AutomationElement>\-Element ohne durch Klicken aktivierbaren Punkt aufgerufen wird.  
  
## Hinweise  
 Diese Ausnahme kann ausgelöst werden, wenn das Anwendungsfenster minimiert wird oder das Element sich außerhalb des Bildschirmbereichs befindet.
