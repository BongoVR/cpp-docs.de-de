# Problembehandlung bei Ausnahmen: System.Windows.Media.Animation.AnimationException
Eine <xref:System.Windows.Media.Animation.AnimationException>\-Ausnahme wird ausgelöst, wenn während des Animiervorgangs einer Eigenschaft ein Fehler auftritt.  
  
## Hinweise  
 Diese Klasse wird nicht für Zeitsteuerungsfehler verwendet, z. B. das Erstellen einer Animation mit einer negativen <xref:System.Windows.Media.Animation.Timeline.Duration*> oder mit einer ungültigen <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio*>\-Einstellung und <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio*>\-Einstellungen.
