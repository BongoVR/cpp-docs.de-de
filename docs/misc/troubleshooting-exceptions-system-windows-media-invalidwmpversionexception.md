# Problembehandlung bei Ausnahmen: System.Windows.Media.InvalidWmpVersionException
Eine <xref:System.Windows.Media.InvalidWmpVersionException>\-Ausnahme wird ausgelöst, wenn die installierte Microsoft Windows Media Player\-Version nicht unterstützt wird.
