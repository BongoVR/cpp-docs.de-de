# Problembehandlung bei Ausnahmen: System.Workflow.ComponentModel.Compiler.WorkflowValidationFailedException
Eine <xref:System.Workflow.ComponentModel.Compiler.WorkflowValidationFailedException> wird ausgelöst, wenn die Validierung für einen Workflow fehlschlägt. Diese Klasse kann nicht vererbt werden.
