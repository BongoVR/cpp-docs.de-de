# Die XML-Dokumentationsdatei &#39;&lt;Name&gt;&#39; konnte nicht erstellt werden: &lt;Meldung&gt;
Die XML\-Dokumentationsdatei kann aufgrund eines Fehlers nicht erstellt werden.  
  
 **Fehler\-ID:** BC42311  
  
 Mit dem Fehler wird möglicherweise die folgende Meldung angezeigt:  
  
-   [Der XML\-Kommentarblock kann keinem Sprachelement zugeordnet werden, das die Anwendung von XML\-Dokumentationskommentaren unterstützt](../misc/xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments.md).  
  
-   [Das XML\-Kommentartag "returns" ist für eine WriteOnly\-Eigenschaft nicht zulässig.](../misc/xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property.md).  
  
-   [Der XML\-Kommentar kann nicht mehrere Male auf einen teilweisen \<Typ\> angewendet werden.](../misc/xml-comment-cannot-be-applied-more-than-once-on-a-partial-type.md).  
  
-   [Das XML\-Kommentartag "returns" ist für ein "declare sub"\-Sprachelement nicht zulässig.](../misc/xml-comment-tag-returns-is-not-permitted-on-a-declare-sub-language-element.md).  
  
-   [Analysefehler in XML\-Dokumentation: Für das Starttag "\<Tag\>" ist kein entsprechendes Endtag vorhanden.](../misc/xml-documentation-parse-error-start-tag-tag-doesn-t-have-a-matching-end-tag.md).  
  
-   [Der \<Parameter\>\-Typparameter des XML\-Kommentars stimmt mit keinem Typparameter in der entsprechenden \<Schlüsselwort\>\-Anweisung überein](../misc/xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement.md).  
  
-   [Typparameter für XML\-Kommentare müssen ein name\-Attribut aufweisen.](../misc/xml-comment-type-parameter-must-have-a-name-attribute.md).  
  
-   [XML comment exception must have a 'cref' attribute](../Topic/XML%20comment%20exception%20must%20have%20a%20'cref'%20attribute.md).  
  
### So beheben Sie diesen Fehler  
  
-   Empfehlungen finden Sie in der bereitgestellten Meldung.
