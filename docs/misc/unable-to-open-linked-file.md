# Unable to Open Linked File
Dem Projekt wurde eine Datei als verknüpfte Datei hinzugefügt.  Als Sie das Projekt geöffnet haben, konnte das Projektsystem die Datei jedoch nicht finden.  
  
 **So beheben Sie diesen Fehler**  
  
-   Löschen Sie die verknüpfte Datei im Projektmappen\-Explorer.  
  
     \- oder \-  
  
-   Verschieben Sie die Datei wieder in das Verzeichnis, in dem sie vom Projektsystem gesucht wird.
