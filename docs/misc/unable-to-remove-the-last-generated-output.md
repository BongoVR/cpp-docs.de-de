# Unable to remove the last generated output
Die zuletzt bekannte generierte Ausgabe wurde nicht von der Festplatte und\/oder aus dem Projekt entfernt.  
  
 Wenn ein benutzerdefiniertes Tool einen Fehler zurückgibt, wird die zuletzt generierte Ausgabe \(falls vorhanden\) von der Festplatte und aus dem Projekt entfernt.  
  
 Dies kann passieren, wenn die generierte Ausgabe auf der Festplatte gesperrt ist.  
  
 **So beheben Sie diesen Fehler**  
  
-   Schließen Sie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)], und führen Sie einen Neustart aus.
