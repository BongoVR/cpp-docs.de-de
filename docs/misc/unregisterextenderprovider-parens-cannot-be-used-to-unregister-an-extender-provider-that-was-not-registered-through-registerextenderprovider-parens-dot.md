# UnregisterExtenderProvider() cannot be used to unregister an Extender Provider that was not registered through RegisterExtenderProvider().
Registrierungsbasierte Anbieter müssen weder explizit registriert werden noch muss ihre Registrierung explizit rückgängig gemacht werden.
