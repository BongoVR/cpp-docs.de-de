# Der Visual Basic-Compiler kann nach folgendem Fehler nicht wiederhergestellt werden: &lt;Fehler&gt;
Der [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)]\-Compiler kann nach folgendem Fehler nicht wiederhergestellt werden: \<`error`\>.  
  
 Einige IDE\-Features wurden deaktiviert.  
  
 Ein interner Fehler ist im [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)]\-Compiler aufgetreten.  
  
 **Fehler\-ID:** BC31021  
  
### So beheben Sie diesen Fehler  
  
1.  Speichern Sie Ihre Arbeit, und starten Sie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] erneut.  
  
2.  Wenn der Fehler erneut auftritt, installieren Sie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] neu.  
  
3.  Wenn der Fehler auch nach der erneuten Installation auftritt, informieren Sie den Produktsupport von Microsoft.
