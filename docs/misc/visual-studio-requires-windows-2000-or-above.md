# Visual Studio requires Windows 2000 or above.
Visual Studio erfordert Windows NT 4.0, Build 1381, mit Service Pack 3 bzw. Windows 95, Build 708, oder höhere Versionen.  
  
### So beheben Sie diesen Fehler  
  
1.  Aktualisieren Sie die derzeit verwendete Version von Windows NT oder Windows auf eine der oben aufgeführten Versionen.
