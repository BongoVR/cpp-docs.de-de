# VsRegEx-Befehlszeilenprogramm
Das Dienstprogramm VsRegEx wurde in Visual Studio 2010 und höher durch [CreateExpInstance\-Dienstprogramm](../Topic/CreateExpInstance%20Utility.md) ersetzt.  
  
## Hinweise  
 In Visual Studio 2008 wurden die Konfigurationsinformationen zur experimentellen Instanz der IDE in der Systemregistrierung gespeichert. In Visual Studio 2010 und höher werden sie im Dateisystem gespeichert.
