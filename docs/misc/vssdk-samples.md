# VSSDK-Beispiele
Die offizielle Website für Beispiele zu "Visual Studio\-Erweiterbarkeit" ist die Website [MSDN Code Gallery](http://go.microsoft.com/fwlink/?LinkID=127810). Die Website enthält einführende Beispiele, Beispiele zur Veranschaulichung neuer Visual Studio\-Features, ausführliche Dokumentationen, Links zu freigegebenen Quellprojekten und weitere Ressourcen, die Sie bei der Erweiterung von Visual Studio unterstützen. Wenn Sie eine Liste dieser Ressourcen anzeigen möchten, geben Sie **VSX** in das Feldfür die Beispielsuche auf der Startseite ein.  
  
 Die [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] SDK\-Beispiele veranschaulichen, wie eine Erweiterung in die integrierte Visual Studio\-Entwicklungsumgebung \(Integrated Development Environment, IDE\) eingebunden wird. Das Quellcodeverzeichnis jedes Beispiels enthält ein Dokument namens "Overview.xml", das Implementierungshinweise zum betreffenden Beispiel enthält.  
  
 Zusätzlich ist ein IronPython\-Beispiel verfügbar. Es ist in das [IronPython Studio\-Projekt](http://go.microsoft.com/fwlink/?LinkID=183554) auf der CodePlex\-Website integriert.
