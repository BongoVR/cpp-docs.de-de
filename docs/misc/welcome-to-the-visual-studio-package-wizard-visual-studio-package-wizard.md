# Willkommen beim Assistenten f&#252;r das Visual Studio-Paket, Assistent f&#252;r das Visual Studio-Paket
Die [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]\-Paketvorlage erstellt eine Basisimplementierung eines VSPackage, das Sie später erweitern können. Sie können die [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]\-Paketvorlage auch dazu verwenden, einem VSPackage optionale Komponenten hinzuzufügen.  
  
 **Abbrechen**  
 Bricht das Erstellen des VSPackage ab.  
  
 **Vorheriges**  
 Wechselt zum vorherigen Schritt.  
  
 **Weiter**  
 Wechselt zum nächsten Schritt.
