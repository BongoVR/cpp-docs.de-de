# Wizard file &lt;name&gt; format is not recognized.
Der Assistent kann nicht geöffnet werden.  Dieser Fehler wird im Allgemeinen durch eine beschädigte Assistentendatei oder einen Assistenten verursacht, die bzw. der nicht mit der Versionsnummer von Visual Studio übereinstimmt.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass die Assistentendatei dieselbe Versionsnummer aufweist wie die Version von Visual Studio, die als Wert von VSWIZARD angegeben ist \(z. B. `VSWIZARD 7.0`\).  
  
     \- oder \-  
  
     Installieren Sie Visual Studio neu, um die beschädigte Datei zu reparieren.
