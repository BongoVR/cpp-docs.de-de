# 2.7 Data Environment
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

In diesem Abschnitt werden einige Direktiven und Klauseln zur Steuerung der Datenumgebung bei der Ausführung paralleler Bereichen wie folgt vor:  
  
-   **threadprivate**\-Direktive \(finden Sie im folgenden Abschnitt\) werden bereitgestellt, um Dateigültigkeitsbereich, der Namespacebereich Blockbereichs oder der statischen Variablen vorzunehmen, die zu einem Thread lokal sind.  
  
-   Klauseln, die möglicherweise auf die Direktive angegeben werden, um die Freigabe von Attributen von Variablen für die Dauer der Ähnlichkeit oder Arbeitsteilungs Konstrukte zu steuern, sind in [2.7.2 Abschnitt](../../parallel/openmp/2-7-2-data-sharing-attribute-clauses.md) auf Seite 25 beschrieben.
