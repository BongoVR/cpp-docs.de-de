# 2. Directives
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Direktiven basieren auf **\#pragma**\-Direktive, die in den Programmiersprachen C und C\+\+\-Standards definiert sind.  Compiler, die OpenMP C und C\+\+ APIs unterstützen, stellen eine Befehlszeilenoption, die Interpretation aller OpenMP\-Compiler Direktiven aktiviert und ermöglicht.
