# A.3   Using Parallel Regions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Die `parallel`\-Direktive \([Abschnitt 2.3](../../parallel/openmp/2-3-parallel-construct.md) auf Seite 8\) kann in Grob korn Ähnlichkeits programmen verwendet werden.  Der im folgenden Beispiel entscheidet jeder Thread im parallelen Bereich, für welchen Teil des globalen Arrays `x` , funktionieren nach den Thread Nummer:  
  
```  
#pragma omp parallel shared(x, npoints) private(iam, np, ipoints)  
{  
    iam = omp_get_thread_num();  
    np =  omp_get_num_threads();  
    ipoints = npoints / np;  
    subdomain(x, iam, ipoints);  
}  
```
