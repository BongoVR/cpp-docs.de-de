# <a name="ompgetthreadnum"></a>omp_get_thread_num
Gibt die Thread-Anzahl der ausgeführten Threads innerhalb Hashteams Thread zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
int omp_get_thread_num( );  
```
