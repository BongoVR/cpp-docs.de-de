# omp_get_thread_num
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Gibt die Thread Nummer des Threads zurück, der innerhalb des Teams Thread ausgeführt wird.  
  
## Syntax  
  
```  
int omp_get_thread_num( );  
```
