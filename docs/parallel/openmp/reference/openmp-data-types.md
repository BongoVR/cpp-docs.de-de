# OpenMP Data Types
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Enthält Links zu den Datentypen, die im OpenMP API verwendet werden.  
  
 Die Visual C\+\+\-Implementierung des OpenMP\-Standards beinhaltet die folgenden Datentypen.  
  
|Datentyp|Beschreibung|  
|--------------|------------------|  
|[omp\_lock\_t](../../../parallel/openmp/reference/omp-lock-t.md)|Ein Typ, der den Status einer Sperre hält, ob die Sperre verfügbar wird oder wenn ein Thread eine Sperre besitzt.|  
|[omp\_nest\_lock\_t](../../../parallel/openmp/reference/omp-nest-lock-t.md)|Ein Typ, der eine der folgenden Informationen über eine Sperre enthält: ob die Sperre verfügbar, und die Identität des Threads, der die Anzahl der Schachtelungs und eine Sperre besitzt.|
