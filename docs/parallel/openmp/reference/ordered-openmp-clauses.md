# ordered (OpenMP Clauses)
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

Erforderlich auf einer parallelen [for](../../../parallel/openmp/reference/for-openmp.md)\-Anweisung, wenn [ordered](../../../parallel/openmp/reference/ordered-openmp-directives.md)\-Direktive in der Schleife verwendet werden sollen.  
  
## Syntax  
  
```  
ordered  
```  
  
## Hinweise  
 `ordered` gilt für [for](../../../parallel/openmp/reference/for-openmp.md) die Direktive.  
  
 Weitere Informationen finden Sie unter [2.4.1 for Construct](../../../parallel/openmp/2-4-1-for-construct.md).
