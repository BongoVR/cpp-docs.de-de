# named_guids
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+\-spezifisch**  
  
 Weist den Compiler an, die GUID\-Variablen im alten Stil zu definieren und zu initialisieren, und zwar in der Form **LIBID\_MyLib**, **CLSID\_MyCoClass**, **IID\_MyInterface** und **DIID\_MyDispInterface**.  
  
## Syntax  
  
```  
named_guids  
```  
  
## Hinweise  
 **Ende C\+\+\-spezifisch**
