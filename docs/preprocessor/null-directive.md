# NULL-Direktive
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die Präprozessordirektive Null ist ein Einzahlzeichen \(**\#**\), das allein auf einer Zeile erscheint.  Sie hat keine Auswirkungen.  
  
## Syntax  
  
```  
  
#  
  
```
