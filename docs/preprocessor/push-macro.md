# push_macro
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Speichert den Wert des *macro\_name*\-Makros oben im Stapel für dieses Makro.  
  
## Syntax  
  
```  
  
#pragma push_macro("  
macro_name  
")  
  
```  
  
## Hinweise  
 Sie können den Wert für *macro\_name* mit **pop\_macro** abrufen.  
  
 Ein Beispiel finden Sie unter [pop\_macro](../preprocessor/pop-macro.md).
