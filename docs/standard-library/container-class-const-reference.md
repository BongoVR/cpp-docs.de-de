# Container-Klasse::const_reference
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Dieses Thema ist in der Visual C\+\+\-Dokumentation nicht als funktionsfähiges Beispiel von den Containern, die in der C\+\+\-Standardbibliothek verwendet werden.  Weitere Informationen finden Sie unter [STL\-Container](../standard-library/stl-containers.md).  
  
 Beschreibt ein Objekt, das als konstanten Verweis auf ein Element der Sequenz gesteuerten dienen kann.  
  
## Syntax  
  
```  
  
typedef T3 const_reference;  
  
```  
  
## Hinweise  
 Es wird hier als Synonym für den angegebenen Typ nicht **T3** beschrieben \(normalerweise **Alloc::const\_reference**\).
