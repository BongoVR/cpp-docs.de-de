# Container-Klasse::const_reverse_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  Dieses Thema ist in der Visual C\+\+\-Dokumentation nicht als funktionsfähiges Beispiel von den Containern, die in der C\+\+\-Standardbibliothek verwendet werden.  Weitere Informationen finden Sie unter [STL\-Container](../standard-library/stl-containers.md).  
  
 Beschreibt ein Objekt, das als konstanter umgekehrter Iterator für die gesteuerte Sequenz dienen kann.  
  
## Syntax  
  
```  
  
typedef T8 const_reverse_iterator;  
  
```  
  
## Hinweise  
 Es wird hier als Synonym für den angegebenen Typ nicht **T8** beschrieben \(in der Regel [reverse\_iterator](../standard-library/container-class-reverse-iterator.md) \<[const\_iterator](../standard-library/container-class-const-iterator.md)`>`\).
