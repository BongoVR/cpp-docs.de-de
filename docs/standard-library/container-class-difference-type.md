# <a name="container-classdifferencetype"></a>Container-Klasse::difference_type
> [!NOTE]
>  Dieses Thema ist in der Dokumentation zu Visual C++ als nicht funktionierendes Beispiel für Container, die in der C++-Standardbibliothek verwendet werden, aufgeführt. Weitere Informationen finden Sie unter [C++ Standard Library Containers (C++-Standardbibliothekcontainer)](../standard-library/stl-containers.md).  
  
 Beschreibt ein Objekt, das die Differenz zwischen den Adressen von zwei beliebigen Elementen in der gesteuerten Sequenz darstellen kann  
  
## <a name="syntax"></a>Syntax  
  
```  
 
typedef T1 difference_type;  
```  
  
## <a name="remarks"></a>Hinweise  
 Sie wird hier als Synonym für den nicht angegebenen Typ **T1** beschrieben (normalerweise **Alloc::difference_type**).
