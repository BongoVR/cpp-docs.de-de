# defer_lock_t-Struktur
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Stellt einen Typ dar, der ein [defer\_lock](../Topic/defer_lock%20Variable.md)\-Objekt definiert, das verwendet wird, um einen der überladenen Konstruktoren von [unique\_lock](../standard-library/unique-lock-class.md) auswählen.  
  
## Syntax  
  
```cpp  
struct defer_lock_t;  
```  
  
## Anforderungen  
 **Header:** mutex  
  
 **Namespace:** std
