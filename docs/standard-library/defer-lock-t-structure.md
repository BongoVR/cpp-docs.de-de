# <a name="deferlockt-structure"></a>defer_lock_t-Struktur

Stellt einen Typ dar, der ein [defer_lock](../standard-library/mutex-functions.md#defer_lock)-Objekt definiert, das zum Auswählen eines überladenen Konstruktors von [unique_lock](../standard-library/unique-lock-class.md) verwendet wird.

## <a name="syntax"></a>Syntax

```cpp
struct defer_lock_t;
```

## <a name="requirements"></a>Anforderungen

**Header:** \<mutex>

**Namespace:** std
