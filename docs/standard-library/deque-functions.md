 Tauscht die Elemente von zwei deque-Objekten aus.  
  
```

    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### <a name="parameters"></a>Parameter  
 `left`  
 Ein Objekt vom Typ `deque`.

 Ein Objekt vom Typ `deque`.
