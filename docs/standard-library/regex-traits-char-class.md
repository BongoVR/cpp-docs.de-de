# Regex_traits &lt; Char &gt;-Klasse
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die Spezialisierung von regex\_traits für char.  
  
## Syntax  
  
```  
template <>  
    class regex_traits<char>  
```  
  
## Hinweise  
 Die\-Klasse ist eine explizite Spezialisierung der Vorlagenklasse [regex\_traits\-Klasse](../standard-library/regex-traits-class.md) für Elemente des Typs `char` \(sodass sie Library\-Funktionen, die Objekte dieses Typs bearbeiten nutzen kann\).  
  
## Anforderungen  
 **Header:** \<regex\>  
  
 **Namespace:** std
