# <a name="regextraitsltwchartgt-class"></a>regex_traits&lt;wchar_t&gt;-Klasse
Die Spezialisierung von „regex_traits“ für „wchar_t“.  
  
## <a name="syntax"></a>Syntax  
  
```  
template <>  
class regex_traits<wchar_t>  
```  
  
## <a name="remarks"></a>Hinweise  
 Die Klasse ist eine explizite Spezialisierung der Vorlagenklasse [regex_traits](../standard-library/regex-traits-class.md) für Elemente des Typs `wchar_t` (sodass sie Bibliotheksfunktionen zum Bearbeiten von Objekten dieses Typs nutzen kann).  
  
## <a name="requirements"></a>Anforderungen  
 **Header:** \<regex >  
  
 **Namespace:** std
