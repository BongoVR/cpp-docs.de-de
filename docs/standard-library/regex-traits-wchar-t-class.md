# regex_traits&lt;wchar_t&gt;-Klasse
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die Spezialisierung von „regex\_traits“ für „wchar\_t“.  
  
## Syntax  
  
```  
template <>  
    class regex_traits<wchar_t>  
```  
  
## Hinweise  
 Die Klasse ist eine explizite Spezialisierung der Vorlagenklasse [regex\_traits\-Klasse](../standard-library/regex-traits-class.md) für Elemente des Typs `wchar_t` \(sodass sie Bibliotheksfunktionen zum Bearbeiten von Objekten dieses Typs nutzen kann\).  
  
## Anforderungen  
 **Header:** \<regex\>  
  
 **Namespace:** std
