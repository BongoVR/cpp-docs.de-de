# <a name="ltsample-containergt"></a>&lt;Sample Container&gt;
> [!NOTE]
>  Dieses Thema ist in der Dokumentation zu Visual C++ als nicht funktionierendes Beispiel für Container, die in der C++-Standardbibliothek verwendet werden, aufgeführt. Weitere Informationen finden Sie unter [C++-Standardbibliothekscontainer](../standard-library/stl-containers.md).  
  
 Zeigt die Struktur der Container-Header in der C++-Standardbibliothek.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
#include <sample container> // Nonfunctional header  
  
```
