# &lt;typeinfo&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Schließen Sie die Standardheader `<typeinfo>` ein, um einige Typen, die dem TypIdentifikationsoperator [typeid\-Operator](../cpp/typeid-operator.md) zugeordnet werden, der Informationen zur statischen und dynamischen Typen führt.  
  
## Syntax  
  
```  
  
#include <typeinfo>  
  
```  
  
## Hinweise  
 Informationen zu Klassen, die im typeinfo \<definiert werden,\>finden Sie unter folgenden Themen:  
  
-   [bad\_cast\-Ausnahme](../cpp/bad-cast-exception.md)  
  
-   [bad\_typeid\-Ausnahme](../cpp/bad-typeid-exception.md)  
  
-   [type\_info\-Klasse](../cpp/type-info-class.md)
