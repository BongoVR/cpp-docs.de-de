# C++-REST-SDK (Codename &quot;Casablanca&quot;)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Mit dem C\+\+ REST SDK \(Codename "Casablanca"\) können Sie problemlos modernen, asynchronen C\+\+\-Code schreiben, der sich mit cloudbasierten Diensten verbinden kann.  
  
 Das Projekt „Casablanca“ ist nicht mehr im Lieferumfang von Visual Studio enthalten.  Die Dokumentation und die Quellen sind auf auf CodePlex gehostet und werden gelegentlich aktualisiert.  Ausführlichere Informationen zu diesem Projekt finden Sie im Abschnitt über das [C\+\+ REST SDK \(Codename "Casablanca"\)](http://casablanca.codeplex.com/).
