# Attribute Categories
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Der Verweis auf ein Attribut beschreibt die verschiedenen Typen von Attributen, die Sie in Visual C\+\+ verwenden können.  
  
## Verwandte Abschnitte  
 [COM\-Attribute](../windows/com-attributes.md)  
 Fügt Code ein, um zahlreiche Bereiche der COM\-Entwicklung zu unterstützen.  
  
 [IDL\-Attribute](../windows/idl-attributes.md)  
 Fügt Code ein, der die zugeordnete IDL\-Datei aus einer Quellcodedatei ohne einen Assistenten und ohne mit der Struktur und die Syntax dieser Datei vertraut zu sein erstellt oder geändert wird.  
  
 [OLE DB\-Consumerattribute](../windows/ole-db-consumer-attributes.md)  
 Fügt auf der Grundlage [OLE DB\-Consumervorlagen](../data/oledb/ole-db-consumer-templates-reference.md)Code ein, um einen Arbeitsen OLE DB\-Consumer zu erstellen, der Aufgaben wie das öffnende Tabellen ausführt, Befehle ausführt und Daten zugreift.  
  
 [Compiler\-Attribute](../windows/compiler-attributes.md)  
 Fügt Code, der eine Reihe allgemeiner Programmieraufgaben ausführt.  
  
 [Attributierte Programmierung](../windows/attributed-programming-concepts.md)  
 Enthält Links zu konzeptionellen Themen, die attributierte Programmierung beschrieben werden.
