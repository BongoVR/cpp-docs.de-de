# Attribute Requirements
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die Anforderungen, die für C\+\+\-Attribute aufgeführt sind, werden die Projekttypen, die Compilereinstellungen und andere Informationen, die erforderlich sind, damit ein Attribut.  Die Kategorien von Informationen sind im Folgenden beschrieben.  
  
> [!NOTE]
>  Verwenden der Attribute für eine Klasse, die von einer Klasse abgeleitet ist, die auch Attribute verwendet, wird nicht unterstützt.  
  
## Header  
 Dieses Feldlisten die Headerdateien, die eingeschlossen werden müssen, bevor ein Attribut verwendet werden kann.  
  
## Project  
 Dieses Feld beschreibt die Projekttypen, in denen ein Attribut verwendet werden kann.  
  
## Compiler  
 Dieses Feld stellt die Compileroptionen, die vorhanden sein müssen, damit dieses Attribut verwendet werden kann.
