# Changing Unassigned Symbols
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Während Sie sich im Dialogfeld [Ressourcensymbole](../windows/resource-symbols-dialog-box.md) befinden, können Sie vorhandene Symbole bearbeiten oder löschen, die nicht bereits einer Ressource oder einem Objekt zugewiesen sind.  
  
### So ändern Sie ein nicht zugewiesenes Symbol  
  
1.  Klicken Sie im Feld **Name** auf das nicht zugewiesene Symbol und dann auf **Ändern**.  
  
2.  Bearbeiten Sie den Symbolnamen oder\-Wert in den angezeigten Feldern des Dialogfelds **Symbol ändern**.  
  
    > [!NOTE]
    >  Um ein Symbol zu ändern, das einer Ressource oder einem Objekt zugewiesen *ist*, müssen Sie den Ressourcen\-Editor oder das Eigenschaftsfenster verwenden.  Weitere Informationen finden Sie unter [Ändern eines Symbols oder Symbolnamens](../windows/changing-a-symbol-or-symbol-name-id.md).  
  
 Informationen zum Hinzufügen von Ressourcen zu verwalteten Projekten finden Sie unter [Ressourcen in Anwendungen](../Topic/Resources%20in%20Desktop%20Apps.md) im *.NET Framework\-Entwicklerhandbuch.*  
  
 Anforderungen  
  
 Win32
