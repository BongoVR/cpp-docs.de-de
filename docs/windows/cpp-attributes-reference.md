# C++ Attributes Reference
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Attribute zur Vereinfachung der COM\-Programmierung und.NET Framework common Language Runtime vereinfachen.  Wenn Sie Attribute in Ihre Quelldateien einschließen, arbeitet der Compiler mit Provider\-DLLs, um Code einzufügen oder den Code in den generierten Objektdateien zu ändern.
