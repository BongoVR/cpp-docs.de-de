# Creating, Moving, and Editing Toolbar Buttons
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Schaltflächen der Symbolleiste können auf einfache Weise erstellt, verschoben, kopiert und bearbeitet werden.  
  
 Eine neue oder leere Schaltfläche wird standardmäßig ganz rechts auf der Symbolleiste angezeigt.  Sie können diese Schaltfläche vor dem Bearbeiten verschieben.  Wenn Sie eine neue Schaltfläche erstellen, wird eine weitere leere Schaltfläche rechts neben der bearbeiteten Schaltfläche angezeigt.  Beim Speichern einer Symbolleiste wird die leere Schaltfläche nicht gespeichert.
