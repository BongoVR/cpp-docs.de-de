# Customizing or Changing Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

In der [Farbenpalette](../windows/colors-window-image-editor-for-icons.md) der Bildbearbeitung werden anfänglich 16 Standardfarben angezeigt.  Neben den bereits angezeigten Farben können Sie eigene benutzerdefinierten Farben erstellen.  Sie können dann [eine benutzerdefinierte Farbpalette speichern und laden](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).  
  
### So ändern Sie die Farben in der Farbpalette  
  
1.  Wählen Sie im Menü **Bild** die Option **Farben anpassen** aus.  
  
2.  Definieren Sie im Dialogfeld [Benutzerdefinierte Farbauswahl](../windows/custom-color-selector-dialog-box-image-editor-for-icons.md) die Farbe, indem Sie RGB\- oder HSL\-Werte in die entsprechenden Textfelder eingeben oder eine Farbe im Feld **Farbverlauf Farbanzeige** auswählen.  
  
3.  Legen Sie die Helligkeit durch Verschieben des Schiebereglers auf der Leiste **Helligkeit** fest.  
  
4.  Viele benutzerdefinierte Farben werden gedithert.  Wenn Sie die Volltonfarbe verwenden möchten, die der Ditheringfarbe Farbe am ehesten entspricht, doppelklicken Sie auf das Feld **Farbe**.  
  
     Wenn Sie sich später für die geditherte Farbe entscheiden, verschieben Sie den Regler auf der Leiste **Helligkeit** oder das Fadenkreuz im Feld **Farbverlauf Farbanzeige** erneut, um das Dithering wiederherzustellen.  
  
5.  Klicken Sie auf **OK**, um die neue Farbe hinzuzufügen.
