# Editing Binary Data
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### So bearbeiten Sie eine Ressource im Binär\-Editor  
  
1.  Markieren Sie das zu bearbeitende Byte.  
  
     Mit der TAB\-TASTE verschieben Sie den Fokus zwischen den Hexadezimal\- und ASCII\-Abschnitten des Binär\-Editors.  Mit der BILD\-AUF\- und der BILD\-AB\-TASTE können Sie sich bildschirmweise durch die Ressource bewegen.  
  
2.  Geben Sie den neuen Wert ein.  
  
     Der Wert wird sofort sowohl im Hexadezimal\- als auch im ASCII\-Abschnitt geändert, und der Fokus wechselt zum nächsten Wert in der Zeile.  
  
    > [!NOTE]
    >  Der Binär\-Editor übernimmt Änderungen beim Schließen automatisch.
