# Editing Managed Resource Files
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Verwaltete Ressourcendateien \(.resx\) sind XML\-Dateien.  Wenn Sie einem Projekt im Dialogfeld **Neues Element hinzufügen** eine verwaltete Ressourcendatei hinzufügen, wird standardmäßig der **Verwaltete Ressourcen\-Editor** geöffnet.  
  
 Weitere Informationen finden Sie unter [Ressourcen im RESX\-Dateiformat](assetId:///0c476133-87e4-47e8-b0ef-4b88f4ef3dc5).
