# Evening the Spacing Between Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Mithilfe des Dialog\-Editors können Sie die Abstände der Steuerelemente zwischen dem ersten und letzten markierten Steuerelement einheitlich gestalten.  
  
### So verteilen Sie Steuerelemente in gleichmäßigen Abständen  
  
1.  Markieren Sie die Steuerelemente, die neu angeordnet werden sollen.  
  
2.  Klicken Sie im Menü **Format** auf **Gleichmäßig verteilen**, und wählen Sie anschließend einen der folgenden Ausrichtungsbefehle:  
  
    -   **Quer**: Verteilt die Steuerelemente gleichmäßig zwischen dem äußeren linken und dem äußeren rechten markierten Steuerelement.  
  
    -   **Nach unten**: Verteilt die Steuerelemente gleichmäßig zwischen dem obersten und dem untersten markierten Steuerelement.
