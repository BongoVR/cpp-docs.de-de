# <a name="guide-settings-dialog-box"></a>Dialogfeld "Führungslinieneinstellungen"
## <a name="layout-guides"></a>Layout-Handbücher  
 Zeigt die Einstellungen für den Layout-Handbüchern.  
  
 **Keine**  
  
 Blendet Layout-Tools.  
  
 **Lineale und Führungslinien**  
  
 Wenn aktiviert, wird das Layout-Tools Lineale hinzugefügt; Handbücher können in der Lineale platziert werden. Die Standard-Handbücher sind die Ränder, die durch Ziehen verschoben werden können. Klicken Sie in der Lineale zum Platzieren einer Führungslinie. Steuerelemente "Snap" Anleitungen, wenn die Steuerelemente über oder neben ihnen verschoben werden. Steuerelemente werden auch mit eine Anleitung verschoben, sobald sie an sie angefügt sind. Wenn Sie ein Steuerelement in ein auf jeder Seite angefügt ist, und eine Anleitung verschoben wird, wird das Steuerelement geändert.  
  
 **Raster**  
  
 Erstellt ein Layoutraster. Neue Steuerelemente werden automatisch am Raster ausgerichtet.  
  
## <a name="grid-spacing"></a>Rasterweite  
 Zeigt die Einstellungen für den Rasterabstand im Feld Dialogeinheiten (DLUs).  
  
 **Breite: DLUs**  
  
 Legt die Breite des Layoutrasters in DLUs fest. Eine horizontale DLU ist die durchschnittliche Breite der Dialogfeld-Schriftart geteilt durch vier.  
  
 **Höhe: DLUs**  
  
 Legt die Höhe des Layoutrasters in DLUs fest. Eine vertikale DLU entspricht der durchschnittlichen Höhe der Schriftart für das Dialogfeld dividiert durch 8.
