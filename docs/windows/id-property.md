# ID Property
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Durch die Eigenschaft **ID** wird auf die einzelnen Zugriffstastentabellen\-Einträge im Programmcode verwiesen.  Dieser Befehlswert wird dem Programm übermittelt, wenn ein Benutzer die Zugriffstaste oder eine Tastenkombination drückt.  Um eine Zugriffstaste funktionsgleich mit einem Menüelement zu machen, müssen Sie deren IDs angleichen \(solange die IDs von Zugriffstastentabelle und Menüressource identisch sind\).  
  
## Anforderungen  
 Win32
