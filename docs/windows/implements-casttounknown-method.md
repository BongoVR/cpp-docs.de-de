# Implements::CastToUnknown-Methode
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Ruft einen Zeiger zur zugrunde liegenden IUnknown\-Schnittstelle.  
  
## Syntax  
  
```  
__forceinline IUnknown* CastToUnknown();  
```  
  
## Rückgabewert  
 Diese Operation immer folgt und gibt den IUnknown\-Zeiger zurück.  
  
## Hinweise  
 Interne Hilfsfunktion.  
  
## Anforderungen  
 **Header:** implements.h  
  
 **Namespace:** Microsoft::WRL
