# in (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Gibt an, dass ein Parameter aus der aufgerufenen Prozedur an die aufrufende Prozedur übergeben werden soll.  
  
## Syntax  
  
```  
  
[in]  
  
```  
  
## Hinweise  
 Das Attribut **in** C\+\+ verfügt über die gleichen Funktionen wie das [in](http://msdn.microsoft.com/library/windows/desktop/aa367051) MIDL\-Attribut.  
  
## Beispiel  
 Weitere Informationen finden Sie unter [bindbar](../windows/bindable.md) als Beispiel zur Verwendung **in**verwendet.  
  
## Anforderungen  
  
### Attribut\-Kontext  
  
|||  
|-|-|  
|**Betrifft**|Schnittstellen für, Schnittstellenmethode|  
|**Wiederholbar**|Nein|  
|**Erforderliche Attribute**|None|  
|**Ungültige Attribute**|**retval**|  
  
 Weitere Informationen über das kontexte finden Sie unter [Attribut\-Kontexte](../windows/attribute-contexts.md).
