# Inserting a Space Between Buttons on a Toolbar
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Normalerweise fügen Sie einen Abstand zwischen Schaltflächen ein, indem Sie Schaltflächen auf der Symbolleiste einfach voneinander wegziehen.  Um einen Abstand zu entfernen, ziehen Sie die Schaltflächen zueinander hin.  
  
### So fügen Sie einen Abstand vor einer Schaltfläche ein, auf die kein weiterer Abstand folgt  
  
1.  Ziehen Sie die Schaltfläche nach rechts oder nach unten, bis sie die nächste Schaltfläche etwa um die Hälfte überlappt.  
  
### So fügen Sie einen Abstand vor einer Schaltfläche ein, auf die ein weiterer Abstand folgt, der erhalten bleiben soll  
  
1.  Ziehen Sie die Schaltfläche, bis der rechte oder untere Schaltflächenrand die nächste Schaltfläche berührt oder geringfügig überlappt.  
  
### So fügen Sie einen Abstand vor einer Schaltfläche ein, auf die ein weiterer Abstand folgt, der entfernt werden soll  
  
1.  Ziehen Sie die Schaltfläche nach rechts oder nach unten, bis sie die nächste Schaltfläche etwa um die Hälfte überlappt.
