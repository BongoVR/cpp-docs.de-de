# OLE DB Consumer Attributes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die OLE DB\-Consumerattribute fügen Code auf Grundlage [OLE DB\-Consumervorlagen](../data/oledb/ole-db-consumer-templates-reference.md)ein, um einen Arbeitsen OLE DB\-Consumer zu erstellen, der Aufgaben wie das öffnende Tabellen ausführt, Befehle ausführt und Daten zugreift.  
  
|Attribut|Beschreibung|  
|--------------|------------------|  
|[db\_accessor](../windows/db-accessor.md)|Bindet Spalten in einem Rowset und bindet sie an die entsprechenden Accessor ist.|  
|[db\_column](../windows/db-column.md)|Bindet eine angegebene Spalte auf das Rowset.|  
|[db\_command](../windows/db-command.md)|Führt einen OLE DB\-Befehl aus.|  
|[db\_param](../windows/db-param.md)|Ordnet die angegebene Membervariable mit einer Eingabe oder einem Ausgabeparameter.|  
|[db\_source](../windows/db-source.md)|Erstellt und kapselt eine Verbindung von einem Anbieter, einer Datenquelle.|  
|[db\_table](../windows/db-table.md)|Öffnet eine OLE DB\-Tabelle.|
