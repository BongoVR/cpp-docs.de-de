# oleautomation
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Gibt an, dass eine Schnittstelle mit Automatisierung kompatibel ist.  
  
## Syntax  
  
```  
  
[oleautomation]  
  
```  
  
## Hinweise  
 Das Attribut **oleautomation** C\+\+ verfügt über die gleichen Funktionen wie das [oleautomation](http://msdn.microsoft.com/library/windows/desktop/aa367129) MIDL\-Attribut.  
  
## Beispiel  
 Weitere Informationen finden Sie in den Beispielen für [defaultvalue](../windows/defaultvalue.md) und [nicht erweiterbar](../windows/nonextensible.md) für eine Beispiel verwenden aus **oleautomation**.  
  
## Anforderungen  
  
### Attribut\-Kontext  
  
|||  
|-|-|  
|**Betrifft**|`interface`|  
|**Wiederholbar**|Nein|  
|**Erforderliche Attribute**|None|  
|**Ungültige Attribute**|**dispinterface**|  
  
 Weitere Informationen über das kontexte finden Sie unter [Attribut\-Kontexte](../windows/attribute-contexts.md).
