# Picking up a Color from an Image to Use Elsewhere (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Das Tool **Farbe wählen** verwandelt beliebige Farben des Bildes in die aktuelle Vordergrund\- oder Hintergrundfarbe. Dies hängt davon ab, ob Sie die linke oder die rechte Maustaste drücken.  Um das Tool **Farbe wählen** zu deaktivieren, wählen Sie ein anderes Tool aus.  
  
> [!TIP]
>  Eine QuickInfo wird angezeigt, wenn Sie mit der Maus auf eine Schaltfläche auf der Symbolleiste zeigen.  Mithilfe dieser Tipps können Sie die Funktionen der einzelnen Schaltflächen leichter identifizieren.  
  
### So übertragen Sie eine Farbe  
  
1.  Klicken Sie auf der Symbolleiste des Grafik\-Editors \(oder unter dem Befehl **Tools** im Menü **Bild**\) auf das Tool **Farbe wählen**.  
  
2.  Wählen Sie im Bild die Farbe aus, die übertragen werden soll.  
  
    > [!NOTE]
    >  Nachdem Sie eine Farbe übertragen haben, aktiviert der Grafik\-Editor das zuletzt verwendete Tool.  
  
3.  Zeichnen Sie in der Vordergrundfarbe, indem Sie die linke Maustaste drücken, oder in der Hintergrundfarbe, indem Sie die rechte Maustaste drücken.
