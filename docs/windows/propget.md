# propget
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Gibt eine Eigenschaftenaccessor für Reservierungen an.  
  
## Syntax  
  
```  
  
[propget]  
  
```  
  
## Hinweise  
 Das Attribut **propget** C\+\+ verfügt über die gleichen Funktionen wie das [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL\-Attribut.  
  
## Beispiel  
 Weitere Informationen finden Sie im Beispiel für [bindbar](../windows/bindable.md) B. für eine Verwendung von **propget**.  
  
## Anforderungen  
  
### Attribut\-Kontext  
  
|||  
|-|-|  
|**Betrifft**|Methode|  
|**Wiederholbar**|Nein|  
|**Erforderliche Attribute**|None|

 Weitere Informationen über das kontexte finden Sie unter [Attribut\-Kontexte](../windows/attribute-contexts.md).
