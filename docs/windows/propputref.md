# propputref
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Gibt eine property\-setting\-Funktion an, die einen Verweis anstelle eines Werts verwendet wird.  
  
## Syntax  
  
```  
  
[propputref]  
  
```  
  
## Hinweise  
 Das Attribut **propputref** C\+\+ verfügt über die gleichen Funktionen wie das [propputref](http://msdn.microsoft.com/library/windows/desktop/aa367147) MIDL\-Attribut.  
  
## Beispiel  
 Weitere Informationen finden Sie im Beispiel für [bindbar](../windows/bindable.md) B. für eine Verwendung von **propputref**.  
  
## Anforderungen  
  
### Attribut\-Kontext  
  
|||  
|-|-|  
|**Betrifft**|Methode|  
|**Wiederholbar**|Nein|  
|**Erforderliche Attribute**|None|  
|**Ungültige Attribute**|**propget**, **propput**|  
  
 Weitere Informationen über das kontexte finden Sie unter [Attribut\-Kontexte](../windows/attribute-contexts.md).
