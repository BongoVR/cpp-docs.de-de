# Removing Space Between Buttons on a Toolbar
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die folgende Prozedur zeigt Ihnen, wie ein Zwischenraum zwischen Schaltflächen auf einer Symbolleiste entfernt wird.  
  
### So entfernen Sie einen Abstand zwischen Schaltflächen auf einer Symbolleiste  
  
1.  Ziehen Sie die Schaltfläche auf einer Seite des Abstands zur Schaltfläche auf der anderen Seite des Abstands, bis sie die nächste Schaltfläche etwa um die Hälfte überlappt.  
  
     Wenn sich auf der Seite der Schaltfläche, von der Sie wegziehen, kein Abstand befindet und Sie die Schaltfläche über die Mitte der benachbarten Schaltfläche hinausziehen, fügt der Symbolleisten\-Editor einen weiteren Abstand auf der gegenüber liegenden Seite derjenigen Schaltfläche ein, die Sie ziehen.
