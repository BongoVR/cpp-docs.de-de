# Saving and Loading Different Color Palettes (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Sie können eine Farbpalette mit [benutzerdefinierten Farben](../windows/customizing-or-changing-colors-image-editor-for-icons.md) speichern und laden.  \(Standardmäßig wird die zuletzt verwendete Farbpalette beim Start von Visual Studio automatisch geladen.\)  
  
> [!TIP]
>  Da die Bildbearbeitung nicht die standardmäßige Farbpalette wiederherstellen kann, sollten Sie diese unter einem Namen, wie z. B. standard.pal oder default.pal speichern, damit Sie die Standardeinstellungen problemlos wiederherstellen können.  
  
### So speichern Sie eine benutzerdefinierte Farbpalette  
  
1.  Wählen Sie im Menü **Bild** die Option **Palette speichern** aus.  
  
2.  Navigieren Sie zu dem Verzeichnis, in dem Sie die Farbpalette speichern möchten, und geben Sie einen Namen für die Palette ein.  
  
3.  Klicken Sie auf **Speichern**.  
  
### So laden Sie eine benutzerdefinierte Farbpalette  
  
1.  Wählen Sie im Menü **Bild** die Option **Palette laden** aus.  
  
2.  Navigieren Sie im [Dialogfenster „Farbpalette laden“](../windows/load-palette-colors-dialog-box-image-editor-for-icons.md) zum entsprechenden Verzeichnis, und wählen Sie die Palette, die Sie laden möchten.  Farbpaletten werden mit der Dateinamenerweiterung „.pal“ gespeichert.
