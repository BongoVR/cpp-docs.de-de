# Selecting Foreground or Background Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Mit Ausnahme des Tools **Radierer** zeichnen die Tools der **Bildbearbeitungs**\-Symbolleiste in der aktuellen Vordergrund\- oder Hintergrundfarbe, wenn Sie die linke bzw. rechte Maustaste drücken.  
  
### So wählen Sie eine Vordergrundfarbe aus  
  
1.  Klicken Sie mit der linken Maustaste auf die gewünschte Farbe auf der [Farbpalette](../windows/colors-window-image-editor-for-icons.md).  
  
### So wählen Sie eine Hintergrundfarbe aus  
  
1.  Klicken Sie mit der rechten Maustaste auf die gewünschte Farbe auf der **Farbpalette**.  
  
 Informationen zum Hinzufügen von Ressourcen zu verwalteten Projekten finden Sie unter [Ressourcen in Anwendungen](../Topic/Resources%20in%20Desktop%20Apps.md) im *.NET Framework\-Entwicklerhandbuch.*

 Keine
