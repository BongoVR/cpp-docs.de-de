# Selecting Multiple Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### So markieren Sie mehrere Steuerelemente  
  
1.  Wählen Sie im [Fenster "Toolbox"](../Topic/Toolbox.md) das Tool **Zeiger** aus.  
  
2.  Ziehen Sie den Zeiger, um ein Markierungsfeld um die Steuerelemente zu zeichnen, die Sie im Dialogfeld markieren möchten.  
  
     Wenn Sie die Maustaste loslassen, werden alle Steuerelemente markiert, die sich vollständig oder teilweise innerhalb des Markierungsfelds befinden.  
  
     \- oder \-  
  
-   Halten Sie die UMSCHALTTASTE gedrückt, und klicken Sie auf die Steuerelemente, die in die Markierung aufgenommen werden sollen.  
  
     \- oder \-  
  
-   Halten Sie die STRG\-TASTE gedrückt, und klicken Sie auf die Steuerelemente, die in die Markierung aufgenommen werden sollen.  
  
#### So entfernen Sie ein Steuerelement aus einer Gruppe markierter Steuerelemente oder fügen einer solchen Gruppe ein Steuerelement hinzu  
  
1.  Halten Sie die UMSCHALTTASTE gedrückt, und klicken Sie auf das Steuerelement, das Sie aus der Gruppe der markierten Steuerelemente entfernen bzw. dieser hinzufügen möchten.  
  
    > [!NOTE]
    >  Wenn Sie die STRG\-TASTE gedrückt halten, während Sie auf ein Steuerelement innerhalb einer Markierung klicken, wird dieses Steuerelement zum bestimmenden Element innerhalb der Markierung.  Weitere Informationen finden Sie unter [Festlegen des bestimmenden Steuerelements](../mfc/specifying-the-dominant-control.md).
