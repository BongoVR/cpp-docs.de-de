# Ein- oder Ausblenden der Symbolleiste des Dialog-Editors
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Wenn Sie den Dialog\-Editor öffnen, wird die dazugehörige Symbolleiste automatisch im oberen Bereich der Projektmappe angezeigt.

 Die Symbolleiste des Dialog\-Editors enthält Schaltflächen, mit deren Hilfe das Steuerelementlayout, z. B. Größe und Ausrichtung, im Dialogfeld angepasst werden kann.  Die Schaltflächen auf der Symbolleiste des Dialog\-Editors entsprechen den Befehlen im Menü **Format**.  Einzelheiten finden Sie unter [Zugriffstasten für den Dialog\-Editor](../mfc/accelerator-keys-for-the-dialog-editor.md).  
  
 Wenn Sie sich im Dialog\-Editor befinden, können Sie die Symbolleiste ein\- und ausblenden, indem Sie sie aus der Liste der verfügbaren Symbolleisten und Fenster auswählen.  
  
### So blenden Sie die Symbolleiste des Dialog\-Editors ein oder aus  
  
1.  Klicken Sie im Menü **Ansicht** auf **Symbolleisten**, und wählen Sie dann im Untermenü **Dialog\-Editor** aus.  
  
    > [!NOTE]
    >  Wenn Sie eine Dialogfeldressource im Dialog\-Editor öffnen, wird in der Standardeinstellung die Symbolleiste des Dialog\-Editors angezeigt. Wenn die Symbolleiste jedoch explizit ausgeblendet wird, muss sie beim nächsten Öffnen einer Dialogfeldressource wieder aktiviert werden.
