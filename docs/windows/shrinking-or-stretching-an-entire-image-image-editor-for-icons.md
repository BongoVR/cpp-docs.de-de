# Shrinking or Stretching an Entire Image (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### So verkleinern oder strecken Sie ein ganzes Bild  
  
1.  Markieren Sie das gesamte Bild.  
  
     Wenn gerade ein Bildbereich markiert ist, Sie jedoch das gesamte Bild markieren möchten, klicken Sie außerhalb des aktuellen Markierungsrahmens auf eine beliebige Stelle im Bild.  
  
2.  Halten Sie die UMSCHALTTASTE gedrückt, und ziehen Sie einen Ziehpunkt, bis das Bild die gewünschte Größe hat.
