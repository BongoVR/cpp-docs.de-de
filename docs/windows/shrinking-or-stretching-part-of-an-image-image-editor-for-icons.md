# Shrinking or Stretching Part of an Image (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### So verkleinern oder strecken Sie einen Teil eines Bildes  
  
1.  Markieren Sie den Teil des Bildes, dessen Größe Sie ändern möchten.  Weitere Informationen finden Sie unter [Markieren eines Bildbereichs](../mfc/selecting-an-area-of-an-image-image-editor-for-icons.md).  
  
2.  Ziehen Sie einen der Ziehpunkte, bis die Markierung die gewünschte Größe hat.
