# switch_is
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Gibt den Ausdruck oder den Bezeichner an, die als diskriminierende Union fungiert, die das Gewerkschaftsmitglied auswählt.  
  
## Syntax  
  
```  
  
[switch_is]  
  
```  
  
## Hinweise  
 Das Attribut **switch\_is** C\+\+ verfügt über die gleichen Funktionen wie das [switch\_is](http://msdn.microsoft.com/library/windows/desktop/aa367275) MIDL\-Attribut.
