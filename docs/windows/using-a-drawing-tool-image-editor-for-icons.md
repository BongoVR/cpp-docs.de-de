# Using a Drawing Tool (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Die Tools für das Freihandzeichnen und das Löschen im Grafik\-Editor funktionieren alle nach demselben Prinzip: Sie wählen zunächst das Tool aus und wählen dann, falls erforderlich, [Vordergrund\- und Hintergrundfarben](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md) sowie Optionen für Größe und Form aus.  Anschließend verschieben Sie den Mauszeiger auf das Bild und klicken oder ziehen den Zeiger, um zu zeichnen oder zu löschen.  
  
 Nachdem Sie das Tool **Radierer**, **Pinsel** oder **Airbrush** ausgewählt haben, enthält die Optionsauswahl die Optionen für das Tool.  
  
> [!TIP]
>  Anstatt das **Radierertool** zu verwenden, erscheint es Ihnen vielleicht praktischer, mit einem der Zeichentools in der Hintergrundfarbe zu zeichnen.  
  
 Zeichentools können entweder über die Symbolleiste des Grafik\-Editors oder im Menü **Bild** ausgewählt werden.  
  
### So wählen und verwenden Sie ein Zeichentool aus der Symbolleiste des Grafik\-Editors  
  
1.  Klicken Sie auf der Symbolleiste des Grafik**\-**Editors auf eine Schaltfläche.  
  
    -   Wenn Sie die linke Maustaste drücken, übermalt das Tool **Radierer** das Bild mit der aktuellen Hintergrundfarbe.  
  
    -   Das Tool **Stift** zeichnet Freihandobjekte mit einer konstanten Stärke von einem Pixel.  
  
    -   Durch die Optionsauswahl werden Form und Größe des Tools **Pinsel** festgelegt.  
  
    -   Das Tool **Airbrush** verteilt Farbpixel willkürlich um den Mittelpunkt des Pinsels.  
  
        > [!TIP]
        >  Wenn Sie mit der Maus auf die Schaltflächen auf der [Symbolleiste des Grafik\-Editors](../mfc/toolbar-image-editor-for-icons.md) zeigen, werden QuickInfos angezeigt.  Mithilfe dieser Tipps können Sie die Funktionen der hier erwähnten Schaltflächen leichter identifizieren.  
  
2.  Wählen Sie ggf. Farben und einen Pinsel aus:  
  
    -   Klicken Sie in der [Farbpalette](../windows/colors-window-image-editor-for-icons.md) mit der linken Maustaste, um eine Vordergrundfarbe auszuwählen, oder mit der rechten Maustaste, um eine Hintergrundfarbe auszuwählen.  
  
    -   Klicken Sie in der [Optionsauswahl](../mfc/toolbar-image-editor-for-icons.md) auf eine Form, die den gewünschten Pinsel darstellt.  
  
3.  Richten Sie den Mauszeiger auf die Stelle des Bildes, an der Sie mit dem Zeichnen oder Malen beginnen möchten.  Der Mauszeiger ändert seine Form entsprechend dem Tool, das Sie ausgewählt haben.  
  
4.  Drücken Sie die linke Maustaste \(für die Vordergrundfarbe\) oder die rechte Maustaste \(für die Hintergrundfarbe\), und halten Sie diese während des Zeichnens gedrückt.  
  
### So wählen und verwenden Sie ein Zeichentool aus dem Menü "Bild"  
  
1.  Klicken Sie im Menü **Bild** auf den Befehl **Tools**.  
  
2.  Wählen Sie im Untermenü das gewünschte Tool aus.
