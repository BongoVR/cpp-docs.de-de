# Using the 256-Color Palette (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Um mit einer Markierung aus der 256\-Farben\-Palette zu zeichnen, müssen Sie die Farben im [Fenster "Farben"](../windows/colors-window-image-editor-for-icons.md) aus der Farbpalette auswählen.  
  
### So wählen Sie eine Farbe für große Symbole aus einer 256\-Farben\-Palette aus  
  
1.  Wählen Sie das große Symbol oder den Cursor aus, oder erstellen Sie ein neues großes Symbol oder einen neuen Cursor.  
  
2.  Wählen Sie im Fenster **Farben** aus der Farbpalette eine der 256 Farben aus.  
  
     Die ausgewählte Farbe wird im Fenster **Farben** in der Farbpalette zur aktuellen Farbe.  
  
    > [!NOTE]
    >  Die anfängliche, für Bilder mit 256 Farben verwendete Palette stimmt mit der Palette überein, die von der Windows\-API CreateHalftonePalette zurückgegeben wird.  Alle für die Windows\-Shell vorgesehenen Symbole sollten diese Palette verwenden, um bei der Umsetzung der Palette ein Flimmern zu vermeiden.  
  
 Informationen zum Hinzufügen von Ressourcen zu verwalteten Projekten finden Sie unter [Ressourcen in Anwendungen](../Topic/Resources%20in%20Desktop%20Apps.md) im *.NET Framework\-Entwicklerhandbuch.* Informationen zum manuellen Hinzufügen von Ressourcendateien zu verwalteten Projekten, zum Zugreifen auf Ressourcen, zum Anzeigen statischer Ressourcen und zum Zuweisen von Ressourcenzeichenfolgen zu Eigenschaften finden Sie unter [Exemplarische Vorgehensweise: Lokalisieren von Windows Forms](assetId:///9a96220d-a19b-4de0-9f48-01e5d82679e5) und [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md).
