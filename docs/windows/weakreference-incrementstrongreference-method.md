# WeakReference::IncrementStrongReference-Methode
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Unterstützt die WRL-Infrastruktur und ist nicht direkt aus dem Code verwendet werden.  
  
## <a name="syntax"></a>Syntax  
  
```  
ULONG IncrementStrongReference();  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Der inkrementierte starken Verweiszähler.  
  
## <a name="remarks"></a>Hinweise  
 Erhöht die starken Verweis von der aktuellen WeakReference-Objekt.  
  
## <a name="requirements"></a>Anforderungen  
 **Header:** implements.h  
  
 **Namespace:** Microsoft::WRL::Details
